====== 개요 ====== 
관리환경을 사용하기 위하여 소속된 프로젝트의 정보를 등록하여 승인을 요청하고, 승인된 프로젝트의 정보를 수정하는 기능이다. 외부에서 행안부 프레임워크를 사용하며 지원대상인 프로젝트인 경우는 신규 프로젝트를 신청할 수 있다.(단, 지원대상 프로젝트에 한하여 허용됨)

====== 기능 설명 ======
  - 프로젝트신청 목록 : 프로젝트신청 목록을 검색 조회한다.
  - 프로젝트신청 등록 : 프로젝트신청 정보를 등록한다.
  - 프로젝트신청 상세 : 프로젝트신청 상세 정보를 조회한다.  
  - 프로젝트신청 수정 : 프로젝트신청 정보를 수정한다.

====== 화면 설명 ======

===== 프로젝트신청 목록 =====
사용자가 프로젝트신청을 한 프로젝트의 목록을 조회한다.

{{:egovframework:mgt:egovprjctreqstlist.jpg|프로젝트신청 목록화면}}

  * 프로젝트 명 선택 : 프로젝트의 상세정보 화면으로 이동한다. 
  * 등록 : 서비스요청을 하는 프로젝트의 정보를 입력하고 등록한다. 


===== 프로젝트신청 등록  =====
프로젝트 신청목록에서 등록버튼을 클릭하면 신규 프로젝트를 등록하는 화면으로 이동한다. 등록 후 프로젝트 신청내용을 변경하게되면 프로젝트의 상태가 승인요청상태로 자동변경된다. 따라서 프로젝트 변경후에는 관리자에 의해 재승인을 받아야만 프로젝트로 입장(관리환경 시스템 로그인)할 수 있다.

{{:egovframework:mgt:egovprjctreqstregist2.jpg|프로젝트신청 등록화면}}

  * 프로젝트 기간 : 달력아이콘을 클릭하여 프로젝트 기간을 선택한다. 
  * 프로젝트 소재지 : 돋보기 아이콘을 클릭하여 프로젝트 소재지를 검색한다. 
  * 소속인원 추가 : 포탈회원 검색 팝업화면을 띄운다.
  * 소속인원 삭제 : 리스트에서 선택한 포탈회원을 삭제한다.
  * 사용모듈 추가 : 배포대상 검색 팝업화면을 띄운다. 
  * 사용모듈 삭제 : 리스트에서 선택한 사용모듈을 삭제한다.
  * 확인 : 입력한 프로젝트 정보를 등록한다.
  * 목록 : 프로젝트신청 목록 조회 화면으로 이동한다. 


===== 포탈회원 검색 팝업 =====
프로젝트의 소속인원을 추가하기 위하여 포탈회원 선택 팝업을 띄운다.

{{:egovframework:mgt:egovportalmbersearchpopup.jpg|포탈회원 검색 팝업화면}}

  * 검색 : **회원ID**, **회원명** 중 선택하고 입력한 조회조건에 따라 회원을 조회한다.
  * 선택 : 프로젝트에 추가하고자 하는 회원을 체크하여 선택한다. 
  * 회원명 선택 : 포탈회원의 상세 정보를 조회한다. 
  * 닫기 : 화면을 종료한다.


===== 배포대상 검색 팝업 =====
프로젝트에서 사용하는 모듈을 추가하기 위하여 배포대상 선택 팝업을 띄운다.

{{:egovframework:mgt:egovprjctwdtblistpupup.jpg|배포대상 검색 팝업화면}}

  * 선택 : 리스트에서 체크된 배포대상명을 프로젝트 등록의 사용모듈 리스트에 추가한다.
  * 닫기 : 화면을 종료한다. 



===== 프로젝트신청 상세 =====
프로젝트의 상세정보를 조회한다. 

{{:egovframework:mgt:egovprjctreqstdetail2.jpg|프로젝트신청 상세화면}}

  * 수정 : 프로젝트신청 정보를 수정하는 화면으로 이동한다.
  * 목록 : 프로젝트신청 목록 화면으로 이동한다. 


===== 프로젝트신청 수정 =====
프로젝트 정보를 수정한다. 신청한 프로젝트의 정보를 수정하게 되면 프로젝트상태 정보가 승인요청으로 자동변경되고 관리자에 의해 재승인을 받아야만 프로젝트로 입장(관리환경 시스템 로그인)할 수 있다.(관리자 승인후 프로젝트 내용 변경시는 관리자 재승인이 필요함을 주지)

{{:egovframework:mgt:egovprjctreqstupdt.jpg|프로젝트신청 수정화면}}

  * 확인 : 수정하기 위하여 입력한 정보를 저장한다. 
  * 목록 : 프로젝트신청 목록 화면으로 이동한다.
