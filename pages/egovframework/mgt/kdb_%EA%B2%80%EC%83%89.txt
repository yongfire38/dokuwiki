====== 개요 ====== 
다양한 KDB정보에 대한 검색을 할 수 있는 기능을 제공한다.

====== 기능 설명 ======
  - KDB검색 목록 : 검색조건에 해당하는 KDB 검색 결과목록이 조회된다.
  - KDB검색 상세 : KDB의 상세 정보를 조회한다. 
  

====== 화면 설명 ======

===== KDB 목록 =====
전체 KDB에 대한 목록을 조회한다.

{{:egovframework:mgt:kdbsearchlist.jpg|KDB검색 목록}}

  * 검색조건 : 다음과 같은 항목의 검색조건을 선택하고 검색어를 입력한다. 
    * 통합검색 : 입력한 검색어를 포함하는 요청제목,요청내용,처리내용,태그 들을 검색한다. 
    * 태그검색 : 입력한 검색어를 포함하는 태그 내용을 가진 KDB정보들을 검색한다.
  * 검색 : 선택한 검색조건을 기준으로  KDB정보를 검색한다.
  * 조회내용
    * 요청제목 : KDB의 요청제목 내용을 보여준다.
    * 요청내용 : KDB의 요청내용의 일부 내용을 보여준다.
    * 요청일 : KDB에 등록된 날짜를 표시한다. 
    * 태그 : KDB에 등록된 KDB정보를 보여준다.
     * 지식유형 > 요청대상 : KDB 지식유형 및 요청대상명을 표시한다.

===== KDB 상세====
KDB상세화면은 KDB정보에 대한 상세내용을 조회하는 화면이다. 

{{:egovframework:mgt:kdbsearchdetail.jpg|KDB검색 상세}}

  * 서비스 요청 
    * 요청 제목 : 서비스 요청 제목을 보여준다.
    * 요청 내용 : 서비스 요청 내용을 보여준다.
    * 요청일 : KDB에 등록된 날짜를 표시한다. 
    * 지식유형 > 요청대상 : 지식유형 및 요청대상명을 표시한다

  * 서비스 처리
    * 처리 내용 : 서비스 요청에 대한 처리 내용을 보여준다.
    * 태그 : 서비스 처리시 등록된 태그정보를 보여준다.

  * 목록 : KDB 목록 조회 화면으로 이동한다
