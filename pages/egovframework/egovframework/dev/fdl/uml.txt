====== UML ======
===== 개요 =====
통합개발환경에서 ER 의 논리모델과 물리모델 작성을 위한 도구로써 DDL 생성, 테이블 명세서 생성, Reverse engineering 등의 기능을 제공합니다ㅏ.

 : 기능에 대한 개요

===== 설명 =====

  * ER Diagram 편집 : 편리한 ER Diagram 편집기능을 제공.
  * Reverse engineering : Oracle, PostgreSQL, MySQL, hsqldb 등의 Database로부터 테이블 Import.
  * DDL 생성 : 작성된 ERD 를 이용한 DDL 생성
  * 테이블 명세서 Export : HTML 형식으로 테이블 명세서 Export.
