====== 개요 ====== 
모니터링 서비스를 사용하기 위해 로그인한다.

\\
====== 기능 설명 ======
  - 로그인 화면 : 모니터링 서비스를 사용하기 위한 인증 화면이다.

\\
====== 화면 설명 ======

===== 로그인 화면 =====
{{:egovframework:oe2:가이드:login.png}}


  * 아이디 : 등록된 아이디를 입력한다.
  * 암호 : 등록된 암호를 입력한다.

==== 화면 상세 정보 ====

  * 디폴트로 세팅된 로그인 정보는 아이디: admin, 암호: admin 이다.
