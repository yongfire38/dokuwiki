

====== 개요 ====== 
변경요청을 위한 변경요청서의 등록/수정/상세조회/목록검색 할 수 있는 기능을 제공한다.

====== 기능 설명 ======
  - 변경요청서 목록 : 변경요청서의 목록을 검색 조회한다.
  - 변경요청서 상세 : 변경요청서 정보를 상세 조회한다.
  - 변경요청서 등록 : 변경요청서 정보를 등록한다.
  - 변경요청서 수정 : 변경요청서 정보를 수정한다.
  - 변경요청서 요청근거 상세팝업 : 변경요청서를 등록한 요청시스템의 출처 정보를 조회한다.
  

====== 화면 설명 ======

===== 변경요청서 목록 =====
변경요청서 목록은 페이지당 10건씩 조회되며 검색조건은 **업무구분**, **긴급여부**, **처리상태**, **변경요청명**, **변경요청자명**이다. 



  * 검색 : 선택 또는 입력한 검색조건을 기반으로 변경요청서 목록을 조회한다.
  * 등록 : 요청된 변경에 대한 변경요청서를 등록하기 위한 화면으로 이동한다.



===== 변경요청서 상세  =====
변경요청서 목록에서 변경요청명을 클릭하면 해당 변경요청서의 정보를 상세 조회한다.



  * 요청근거 : 변경요청서를 등록한 요청시스템의 출처 정보를 팝업조회한다.
  * 수정 : 변경요청서 정보를 수정할 수 있는 편집 가능한 화면으로 이동한다.
  * 삭제 : 변경요청서 정보를 삭제한다.
  * 목록 : 변경요청서 목록 조회 화면으로 이동한다. 



===== 변경요청서 등록 =====
변경요청서를 등록하는 기능을 제공한다.



  * 저장 : 입력한 변경요청서 정보를 등록한다. 
  * 목록 : 변경요청서 목록 조회 화면으로 이동한다. 


===== 변경요청서 수정 =====
변경요청서 상세 조회 화면에서 수정 버튼을 클릭하여 변경요청서 정보를 수정한다.



  * 저장 : 입력한 변경요청서 정보를 수정한다. 
  * 목록 : 변경요청서 목록 화면으로 이동한다. 


===== 변경요청서 요청근거 상세팝업 =====
변경요청서 상세 화면에서 요청근거를 클릭하면 변경요청서를 등록한 요청시스템의 출처 정보를 상세 조회한다.



  * 닫기 : 화면을 종료한다.
