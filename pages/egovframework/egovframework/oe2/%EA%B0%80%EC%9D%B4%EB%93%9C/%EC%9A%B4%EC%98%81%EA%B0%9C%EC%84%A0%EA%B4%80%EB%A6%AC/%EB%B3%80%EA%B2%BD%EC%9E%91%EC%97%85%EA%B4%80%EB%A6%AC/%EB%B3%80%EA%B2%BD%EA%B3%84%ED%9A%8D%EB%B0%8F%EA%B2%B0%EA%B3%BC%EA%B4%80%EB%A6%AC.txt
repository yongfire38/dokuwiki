

====== 개요 ====== 

변경요청처리에 대하여 변경작업의 계획을 등록/수정/상세조회/목록검색 할 수 있는 기능을 제공한다.
관리자에 의해 승인된 변경작업 계획에 대하여 변경작업 정보를 등록/수정/상세조회/목록검색 할 수 있는 기능을 제공한다.


====== 기능 설명 ======

  - 변경작업 목록 : 변경작업계획/변경작업결과 목록을 검색 조회한다.
  - 변경작업계획 상세 : 변경작업계획 정보를 상세 조회한다.
  - 변경작업계획 등록 : 변경작업계획 정보를 등록한다.
  - 변경작업 상세 : 변경작업 정보를 상세 조회한다.
  - 변경작업 등록 : 변경작업 정보를 등록한다.
  - 변경요청서 상세 팝업 : 변경요청처리에 묶인 변경요청서의 상세 정보를 조회한다.
  

====== 화면 설명 ======

===== 변경작업 목록 =====
변경작업 목록은 페이지당 10건씩 조회되며 검색조건은 **처리상태**, **긴급찰;**, **접수일**이다. 



  * 검색 : 선택 또는 입력한 검색조건을 기반으로 변경작업 목록을 조회한다.


===== 변경작업계획 상세  =====
변경작업 목록에서 처리상태가 계획검토인 것을 클릭하면 해당 변경작업계획의 정보를 상세 조회한다.



  * 변경요청서조회 : 변경요청처리에 묶인 변경요청서 정보를 조회한다.
  * 목록 : 변경작업 목록 조회 화면으로 이동한다. 



===== 변경작업계획 등록 =====
변경작업 목록 화면에서 처리상태가 접수/계획인 것을 클릭하면 변경작업계획 정보를 등록하는 화면으로 이동하는 기능을 제공한다.



  * 변경요청서조회 : 변경요청처리에 묶인 변경요청서 정보를 조회한다.
  * 계획검토 요청 : 관리자에게 변경작업계획 검토를 요청한다.
  * 담당자검토 : 계획검토 담당자권한위임인 경우 변경담당자 스스로 변경작업계획 검토를 승인한다.
  * 저장 : 입력한 변경작업계획 정보를 등록한다.
  * 목록 : 변경작업 목록 조회 화면으로 이동한다.



===== 변경작업 상세  =====
변경작업 목록에서 처리상태가 완료검토/완료인 것을 클릭하면 해당 변경작업 정보를 상세 조회한다.



  * 목록 : 변경작업 목록 조회 화면으로 이동한다. 



===== 변경작업 등록 =====
변경작업 목록 화면에서 처리상태가 작업인 것을 클릭하면 변경작업 정보를 등록하는 화면으로 이동하는 기능을 제공한다.


  * 완료검토 요청 : 관리자에게 변경작업 검토를 요청한다.
  * 변경완료 : 변경완료검토 담당자권한위임인 경우 변경담당자 스스로 변경작업 검토를 승인한다.
  * 저장 : 입력한 변경작업 정보를 등록한다.
  * 목록 : 변경작업 목록 조회 화면으로 이동한다.


===== 변경요청서 상세팝업 =====
변경요청처리에 묶인 변경요청서 정보를 상세 조회한다.



  * 요청근거 : 변경요청서를 등록한 요청시스템의 출처 정보를 조회한다.
  * 닫기 : 화면을 종료한다.
