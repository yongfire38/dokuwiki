====== 개요 ====== 
 현재 모니터링되고 있는 하나 이상의 어플리케이션 중 하나를 선택하여 세부 항목을 확인하기 위한 선택화면이다. \\

====== 기능 설명 ======
  - 어플리케이션 선택 : 모니터링 세부 항목을 확인할 어플리케이션명과 서버 호스트명을 선택한다. 
  - 어플리케이션 선택 정보 : 현재 선택된 어플리케이션 정보를 확인한다.

\\

====== 화면 설명 ======

===== 모니터링 대상 선택 =====
{{:egovframework:oe2:가이드:어플리케이션:EgovApplicationSelect.png}}

  * 어플리케이션 : 모니터링 대상 어플리케이션 목록을 보여준다.
  - 기본적으로 등록된 어플리케이션 중에 현재 모니터링 정보가 수신되어 데이타베이스에 저장되는 어플리케이션 목록을 보여준다.
   - 목록 중에 (unknown)이 붙어 있는 것은 등록되어 있지 않지만, 모니터링 정보가 수신되고 있는 어플리케이션 이름을 보여 준다.
  * 서버 : 어플리케이션이 실행되고 있는 서버명을 보여 준다. 
  - 동일 어플리케이션에 한해 서버명은 다중 선택이 가능하다.
  * 조회조건 : 실시간 또는 기간 선택 조회가 가능하다.
  - 실시간 조회는 기본적으로 어플리케이션 에이전트가 실시간으로 전송하고 있는 모니터링 정보를 보여준다.
  - 조회기간 선택 조회는 해당 조회기간동안 데이터베이스에 저장된 모니터링 정보를 보여준다.
  - 단, 조회기간 선택 조회는 최대 24시간까지 조회가 가능하다.

  * 저장 : 선택한 정보를 저장한다.


\\

===== 모니터링 대상 선택 정보 =====
{{:egovframework:oe2:가이드:어플리케이션:EgovConfigurationDetail.png}}

  * 어플리케이션 : 현재 택한 모니터링 대상 어플리케이션 이름을 보여준다.
  * 서버 : 현재 택된 서버 호스트명을 보여 준다. 
  * 조회조건 : 현재 선택된 조회 조건 정보를 보여준다. 
  * 모니터링 대상 선택 : 현재 선택된 어플리케이션 선택 정보를 변경할 경우 모니터링 대상 선택 버튼을 눌러 다시 선택한다. \\
