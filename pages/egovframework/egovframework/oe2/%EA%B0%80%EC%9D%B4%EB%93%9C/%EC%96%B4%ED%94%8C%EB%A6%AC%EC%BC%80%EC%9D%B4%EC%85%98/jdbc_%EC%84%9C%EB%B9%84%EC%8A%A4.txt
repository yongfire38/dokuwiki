====== 개요 ====== 
JDBC 서비스 목록 및 상세 페이지는 JDBC 서비스 레이어의 호출에 걸린 시간과 호출횟수 등의 요약정보를 제공한다.

JDBC 서비스를 수행하는 시간(inclusive mode)과 JDBC 서비스만 수행하는데에만 걸린시간( exclusive mode )을 구분하여 보여준다.


\\

====== 기능 설명 ======
  - JDBC 서비스 목록 : JDBC 서비스의 수행정보의 목록을 보여준다.
  - JDBC 서비스 현황 상세정보 : JDBC 서비스의 수행정보를 상세하게 보여준다.

\\

====== 화면 설명 ======

===== JDBC 서비스 목록 =====
{{:egovframework:oe2:가이드:어플리케이션:EgovJdbcServicetList.png}}

  * 검색 : JDBC 서비스명, 총시간,호출갯수를 대상으로 JDBC 서비스명 검색은 JDBC 서비스명에 검색어가 포함되어있는 항목을 검색하며 총시간 및 호출갯수는 검색 기간 및 갯수 이상인 항목을 검색하여 보여준다.

  * 전체(inclusive) mode : 해당 레이어를 수행하는데 걸린 시간( 다른 레이어의 수행시간은 포함 )
  * 단계(exclusive) mode : 해당 레이어만 수행하는데 걸린 시간( 다른 레이어의 수행시간은 제외 )

  * JDBC 서비스명 : 실행된 JDBC 서비스명을 보여준다.
  * 총시간   : JDBC 서비스의 총 실행시간을 보여준다.
  * 호출갯수 : HJDBC 서비스가 몇 번이나 호출 되었는지를 보여준다.
  * Avg      : 평균실행시간을 보여줍니다.총실행시간 / 호출갯수
  * Adj.Avg  : 처음 실행시간을 제외한 평균을 보여줍니다. ( 총실행시간 - 최초실행시간 ) / ( 호출갯수 - 1 )
  * Min      : JDBC 서비스의 실행시간이 가장 짧았을 때의 실행시간을 보여준다.
  * Max      : JDBC 서비스의 실행시간이 가장 기었을 때의 실행시간을 보여준다.
  * First    : JDBC 서비스가 처음 실행되었을 때의 값을 보여준다.
  * Last     : JDBC 서비스가 마지막 실행되었을 때의 값을 보여준다.

\\

===== JDBC 서비스 현황 상세정보 =====
JDBC 서비스 목록에서 대상 JDBC 서비스명 부분을 클릭하면 JDBC 서비스 현황 상세정보 화면에서 선택된 대상의 상세정보를 확인할 수 있다.

{{:egovframework:oe2:가이드:어플리케이션:EgovJdbcServicetDetail.png}}

  * 트리구조         : JDBC 서비스의 실행 정보를 트리 구조로 보여준다.

  * JDBC 서비스 요약 : 위의 트리구조 중 JDBC 서비스의 요약정보를 보여준다.

  * 목록 : JDBC 서비스 목록 화면으로 이동한다.
