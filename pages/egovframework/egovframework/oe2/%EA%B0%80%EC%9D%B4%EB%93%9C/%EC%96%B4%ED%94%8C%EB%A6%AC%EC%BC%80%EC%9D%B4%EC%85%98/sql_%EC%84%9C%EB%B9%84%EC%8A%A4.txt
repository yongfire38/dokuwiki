====== 개요 ====== 
SQL 서비스 목록 및 상세페이지는 SQL 서비스 레이어의 요약을 제공한다.
여기서는 SQL 서비스의 실행했을 때 걸리는 시간이 어느 정도인지를 볼 수 있다. 

SQL 서비스를 수행하는 시간(inclusive mode)과 순수히 SQL서비스를 수행하는데에만 걸린시간( exclusive mode )을 구분하여 보여준다.

\\

====== 기능 설명 ======
  - SQL 서비스 목록 : SQL 서비스의 수행정보의 목록을 보여준다.
  - SQL 서비스 현황 상세정보 : SQL 서비스의 수행정보를 상세하게 보여준다.

\\

====== 화면 설명 ======

===== SQL 서비스 목록 =====
{{:egovframework:oe2:가이드:어플리케이션:EgovSqlServicetList.png}}


  * 검색 : SQL, 총시간,호출갯수를 대상으로 SQL 검색은 SQL에 검색어가 포함되어있는 항목을 검색하며 총시간 및 호출갯수는 검색 기간 및 갯수 이상인 항목을 검색하여 보여준다.

  * 전쳬(inclusive) mode : 해당 레이어를 수행하는데 걸린 시간( 다른 레이어의 수행시간은 포함 )
  * 단계(exclusive) mode : 해당 레이어만 수행하는데 걸린 시간( 다른 레이어의 수행시간은 제외 )

  * SQL      : 실행된 SQL을 보여준다.
  * 총시간   : SQL 서비스의 총 실행시간을 보여준다.
  * 호출갯수 : SQL 서비스가 몇 번이나 호출 되었는지를 보여준다.
  * Avg      : 평균실행시간을 보여줍니다.총실행시간 / 호출갯수
  * Adj.Avg  : 처음 실행시간을 제외한 평균을 보여준다. ( 총실행시간 - 최초실행시간 ) / ( 호출갯수 - 1 )
  * Min      : SQL 서비스의 실행시간이 가장 짧았을 때의 실행시간을 보여준다.
  * Max      : SQL 서비스의 실행시간이 가장 기었을 때의 실행시간을 보여준다.
  * First    : SQL 서비스가 처음 실행되었을 때의 값을 보여준다.
  * Last     : SQL 서비스가 마지막 실행되었을 때의 값을 보여준다.

\\
