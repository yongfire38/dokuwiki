====== 개요 ====== 
HTTP 서비스 목록 및 상세페이지는 HTTP 서비스 레이어의 요약을 제공한다.
여기서는 HTTP 서비스의 실행했을 때 걸리는 시간이 어느 정도인지를 볼 수 있다. 

HTTP Request를 요청하여 해당 요청을 수행하는 시간(inclusive mode)과 순수히 HTTP서비스를 요청하는데에만 걸린시간( exclusive mode, 어플리케이션 서비스, JDBC 서비스 등에서 건린시간을 제외한 )을 구분하여 보여준다.

\\

====== 기능 설명 ======
  - HTTP 서비스 목록 : HTTP 서비스의 수행정보의 목록을 보여준다.
  - HTTP 서비스 현황 상세정보 : HTTP 서비스의 수행정보를 상세하게 보여준다.

\\

====== 화면 설명 ======

===== HTTP 서비스 목록 =====
{{:egovframework:oe2:가이드:어플리케이션:egovHttpServicetList.png}}


  * 검색 : URL, 총시간,호출갯수를 대상으로 URL 검색은 URL에 검색어가 포함되어있는 항목을 검색하며 총시간 및 호출갯수는 검색 기간 및 갯수 이상인 항목을 검색하여 보여줍니다.

  * 전체(inclusive) mode : 해당 레이어를 수행하는데 걸린 시간( 다른 레이어의 수행시간은 포함 )
  * 단계(exclusive) mode : 해당 레이어만 수행하는데 걸린 시간( 다른 레이어의 수행시간은 제외 )

  * URL      : 실행된 HTTP 요청의 URL을 보여줍니다.
  * 총시간   : HTTP 요청의 총 실행시간을 보여줍니다.
  * 호출갯수 : HTTP 요청이 몇 번이아 호출 되었는지를 보여줍니다.
  * Avg      : 평균실행시간을 보여줍니다.총실행시간 / 호출갯수
  * Adj.Avg  : 처음 실행시간을 제외한 평균을 보여줍니다. ( 총실행시간 - 최초실행시간 ) / ( 호출갯수 - 1 )
  * Min      : HTTP요청의 실행시간이 가장 짧았을 때의 실행시간을 보여줍니다.
  * Max      : HTTP요청의 실행시간이 가장 기었을 때의 실행시간을 보여줍니다.
  * First    : HTTP요청을 처음 실행했을 때의 값을 보여줍니다.
  * Last     : HTTP요청을 마지막 실행했을 때의 값을 보여줍니다.

===== HTTP 서비스 현황 상세정보 =====
HTTP 서비스 목록에서 대상 URL 부분을 클릭하면 HTTP 서비스 현황 상세정보 화면에서 선택된 대상의 상세정보를 확인할 수 있습니다.

{{:egovframework:oe2:가이드:어플리케이션:egovHttpServicetDetail.png}}

  * 트리구조         : HTTP 서비스의 실행 정보를 트리 구조로 보여줍니다.

  * HTTP 서비스 요약 : 위의 트리구조 중 HTTP 서비스의 요약정보를 보여줍니다.
  * JDBC 서비스 요약 : 위의 트리구조 중 JDBC 서비스의 요약정보를 보여줍니다.
  * SQL 서비스 요약  : 위의 트리구조 중 SQL 서비스의 요약정보를 보여줍니다.

  * 목록 : HTTP 서비스 목록 화면으로 이동한다.
