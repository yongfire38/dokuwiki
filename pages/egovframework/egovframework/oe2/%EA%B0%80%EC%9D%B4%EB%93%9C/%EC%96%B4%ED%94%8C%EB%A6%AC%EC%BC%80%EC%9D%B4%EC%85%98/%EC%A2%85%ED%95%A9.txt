====== 개요 ====== 
종합화면은 선택된 모니터링 대상 어플리케이션의 HTTP 서비스, 어플리케이션 서비스, JDBC 서비스, SQL  서비스 및 오류에 대한 모니터링 요약정보를 보여준다.


\\

====== 기능 설명 ======
  - 종합 화면 : 종합화면은 HTTP 서비스, 어플리케이션 서비스, JDBC 서비스, SQL  서비스 및 오류에 대한 요약정보를 보여준다.

\\

====== 화면 설명 ======

===== 종합 화면 =====
{{:egovframework:oe2:가이드:어플리케이션:wiki_oe2_app3.png}}

  * 전체(inclusive) mode : 해당 레이어를 수행하는데 걸린 시간( 다른 레이어의 수행시간은 포함 )
  * 단계(exclusive) mode : 해당 레이어만 수행하는데 걸린 시간( 다른 레이어의 수행시간은 제외 )
  * 서비스 구분    : 각 레이어( HTTP, APPLICATION, JDBC, SQL ) 와 오류( EXCEPTION )의 구분
  * 실행수         : 각 레이어의 실행 수 및 오류( EXCEPTION ) 발생 건 수를 보여준다.
  * 실행수 비율         : 각 레이어의 실행 수 및 오류( EXCEPTION ) 발생 건 수 비율을 보여준다.
  * 실행시간       : 각 레이어의 실행된 시간을 보여준다.
  * 실행시간 비율       : 각 레이어의 실행된 시간의 비율을 보여준다.

 위 그림에서 실행시간 기준 챠트를 보면 해당 어플리케이션을 실행하였을 때 Application Service 영역에 시간이 많이 소모되었음을 확인할 수 있다.
