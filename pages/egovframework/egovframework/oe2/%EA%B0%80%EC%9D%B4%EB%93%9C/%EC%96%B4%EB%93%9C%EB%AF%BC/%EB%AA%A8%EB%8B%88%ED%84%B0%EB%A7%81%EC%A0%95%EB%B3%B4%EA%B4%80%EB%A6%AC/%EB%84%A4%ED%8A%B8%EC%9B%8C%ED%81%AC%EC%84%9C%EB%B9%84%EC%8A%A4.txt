

====== 개요 ====== 
네트워크서비스 정보를 관리하기 위하여 모니터링도구관리자가 네트워크서비스를 등록/수정/상세조회/목록검색 할 수 있는 기능을 제공한다.

====== 기능 설명 ======

  - 네트워크서비스 정보 목록 : 네트워크서비스 정보 목록을 검색 조회한다.
  - 네트워크서비스 정보 상세 : 등록된 네트워크서비스에대한 정보를 상세 조회한다.
  - 네트워크서비스 정보 등록 : 식별된 네트워크서비스에대한 정보를 등록한다.
  - 네트워크서비스 정보 수정 : 등록한 네트워크서비스에대한 정보를 수정한다.
  

====== 화면 설명 ======

===== 네트워크서비스 정보 목록 =====
네트워크서비스 정보  목록은 페이지당 10건씩 조회되며 검색조건은 서비스명, 시스템명이다. 

{{:egovframework:oe2:가이드:어드민:networkinfolist.gif|}}

  * 검색 : 선택 또는 입력한 검색조건을 기반으로 등록한 네트워크서비스에대한 정보를 목록으로 조회한다.
  * 등록 : 네트워크서비스 정보를 등록하기 위한 화면으로 이동한다.


===== 네트워크서비스 정보 상세  =====
네트워크서비스 정보 목록에서 서비스명을 클릭하면 해당 네트워크서비스에 대한 정보를 상세 조회한다.

{{:egovframework:oe2:가이드:어드민:networkinfodetail.gif|}}

  * 수정 : 네트워크서비스 정보를 수정할 수 있는 편집 가능한 화면으로 이동한다.
  * 목록 : 네트워크서비스 정보 목록 조회 화면으로 이동한다. 


===== 네트워크서비스 정보 등록 =====
식별한 위험을 등록할 수 있는 기능을 제공한다.

{{:egovframework:oe2:가이드:어드민:networkinforegist.gif|}}

  * 저장 : 입력한 네트워크서비스 정보를 등록한다.
  * 목록 : 네트워크서비스 정보 목록 조회 화면으로 이동한다.

===== 네트워크서비스 정보  수정 =====
네트워크서비스 정보 상세조회 화면에서 수정 버튼을 클릭한 경우  해당 네트워크서비스에대한 관리정보를 수정할 수 있는 기능을 제공한다.

{{:egovframework:oe2:가이드:어드민:networkinfoupdate.gif|}}

  * 저장 : 입력한 네트워크서비스 정보를 수정한다.
  * 목록 : 네트워크서비스 정보 목록 조회 화면으로 이동한다.
