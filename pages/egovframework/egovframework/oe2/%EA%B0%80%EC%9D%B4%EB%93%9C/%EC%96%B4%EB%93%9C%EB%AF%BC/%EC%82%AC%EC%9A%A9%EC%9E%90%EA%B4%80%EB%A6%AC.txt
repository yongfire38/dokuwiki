
====== 개요 ====== 
사용자정보를 관리하기 위하여 모니터링관리자가 위험을 등록/수정/상세조회/목록검색 할 수 있는 기능을 제공한다.

====== 기능 설명 ======

  - 사용자 정보 목록 : 사용자 정보 목록을 검색 조회한다.
  - 사용자 정보 상세 : 등록된 사용자에대한 정보를 상세 조회한다.
  - 사용자 정보 등록 : 식별된 사용자에대한 정보를 등록한다.
  - 사용자 정보 수정 : 등록한 사용자에대한 정보를 수정한다.
  

====== 화면 설명 ======

===== 사용자 정보 목록 =====
사용자 정보 목록은 페이지당 10건씩 조회되며 검색조건은 **사용자아이디**, **사용자명**, **이메일주소**, **역할**이다. 

{{:egovframework:oe2:가이드:어드민:userlist.gif|}}

  * 검색 : 선택 또는 입력한 검색조건을 기반으로 등록한 사용자에대한 정보를 목록으로 조회한다.
  * 등록 : 사용자를 등록하기 위한 화면으로 이동한다.


===== 사용자 정보 상세  =====
사용자 정보 목록에서 위험명을 클릭하면 해당 사용자에 대한 정보를 상세 조회한다.

{{:egovframework:oe2:가이드:어드민:userdetail.gif|}}

  * 목록 : 사용자 정보 목록 조회 화면으로 이동한다.
  * 삭제 : 해당 컨테이너 정보를 삭제한다.  
  * 수정 : 사용자 정보를 수정할 수 있는 편집 가능한 화면으로 이동한다.


===== 사용자 정보 등록 =====
사용자 정보를 등록할 수 있는 기능을 제공한다.

{{:egovframework:oe2:가이드:어드민:userregist.gif|}}

  * 저장 : 입력한 사용자 정보를 등록한다.
  * 목록 : 사용자 정보 목록 조회 화면으로 이동한다.
  * 초기화 : 입력한 컨테이너 정보를 초기화한다.
  * 사용자역할 : 모니터링관리자와 모니터링사용자 역할이 있으며, 모니터링사용자는 어드민 메뉴에 대한 권한이 없다.

===== 사용자 정보 수정 =====
사용자 정보 상세조회 화면에서 수정 버튼을 클릭한 경우  해당 사용자에 대한 관리정보를 수정할 수 있는 기능을 제공한다.

{{:egovframework:oe2:가이드:어드민:userupdate.gif|}}

  * 저장 : 입력한 사용자 정보를 수정한다.
  * 목록 : 사용자 정보 목록 조회 화면으로 이동한다.
  * 초기화 : 입력한 컨테이너 정보를 초기화한다.
