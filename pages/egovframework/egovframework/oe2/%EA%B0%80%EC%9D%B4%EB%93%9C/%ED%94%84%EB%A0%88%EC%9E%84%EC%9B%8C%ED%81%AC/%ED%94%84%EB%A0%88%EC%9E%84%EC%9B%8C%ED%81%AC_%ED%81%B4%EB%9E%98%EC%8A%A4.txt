====== 개요 ====== 
프레임워크 클래스 모니터링 정보는 전자정부 표준프레임워크가 적용된 클래스 정보를 제공한다.\\

전자정부 표준프레임워크를 사용한 웹 어플리케이션에 로딩된 전자정부에서 제공하는 클래스를 확인하고 분석 가능하다. 사용자에게 실시간으로 프레임워크 클래스의 로딩 정보를 보여준다.\\
프레임워크 클래스 모니터링 정보는 RMI 통신을 사용하여, context-monitoring.xml에 정의된 MBean Export Name이 사용된다.\\


====== 기능 설명 ======
  - 프레임워크 클래스 모니터링 정보 목록 : 전자정부 표준프레임워크를 통해 로딩된 클래스 모니터링 정보 목록을 보여준다.
  - 프레임워크 클래스에 조회되는 클래스나 인터페이스는 패키지명이 egovframework로 시작되는 클래스나 인터페이스만을 조회해서 보여준다. 조회하고자 하는 클래스나 인터페이스를 추가하고자 한다면 mbean.properties의 Filters 항목에 추가시켜준다.
\\

====== 화면 설명 ======

===== 프레임워크 클래스 모니터링 정보 목록 =====
전자정부 프레임워크 클래스를 모니터링 하는 화면이다. 검색조건은 서버명, 컨테이너명이며 클래스 명으로도 검색이 가능하다.

{{:egovframework:oe2:가이드:프레임워크:모델4.jpg|}}


  * 검색 : 선택 또는 입력한 검색조건을 기반으로 서비스 목록을 조회한다.

==== 화면 검색 용어 설명 ====
  * 서버 : 로딩된 클래스가 속하는 서버 명
  * 컨테이너 : 로딩된 클래스가 속하는 컨테이너 명
  * 클래스 : 로딩된 클래스의 패키지 및 클래스 명
  * 클래스 타입 : 클래스인지 인터페이스 인지 여부
  * 등록시간 : 클래스가 로딩된 시간

==== 화면 상세 정보 ====
  * 모니터링 대상인 어플리케이션의 환경에서 IBM JDK를 사용한 경우에는 클래스의 정보를 모니터링 할 수 없다. 이 경우, SUN JDK를 사용하기를 귄장한다.
  * 서버 별, 컨테이너 별, 클래스 명 별 전자정부 프레임워크가 적용된 클래스의 모니터링 정보를 사용자에게 제공한다.
