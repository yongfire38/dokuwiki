====== 개요 ====== 
프레임워크 적용 HTTP 서비스 목록 및 상세페이지는 HTTP 서비스 레이어의 요약을 제공합니다.

\\
====== 기능 설명 ======
  - HTTP 서비스 목록 : HTTP 서비스의 수행정보의 목록을 보여줍니다.
  - HTTP 서비스 현황 상세정보 : HTTP 서비스의 수행정보를 상세하게 보여줍니다.

\\

  * 등록 : 새로운 서비스 정보를 등록하기 위한 화면으로 이동한다.
