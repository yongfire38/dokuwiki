====== 개요 ====== 
쓰레드 모니터링 페이지는 쓰레드 현재 활성화된 수, 대기수, 총 쓰레드 수 등 전반적인 쓰레드 정보를 제공한다.

WAS(Web Application Server)에서 보여주는 쓰레드의 상세 정보를 확인가능하며, 쓰레드 정보 통계을 통해 쓰레드 모니터링 정보 목록 화면의 그래프 보다 상세한 데이터를 그래프로 확인 가능하다. 특정 시점과 기간에 따라 쓰레드 정보를 통계성으로 알려준다.

\\

====== 기능 설명 ======
  - 쓰레드 모니터링 정보 목록 : 쓰레드 모니터링 정보 목록을 보여준다. WAS(Web Application Server)에서 보여주는 쓰레드의 상세 정보를 확인가능하다.
  - 쓰레드 정보 통계 : 쓰레드 모니터링 정보 목록  화면의 그래프 보다 상세한 데이터를 그래프로 확인 가능하다. 특정 시점과 기간에 따라 쓰레드 정보를 통계성으로 보여준다.

\\

====== 화면 설명 ======

===== 쓰레드 모니터링 정보 목록 =====
쓰레드 정보를 나타내는 화면이다. 검색조건은 서버명과 컨테이너명이다.

{{:egovframework:oe2:가이드:프레임워크:쓰레드2.jpg|}}


  * 검색 : 선택 또는 입력한 검색조건을 기반으로 모니터링 목록을 조회한다.
  * 리프레쉬: 최신 모니터링 정보를 재전송한다.

==== 화면 검색 용어 설명 ====
  * 시간 : 마지막으로 수집된 쓰레드의 수집 시간
=== Tomcat(톰켓) 과 JBoss(제이보스) ===
  * 현재 쓰레드 수 : 현재 활성중인 쓰레드 수
  * 살아있는 쓰레드 수 : 현재 전체 쓰레드 수
  * 최대 쓰레드 수 : 웹 어플리케이션 서버(WAS) 상에 설정된 최대 쓰레드 수
  * Busy 쓰레드 수 : Busy 쓰레드 수
=== JEUS(제우스) ===
  * 활성중인 쓰레드 수 : 활성중인 쓰레드의 수. Thread Count는 그래프에서 사용하는 용어
  * 총 쓰레드 수 : 웹 어플리케이션 서버(WAS)상에 설정되어진 총(max) 쓰레드 갯수
  * 대기중인 큐 수 : 현재 큐에 대기 건 수
=== WebLogic(웹로직) ===
  * 현재 쓰레드 수 : 현재 활성화된 쓰레드 수  
  * 전체 쓰레드 수 : 전체 쓰레드 수
  * 완료된 요청 수 : 종료된 쓰레드 수
  * Idle 쓰레드 수 : 기동되지 않는 쓰레드 수
  * Hodding 쓰레드 수 : 가용 가능한 쓰레드 수
  * 큐 길이 : 큐의 전체 용량. Queue Size란 그래프에서 사용하는 용어
  * 큐 용량 : 큐의 설정 량
  * 준비된 쓰레드 수 : 대기중인 쓰레드 수
  * 매초 평균 완료 수 : 매초 완료된 쓰레드의 평균 수

==== 화면 상세 정보 ====
  * 어드민에서 설정하는 JNDI 포트를 통하여 얻어오는 정보이며, 웹 어플리케이션 서버(WAS)를 통해서 데이터를 얻어온다. 따라서 해당 WAS 콘솔 상의 쓰레드 정보 에서도 확인할 수 있다.
  * 쓰레드 정보에 사용된 용어는 해당 WAS(Web Application Server)에서 사용중인 용어를 번역하여 사용되었다.
  * 서버별, 컨테이너 별 쓰레드 모니터링 정보를 사용자에게 제공한다.
  * 당일 쓰레드 모니터링 정보가 있는 컨테이너만 화면에 조회된다.


\\

===== 쓰레드 정보 통계 =====
쓰레드의 정보를 통계성으로 보여주는 화면이다. 검색조건은 서버명, 컨테이너명, 기간 이다.

{{:egovframework:oe2:가이드:프레임워크:statsthread.jpg|}}

==== 화면 상세 정보 ====
  * 검색 : 서버명, 컨테이너명, 기간구분, 기간 선택 후 검색버튼을 클릭하여 검색조건에 해당하는 챠트를 조회한다. \\
  * 분 : 분 선택 시 60분 이내의 데이터만 조회 가능함 
  * 시 : 시 선택 시 24시간 이내의 데이터만 조회 가능함 
  * 일 : 일 선택 시 60일 이내의 데이터만 조회 가능함 
  * 월 : 월 선택 시 24개월 이내의 데이터만 조회 가능함 

\\

==== 참고사항 ====
  * 선택한 기간 중 데이터가 없는 날짜 및 시간은 표기되지 않음
  * 챠트 X축의 날짜 및 시간의 처음값은 조회한 기간보다 앞선 일자 및 시간으로 표기됨
