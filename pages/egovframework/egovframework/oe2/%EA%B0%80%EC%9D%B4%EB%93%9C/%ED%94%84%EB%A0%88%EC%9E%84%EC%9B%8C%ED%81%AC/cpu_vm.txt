====== 개요 ====== 
CPU 정보 모니터링 화면은 현재 CPU사용현황과 일정기간동안의 CPU사용추이정보를 제공한다.

CPU 사용량 수치와 프로세스 수를 확인할 수 있으며, 전체 프로세스의 평균 CPU 사용량 그래프를 통해 추이를 관측 가능하다.\\
실시간으로 CPU의 변화량을 관측할 수 있으며, 설정해 놓은 임계치 초과 여부를 즉각적으로 확인할 수 있다. \\
CPU 모니터링 데이터는 어드민에 설정해 놓은 RMI 포트를 통하여 정보를 가져오며, JConsole 상에서도 동일한 CPU 정보를 얻을 수 있다.\\

\\

====== 기능 설명 ======
  - CPU 정보 목록: CPU 모니터링 실시간 현황을 그래프와 데이터 수치로 보여준다. 모니터링 되고 있는 전체 컨테이너의 정보 목록을 볼 수 있다.

\\
====== 화면 설명 ======

===== CPU 정보 목록 =====
CPU 정보를 보여주는 화면이다. 검색조건은 서버명과 컨테이너 명이다.

{{:egovframework:oe2:가이드:프레임워크:cpu화면.jpg|}}


  * 검색 : 선택 또는 입력한 검색조건을 기반으로 모니터링 목록을 조회한다.
  * 리프레쉬 : 최신 모니터링 정보를 재전송한다.

==== 화면 검색 용어 설명 ====
  * 시간 : 마지막으로 CPU 데이터가 수집되었던 때의 시간
  * CPU 사용량 : 전체 CPU 사용량을 전체 프로세스로 나눈 평균 CPU 사용량. CPU Usage는 그래프에 표기된 명칭
  * 프로세스 수 : 전체 CPU를 사용하는 프로세스의 갯수

==== 화면 상세 정보 ====

  * JFreeChar를 이용하여 'CPU 사용량(CPU USAGE)' 대한 시간 별 퍼센트 그래프가 그려졌다.
  * 'CPU 사용량'은 전체 프로세스에 대한 CPU 사용량의 평균값을 의미한다.
  * 컨테이너 별 CPU 모니터링 정보를 사용자에게 제공한다.
