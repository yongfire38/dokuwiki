====== 개요 ====== 
표준프레임워크 활용 시 커스터마이징에 의한 기능 및 아키텍처의 표준 준수 수준을 확인, 점검 받기 위한 호환성지원신청 기능을 제공한다. 

====== 기능 설명 ====== 
  * 호환성지원신청 목록 : 사용자는 [[egovframework:mgt:호환성지원신청#호환성지원신청 목록]] 화면을 통하여 자신이 신청한 호환성지원 목록을 조회하고, 검색할 수 있다.
  * 호환성지원신청 등록 : 사용자는 [[egovframework:mgt:호환성지원신청#호환성지원신청 등록]] 화면을 통하여 전자정부 표준프레임워크를 적용한 시스템의 상세 내용을 입력하고, 전자정부 표준프레임워크 관리환경 시스템에 해당 호환성지원신청을 등록한다.
  * 호환성지원신청 상세조회 : 호환성지원신청 목록에서 하나의 신청 건을 선택하여 그에 대한 상세 정보를 조회한다. 해당 화면에서 다음의 기능을 처리할 수 있다.
    * 상세조회 중인 신청 수정
    * 상세조회 중인 신청 삭제
    * 신청 목록으로의 이동
  * 호환성지원신청 수정 : 신청하지 않고 임시저장된 신청 건을 [[egovframework:mgt:호환성지원신청#호환성지원신청 수정]] 화면을 통해 수정한다.
  * 호환성지원신청 항목 : 사용자는 [[egovframework:mgt:호환성지원신청#호환성지원신청 항목]] 화면을 통하여 전자정부 표준프레임워크 호환성 점검항목을 선택하고, 점검에 대한 반영도 및 의견을 입력하고, 신청한다.


====== 화면 설명 ====== 
===== 호환성지원신청 목록 =====
사용자가 등록한 호환성지원신청 건들의 목록을 조회한다.\\
{{:egovframework:mgt:fci_req02.gif|호환성지원신청 목록 화면}}
  * 검색조건 : 다음과 같은 항목의 검색조건을 선택하고 검색어를 입력한다. 검색조건을 선택하지 않을 경우, 등록된 모든 호환성지원신청건 들을 보여준다.
    * 호환성분석대상명 : 입력한 검색어를 포함하는 호환성분석대상명을 가진 호환성지원신청건 들을 검색한다.
    * 처리상태 : 입력한 검색어를 포함하는 처리상태를 가진 호환성지원신청건 들을 검색한다.
  * 검색 : 입력한 검색조건을 기준으로 호환성지원신청 목록을 검색한다.\\

  * 순번 : 주어진 검색조건을 기반으로한 호환성지원신청 목록의 순번이다.
  * 호환성분석대상명 : 호환성지원신청된 시스템명이다.
  * 사업자(업체)명 : 호환성지원신청된 시스템의 사업자 또는 업체의 이름이다.
  * 처리상태 : 호환성지원신청된 시스템 지원처리 상태이다.

  * 등록 : 새로운 호환성지원신청의 등록을 위하여 [[egovframework:mgt:호환성지원신청#호환성지원신청 등록]] 화면으로 이동한다.

===== 호환성지원신청 등록 =====
새로운 호환성지원신청을 작성하고 시스템에 등록한다.\\
{{:egovframework:mgt:fci_req01.gif|호환성지원신청 등록}}
  * **호환성분석대상명** : 호환성지원의 대상이 되는 시스템명을 입력한다. 필수입력 항목이다.
  * **호환성분류** : 호환성지원의 대상이 되는 시스템의 표준프레임워크의 활용정도 및 기능에 따라 구분 선택한다. 다음의 항목들을 선택할 수 있다.
    * 어플리케이션 : 표준프레임워크 및 공통컴포넌트를 활용하여 개발된 업무기능을 제공하는 웹 어플리케이션 
    * 파생프레임워크
      * 표준프레임워크에서 제공하지 않는 기능을 개발 및 추가하여 특정업무 개발시에 필요한 기반을 제공하는 개발프레임워크
      * 확장(파생)되어 제공하는 서비스 또는 기능이 표준프레임워크 활용가이드를 준수하는 프레임워크
    * 솔루션
      * 웹 시스템형 솔루션 : WAS, 웹서버, DB 등이 일체화 되어 웹형태의 업무기능을 서비스하는 솔루션(PMS , Service Desk 등)
      * 웹 컴포넌트형 솔루션 : 표준프레임워크를 활용한 시스템의 기능 구현을 지원하는 컴포넌트형 솔루현(UI 컴포넌트, 암호화 컴포넌트 등)
      * 웹 기반 솔루션 : 웹시스템을 구축하기 위하여 필요한 기반기능을 제공하는 솔루션 (웹서버, WAS, DB 등)
    * 공통컴포넌트 : 표준프레임워크를 활용한 시스템 또는 프레임워크 상에서 구동되면서 사용자가 필요로 하는 기능을 제공하는 컴포넌트 
  * **사업자(업체)명** : 호환성지원의 대상이 되는 시스템의 사업자 또는 업체의 이름을 입력한다. 필수입력 항목이다.
  * **호환성분석신청자** : 호환성지원의 대상이 되는 시스템의 분석신청담당자로 호환성지원신청자명으로 자동입력된다.
  * **신청자전화번호** : 호환성지원의 대상이 되는 시스템의 호환성분석신청자에게 연락가능한 전화번호를 입력한다. 필수입력 항목이다.
  * **신청자팩스번호** : 호환성지원의 대상이 되는 시스템의 호환성분석신청자에게 연락가능한 팩스번호를 입력한다. 필수입력 항목이다.
  * **신청자이메일** : 호환성지원의 대상이 되는 시스템의 호환성분석신청자에게 연락가능한 이메일 주소를 입력한다. 필수입력 항목이다.
  * **분석대상설명** : 호환성지원의 대상이 되는 시스템의 사업내용 및 개요을 입력한다. 필수입력 항목이다.
  * **버전** : 호환성지원의 대상이 되는 시스템의 버전을 입력한다. 필수입력 항목이다.
  * **분석날짜** : 호환성지원의 대상이 되는 시스템의 호환성점검을 실시할 날짜를 입력한다. 필수입력 항목이다.
  * **분석장소** : 호환성지원의 대상이 되는 시스템의 호환성점검을 실시할 장소의 주소를 입력한다. 필수입력 항목이다.
  * **첨부파일** : 호환성지원신청에 필요한 파일들을 첨부할 수 있다. 최대 1개까지 첨부가능하다.\\

  * 저장 : 작성한 호환성지원신청 내용을 시스템에 등록, 저장한다. 저장 후, [[egovframework:mgt:호환성지원신청#호환성지원신청 목록]]  화면으로 이동한다.
  * 목록 : 작성 중인 호환성지원신청 등록을 취소하고 호환성지원신청 목록을 조회한다. [[egovframework:mgt:호환성지원신청#호환성지원신청 목록]] 화면으로 바로 이동한다.


===== 호환성지원신청 상세조회 =====
호환성지원신청 상세조회 화면은 호환성지원신청 등록 내용 조회할 수 있으며 지원결과에 따라 결과내용이 출력된다.

==== 호환성지원신청정보 ====
호환성지원신청 목록에서 선택한 호환성지원신청 건에 대한 신청 정보를 표시한다. 화면 상의 **__다음__** 버튼, **__수정__** 버튼과 **__삭제__** 버튼은 해당 호환성지원신청 건의 처리 상태가 **__임시저장 단계__**일 경우에만 보여진다.\\
즉, 접수 담당자가 해당 호환성지원신청 건을 확인하고 접수하기 전에만 호환성지원신청의 수정 또는 삭제가 가능하다.

{{:egovframework:mgt:fci_req03.gif|호환성지원신청 상세조회 화면}}
  * **다음** : 해당 호환성지원신청건의 항목 등록을 한다. 클릭 시, [[egovframework:mgt:호환성지원신청#호환성지원신청 항목]]  화면으로 이동한다.
  * **수정** : 해당 호환성지원신청 건을 수정한다. 클릭 시, [[egovframework:mgt:호환성지원신청#호환성지원신청 수정]] 화면으로 이동한다.
  * **삭제** : 해당 호환성지원신청 건을 삭제한다. 삭제 후, [[egovframework:mgt:호환성지원신청#호환성지원신청 목록]] 화면으로 이동한다.
  * 목록 : 이전 화면인 [[egovframework:mgt:호환성지원신청#호환성지원신청 목록]] 화면으로 이동한다.

===== 호환성지원신청 수정 =====
기존에 신청한 호환성지원신청 건을 수정한다.\\
해당 호환성지원신청 건의 처리 단계가 **__임시저장__** 단계일 경우, [[egovframework:mgt:호환성지원신청#호환성지원신청 상세조회]]  화면에서 **수정** 버튼을 클릭하여 접근할 수 있다.\\
{{:egovframework:mgt:fci_req04.gif|호환성지원신청 수정화면}}
  * **저장** : 수정한 내용을 시스템에 저장하여 반영한다. 신청정보가 수정되면 상세항목은 초기화된다.
  * 목록 : 수정 중인 사항을 취소하고, [[egovframework:mgt:호환성지원신청#호환성지원신청 목록]] 화면으로 돌아간다.

===== 호환성지원신청 항목 =====
호환성지원신청을 임시저장후 세부항목을 시스템에 등록한다.\\
{{:egovframework:mgt:fci_req05.gif|호환성지원신청 항목 화면}}
  * **선택** : 호환성지원시 지원대상이 되는 시스템에서 표준프레임워크를 적용한 항목을 체크한다. 필수입력 항목이다.
  * 항목명 : 표준프레임워크의 호환성점검항목명이다.
  * 항목설명 : 표준프레임워크의 호환성점검항목에 대한 설명이며, 지원대상 시스템에서 지원점검의 기준이 된다.
  * **신청요약** : 지원대상 시스템이 점검항목에 대해 적용한 내용, 이슈사항등을 입력한다.\\

  * 신청 : 작성한 호환성지원신청 내용을 시스템에 등록, 저장한다. 상태코드는 **__신청__**으로 되며 저장 후, [[egovframework:mgt:호환성지원신청#호환성지원신청 목록]] 화면으로 이동한다.
  * 저장 : 작성한 호환성지원신청 내용을 시스템에 등록, 저장한다. 상태코드는 **__임시저장__**으로 되며 저장 후, [[egovframework:mgt:호환성지원신청#호환성지원신청 목록]] 화면으로 이동한다.
  * 목록 : 작성 중인 호환성지원신청 항목을 취소하고 호환성지원신청의 목록을 조회한다. [[egovframework:mgt:호환성지원신청#호환성지원신청 목록]]화면으로 바로 이동한다.
