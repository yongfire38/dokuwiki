====== 개요 ====== 
전자정부 표준프레임워크에서 제공되는 실행환경 구성요소를 소개하고 상세가이드를 검색할 수 있는 서비스 기능을 제공한다. 

====== 기능 설명 ====== 
  * 적용지원검색 : 사용자는 [[egovframework:mgt:적용지원검색#적용지원검색]] 화면을 통하여 전자정부 표준프레임워크 실행환경 구성요소를 선택하고, 구성요소의 상세항목을 검색한다.
  * 적용지원상세검색 : 전자정부 표준프레임워크 실행환경 구성요소를 이루는 상세항목을 [[egovframework:mgt:적용지원검색#적용지원상세검색]] 화면을 통하여 조회하며 상세항목의 가이드를 검색한다.
  * 적용지원검색결과 : 사용자가 선택한 구성요소 항목에 대한 가이드는 [[egovframework:mgt:적용지원검색#적용지원검색결과]] 화면을 통해 조회한다.

====== 화면 설명 ====== 
===== 적용지원검색 =====
표준프레임워크 실행환경 구성요소를 선택하고 상세항목을 검색한다.\\
{{:egovframework:mgt:aps_req01.gif|적용지원검색}}
  * **신청분류선택** : 선택한 경우 아래 항목이 활성화되고 선택할 수 있다
    * 포탈시스템 : 포탈사이트의 기능을 가지는 시스템에서 적용할 수 있는 상세항목을 검색한다.
    * 행정시스템 : 행정업무처리 기능을 가지는 시스템에서 적용할 수 있는 상세항목을 검색한다.
    * 금융시스템 : 금융업무처리 기능을 가지는 시스템에서 적용할 수 있는 상세항목을 검색한다.
    * 홈페이지 : 홈페이지의 기능을 가지는 시스템에서 적용할 수 있는 상세항목을 검색한다.
  * **신청분류선택** : 선택하지 않은 경우 아래 항목이 활성화되고 선택할 수 있다
    * 공통기반
    * 화면처리
    * 업무처리
    * 데이터처리
    * 연계통합

  * 다음 : 선택한 항목에 따른 표준프레임워크 실행환경 구성요소의 상세항목 검색 [[egovframework:mgt:적용지원검색#적용지원상세검색]] 화면으로 이동한다.

===== 적용지원상세검색 =====
표준프레임워크 실행환경 구성요소의 상세항목을 선택하고 가이드를 검색한다.\\
{{:egovframework:mgt:aps_req02.gif|적용지원상세검색}}
  * **상세항목선택** : 선택한 경우 선택한 상세항목의 가이드를 검색할 수 있다

  * 결과 : 선택한 상세항목의 가이드를 조회할 수 있는 [[egovframework:mgt:적용지원검색#적용지원검색결과]] 화면으로 이동한다.


===== 적용지원검색결과 =====
선택된 상세항목의 개요와 가이드를 조회할 수 있다.\\
{{:egovframework:mgt:aps_result01.gif|적용지원검색결과}}
  * 자세히보기 : 표준프레임워크 구성요소 상세항목의 Wiki 가이드를 확인할 수 있다.

  * 재검색 : 검색결과를 취소하고, [[egovframework:mgt:적용지원검색#적용지원검색]] 화면으로 돌아간다.
