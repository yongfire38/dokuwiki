====== 개요 ====== 
사용자가 속한 프로젝트 단위의 SR 목록과 상세 정보를 조회하는 기능을 제공한다. 

====== 기능 설명 ====== 
  * SR 목록 : 사용자는 자신이 속해 있는 프로젝트에서 요청한 SR 목록을 조회하고, 검색할 수 있다. 프로젝트 목록을 변경하여 프로젝트를 바꾸어 선택하거나, 검색 조건을 달리하여 원하는 조건에 맞는 SR 목록을 조회할 수 있다.
  * SR 상세조회 : SR목록에서 하나의 SR 건을 선택하여 그에 대한 상세 정보를 조회한다. 해당 화면에서 다음의 기능을 처리할 수 있다.
    * 상세조회 중인 SR 수정
    * 상세조회 중인 SR 삭제
    * 완료된 SR 건의 경우 만족도 처리
    * SR 목록으로의 이동

====== 화면 설명 ====== 
===== SR 목록 =====
로그인한 사용자의 소속 프로젝트에서 등록한 SR 요청 건들의 목록을 조회한다.\\
{{:egovframework:mgt:mgt_srm_req_srList01.png|SR 목록 화면}}
  * 프로젝트 : 사용자가 속한 프로젝트의 목록을 나타내고 변경할 수 있도록 한다. 기본값은 사용자의 최근 프로젝트가 선택된다.
  * 검색조건 : 다음과 같은 항목의 검색조건을 선택하고 검색어를 입력한다. 검색조건을 선택하지 않을 경우, 선택된 프로젝트의 전체 SR 요청건 들을 보여준다.
    * 요청자 ID : 입력한 검색어를 포함하는 요청자ID를 가진 SR 요청건 들을 검색한다.
    * 제목 : 입력한 검색어를 포함하는 SR요청 제목을 가진 SR 요청건 들을 검색한다.
    * 내용 : 입력한 검색어를 포함하는 SR요청 요청내용을 가진 SR 요청건 들을 검색한다.
  * 검색 : 선택한 프로젝트와 입력한 검색조건을 기준으로 SR 목록을 검색한다.\\

  * 번호 : 주어진 검색조건을 기반으로한 SR 목록의 순번이다.
  * 요청번호 : SR의 고유한 식별 번호로써, **SR+년도(4자리)+월(2자리)+ '_' + 5자리 숫자**로 구성된다.
  * 제목 : SR의 제목을 표시한다. 제목이 너무 길어 지정된 제한길이를 초과하면, 초과한 단어에 대해서 '...'으로 표기하여 보여준다. 제목을 클릭하여, 해당 SR의 상세 정보를 확인할 수 있다.
  * 처리상태 : 현재 SR의 처리 상태를 표시한다. SR 목록에서는 다음의 3단계로 표시된다.
    * 요청 : 해당 SR이 등록만 되어 있고, 아직 담당자의 처리 과정이 시작되지 않은 건 임을 보여준다.
    * 처리중 : 해당 SR이 접수되어 처리 과정을 거치고 있고, 현재 진행 중인 건 임을 보여준다.
    * 완료 : 처리 담당자에 의해 해당 SR이 종료되었음을 보여준다. 
  * 프로젝트 : 해당 SR을 등록한 프로젝트를 나타낸다.
  * 요청자ID : 해당 SR을 등록한 사용자의 ID를 나타낸다.
  * 요청일 : 해당 SR을 등록한 날짜를 표시한다.\\

  * 등록 : 새로운 SR의 등록을 위하여 [[egovframework:mgt:SR 요청]] 화면으로 이동한다.

===== SR 상세조회 =====
SR 상세조회 화면은 조회하는 대상 SR 건의 처리 상태에 따라, 아래의 4개 화면들 모두로 구성되거나, 일부만 선택적으로 구성되어 사용자의 웹 브라우저에 출력된다.

==== SR 요청정보 ====
SR 목록에서 선택한 SR 건에 대한 요청 정보를 표시한다. 화면 상의 **__수정__** 버튼과 **__삭제__** 버튼은 해당 SR 건의 처리 상태가 **__요청 단계__**일 경우에만 보여진다.\\
즉, 처리 담당자가 해당 SR 건을 확인하고 처리하기 전에만 SR의 수정 또는 삭제가 가능하다.

{{:egovframework:mgt:mgt_srm_req_srdetail01.png|SR 상세조회 화면(SR 요청정보)}}
  * 요청번호 : 해당 SR의 고유 식별 번호
  * 요청유형 : SR을 요청하게 된 동기 또는 목적 유형으로, 다음 항목 중의 하나가 표시된다.
    * 단순문의/장애/오류/변경요청/기타
  * 요청대상 : SR의 대상이 되는 전자정부 표준프레임워크의 구성요소로, 다음 항목 중의 하나가 표시된다.
    * 개발환경/실행환경/공통컴포넌트/기타
  * 요청자 : 해당 SR을 등록한 사용자의 소속 프로젝트와 사용자 이름을 나타낸다.
  * 요청일시 : 해당 SR이 등록된 날짜와 시간을 표시한다.
  * 요청제목 : 해당 SR의 요청제목
  * 요청내용 : 해당 SR의 상세 요청 내역을 표시한다.
  * 첨부파일 : 해당 SR에 첨부파일이 있을 경우, 이를 표시하고 다운로드 할 수 있는 URL 링크가 걸린다.\\

  * **수정** : 해당 SR 요청 건을 수정한다. 클릭 시, [[egovframework:mgt:SR 요청#SR 수정]] 화면으로 이동한다.
  * **삭제** : 해당 SR 건을 삭제한다. 삭제 후, [[egovframework:mgt:SR 목록]] 화면으로 이동한다.
  * 목록 : 이전 화면인 [[egovframework:mgt:SR 목록]] 화면으로 이동한다.

==== SR 처리결과 ====
요청한 SR에 대한 처리결과 정보를 표시한다. SR의 처리 상태가 가장 마지막 단계인 **__완료 단계__**일 경우에만 보여진다.\\
{{:egovframework:mgt:mgt_srm_req_srdetail02.png|SR 상세조회 화면(SR 처리결과)}}
  * 처리자 : 해당 SR의 최종 처리자 이름이 표시된다.
  * 처리일시 : 해당 SR이 처리된 날짜와 시간을 표시한다.
  * 처리유형 : 해당 SR이 최종 처리된 유형으로, 다음 항목 중의 하나가 표시된다.
    * 1선처리/2선처리/현장지원처리/변경처리/반려
  * 처리내용 : 최종 처리자가 해당 SR에 대해 작성한 상세 처리결과 내용을 표시한다.

==== SR 처리 만족도 ====
최종 완료된 SR 건에 대한 처리 만족도를 선택하거나 확인할 수 있으며, SR의 처리 상태가 가장 마지막 단계인 **__완료 단계__**일 경우에만 표시되고 평가할 수 있다.\\
**서비스**, **기술**, **신속도**의 3가지 관점에서 만족도를 별(★)점을 이용하여 평가할 수 있다.\\
라디오 버튼을 이용하여 각각의 만족도를 선택한 후, **확인** 버튼을 클릭하여 시스템에 반영한다.\\
{{:egovframework:mgt:mgt_srm_req_srdetail03.png|SR 상세조회 화면(SR 처리 만족도)}}
  * 확인 : 3개의 만족도가 모두 평가될 때까지 반복해서 변경/확인처리 할 수 있으며, 서비스/기술/신속도 만족도를 모두 선택하게 되면 해당 버튼은 사라진다.

==== SR 처리상태 ====
해당 SR의 현재까지의 처리 상태를 확인할 있다.\\
{{:egovframework:mgt:mgt_srm_req_srdetail04.png|SR 상세조회 화면(SR 처리상태)}}\\
지나간 단계는 옅은 하늘색으로 표시되며, 현재 완료되지 않은 최종 상태의 경우 노란색으로, 현재 완료된 최종 상태의 경우 파란색으로 표시된다.
