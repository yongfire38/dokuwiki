====== 개요 ====== 
관리환경을 사용하기 위하여 현재 진행중인 프로젝트를 대상으로 소속원으로 신청을 하고, 각 프로젝트 책임자로부터 승인되어 소속원으로 된 후 해당 프로젝트의 소속원으로 시스템에 로그인 할수 있도록 하는 기능이다. 포탈화면에서 SR요청을 위해 관리환경 접속하기 위해서는 프로젝트 소속신청에 의해서 가능하다. 

====== 기능 설명 ======
  - 프로젝트소속신청 대상목록 : 프로젝트 소속원으로 신청할 수 있는 대상 프로젝트 목록을 검색 조회한다.(현재 진행중인 프로젝트)
  - 프로젝트 소속신청 등록 : 선택한 프로젝트요약정보를 확인하고 소속원으로 신청하거나 신청을 취소한다.(프로젝트 일반 사용자인 경우 사용하는 화면)
  - 프로젝트 소속신청 승인 : 프로젝트에 소속신청한 사용자 목록을 확인하여 소속원으로 승인하거나 취소한다.(프로젝트 책임자인 경우 사용하게 되는 화면)

====== 화면 설명 ======

===== 프로젝트소속신청 대상목록 =====
프로젝트 소속원으로 신청할 수 있는 대상 프로젝트 목록을 조회한다. 이 목록 페이지를 통해서 포탈화면에서 관리환경으로 이동하게 된다. 최초 로그인시 입장가능한 프로젝트 정보가 유일하게 1건인 경우는 자동으로 유일한 프로젝트 정보의 사용자로 로그인 처리되어 관리환경 첫 화면으로 이동하게 된다. 입장 가능한 프로젝트 정보가 없거나 2건 이상인 경우는 목록이 출력되며 프로젝트 정보를 신규로 등록하거나 소속신청절차를 거쳐 관리환경으로 입장(로그인)한다.

{{:egovframework:mgt:프로젝트소속신청대상목록_.gif|}}

  * 프로젝트 명 선택 : 프로젝트의 상세정보 화면으로 이동한다. (일반사용자:신청화면으로 이동, 책임자:승인화면으로 이동)
  * 입장 : 프로젝트 상태가 승인이면서 소속원으로 등록되어 있는 경우 입장 링크를 클릭하여 해당 프로젝트를 대상으로 로그인 할 수 있다. 최초 로그인시 입장가능한 프로젝트 정보가 유일하게 1건인 경우는 자동으로 유일한 프로젝트 정보의 사용자로 로그인 처리되어 관리환경 첫 화면으로 이동하게 된다.
  * 프로젝트 등록 : 서비스요청을 하는 프로젝트의 정보를 입력하고 등록한다.(프로젝트 신청의 등록과 동일함)
  * 종료 : 화면을 종료한다.


===== 프로젝트 소속신청 등록  =====
프로젝트소속신청 대상목록에서 프로젝트명을 클릭하면 해당 프로젝트의 요약정보화면(신청화면 또는 승인화면)으로 이동한다. 이때 사용자가 일반 사용자인 경우는 해당프로젝트에 대한 소속신청화면으로 이동하게된다. 소속신청 화면에서는 해당 프로젝트의 소속원으로 신청하거나 신청내역을 취소할수 있다. 단, 책임자가 프로젝트 소속원으로 이미 승인한 경우는 즉시 취소할수 없고 소속원에서 먼저 제외한 후 소속신청을 취소할 수 있다.

{{:egovframework:mgt:프로젝트요약정보.gif|}}

  * 프로젝트 소속신청 : 현재 화면상에 요약정보가 출력된 프로젝트의 소속원으로 신청한다.
  * 프로젝트 소속신청 취소 : 현재 화면상에 요약정보가 출력된 프로젝트의 소속원으로 신청한 내역을 취소한다.
  * 목록 : 프로젝트 소속신청 대상목록 조회 화면으로 이동한다. 

===== 프로젝트 소속신청 승인  =====
프로젝트소속신청 대상목록에서 프로젝트명을 클릭하면 해당 프로젝트의 요약정보화면(신청화면 또는 승인화면)으로 이동한다. 이때 사용자가 프로젝트 책임자인 경우는 해당프로젝트에 대한 소속원 승인화면으로 이동하게된다. 소속신청 승인화면에서는 해당 프로젝트의 소속원으로 신청한 사용자를 프로젝트의 소속원으로 등록하거나 현재 소속원을 제외(신청자 목록으로 이동)시킬 수 있다.

{{:egovframework:mgt:프로젝트소속신청승인.gif|}}

  * 프로젝트 소속신청 승인 : 현재 화면상에 출력된 소속원 목록과 신청자 목록으로 프로젝트 소속원 정보를 수정한다.
  * 목록 : 프로젝트 소속신청 대상목록 조회 화면으로 이동한다.
