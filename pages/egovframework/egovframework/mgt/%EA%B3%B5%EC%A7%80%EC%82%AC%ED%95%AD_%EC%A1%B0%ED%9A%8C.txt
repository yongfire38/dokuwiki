====== 개요 ====== 
관리환경 공지사항 목록과 상세 정보를 조회하는 기능을 제공한다. 

====== 기능 설명 ====== 
  * 공지사항 목록 : 사용자는 관리환경 공지사항 목록을 조회하고, 검색할 수 있다. 
  * 공지사항 상세조회 : 공지사항 목록에서 하나의 공지사항 건을 선택하여 그에 대한 상세 정보를 조회한다. 해당 화면에서 공지사항 목록으로의 이동 기능을 처리할 수 있다.

====== 화면 설명 ====== 
===== 공지사항 목록 =====
관리환경 공지사항 목록을 조회한다.\\
{{:egovframework:mgt:공지사항목록.jpg|공지사항목록}}
  * 검색조건 : 다음과 같은 항목의 검색조건을 선택하고 검색어를 입력한다. 검색조건을 선택하지 않을 경우, 관리환경 전체 공지사항들을 보여준다.
    * 제목 : 입력한 검색어를 포함하는 제목을 가진 공지사항들을 검색한다.
    * 내용 : 입력한 검색어를 포함하는 내용을 가진 공지사항들을  검색한다.
    * 작성자 : 입력한 검색어를 포함하는 작성자를 가진 공지사항들을  검색한다.
  * 검색 : 입력한 검색조건을 기준으로 공지사항 목록을 검색한다.\\

  * 번호 : 주어진 검색조건을 기반으로한 공지사항 목록의 순번이다.
  * 제목 : 공지사항의 제목을 표시한다. 제목이 너무 길어 지정된 제한길이를 초과하면, 초과한 단어에 대해서 '...'으로 표기하여 보여준다. 제목을 클릭하여, 해당 공지사항의 상세 정보를 확인할 수 있다.
  * 게시시작일 : 공지사항이 유효한 시작일자를 표시한다.
  * 게시종료일 : 공지사항이 유효한 종료일자를 표시한다.
  * 작성자: 해당 공지사항을 등록한 사용자를 나타낸다.
  * 등록일 : 해당 공지사항을 등록한 날짜를 표시한다.
  * 조회수 : 해당 공지사항이 조회된 횟수를 표시한다.\\

  * 등록 : 새로운 공지사항의 등록을 위하여 공지사항 등록 화면으로 이동한다. 단 등록권한이 없는 경우 권한 오류 메시지가 표시된다.

===== 공지사항 상세조회 =====
공지사항의 상세 내용을 조회한다.\\
{{:egovframework:mgt:공지사항상세.jpg|공지사항상세}}
  * 작성자: 해당 공지사항을 등록한 사용자를 나타낸다.
  * 등록일 : 해당 공지사항을 등록한 날짜를 표시한다.
  * 제목 : 공지사항의 제목을 표시한다. 
  * 내용 : 공지사항의 내용을 표시한다.
  * 게시시작일 : 공지사항이 유효한 시작일자를 표시한다.
  * 게시종료일 : 공지사항이 유효한 종료일자를 표시한다.\\

  * 수정 : 해당 공지사항의 수정을 위하여 공지사항 수정 화면으로 이동한다. 단, 수정권한이 없는 경우 권한 오류 메시지가 표시된다.
  * 삭제 : 해당 공지사항의 삭제를 위하여 삭제확인 메시지 창이 실행되며, 확인을 누르면 삭제됨과 동시에 공지사항 목록 화면으로 이동하고, 취소를 누르면 삭제되지 않고 다시 공지사항 상제조회 화면으로 돌아간다. 단, 삭제권한이 없는 경우 권한 오류 메시지가 표시된다.
  * 목록 : 해당 공지사항의 목록 화면으로 이동한다.
