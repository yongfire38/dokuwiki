

====== 개요 ====== 
운영시스템에 대한 개선 및 업그레이드, 오류 등의 사유로 변경요청이 발생한 경우 변경요청을 위한 변경요청서의 등록/수정/상세조회/목록검색을 할 수 있는 기능을 제공한다.

====== 기능 설명 ======
  - 변경요청서 목록 : 변경요청서의 목록을 검색 조회한다.
  - 변경요청서 상세 : 변경요청서 정보를 상세 조회한다.
  - 변경요청서 등록 : 변경요청서 정보를 등록한다.
  - 변경요청서 수정 : 변경요청서 정보를 수정한다.
  - 변경요청서 요청근거 상세팝업 : 운영개선요청시스템으로부터 변경이관 처리된 경우, 운영개선요청서정보를 조회한다.

====== 화면 설명 ======

===== 변경요청서 목록 =====
변경요청서 목록은 페이지당 10건씩 조회되며 검색조건은 **업무구분**, **긴급여부**, **처리상태**, **변경요청명**, **변경요청자명**이다. 


{{:egovframework:oe1:가이드:운영개선관리:변경작업관리:변경요청서목록.jpg|}}


  * 검색 : 선택 또는 입력한 검색조건을 기반으로 변경요청서 목록을 조회한다.
  * 등록 : 변경요청정보를 등록하기 위한 화면으로 이동한다.



===== 변경요청서 상세  =====
변경요청서 목록에서 변경요청명을 클릭하면 해당 변경요청서정보를 상세 조회한다.


{{:egovframework:oe1:가이드:운영개선관리:변경작업관리:변경요청서상세.jpg|}}


  * 요청근거 : 운영개선요청시스템으로부터 변경이관처리된 경우, 운영개선요청서정보를 팝업조회한다.
  * 수정 : 변경요청서 정보를 수정할 수 있는 편집 가능한 화면으로 이동한다.
  * 삭제 : 변경요청서 정보를 삭제한다.
  * 목록 : 변경요청서 목록 조회 화면으로 이동한다. 



===== 변경요청서 등록 =====
변경요청서를 등록하는 기능을 제공한다.


{{:egovframework:oe1:가이드:운영개선관리:변경작업관리:변경요청서등록.jpg|}}


  * 저장 : 입력한 변경요청서 정보를 등록한다. 
  * 찾아보기 : 첨부파일을 선택하기 위한 팝업을 띄운다.
  * 목록 : 변경요청서 목록 조회 화면으로 이동한다. 


===== 변경요청서 수정 =====
변경요청서 상세 조회 화면에서 수정 버튼을 클릭하여 변경요청서 정보를 수정한다.


{{:egovframework:oe1:가이드:운영개선관리:변경작업관리:변경요청서_수정.jpg|}}


  * 저장 : 변경요청서 정보를 수정한다. 
  * 목록 : 변경요청서 목록 화면으로 이동한다. 
  * 찾아보기 : 첨부파일을 선택하기 위한 팝업을 띄운다.
  * 첨부파일 목록에서 파일명 옆의 X 이미지를 클릭하면 해당 첨부파일을 삭제한다.



===== 변경요청서 요청근거 상세팝업 =====
변경요청서 상세 화면에서 요청근거를 클릭하면 운영개선요청시스템으로부터 변경이관 처리된 경우, 운영개선요청서정보를 조회한다.


{{:egovframework:oe1:가이드:운영개선관리:변경작업관리:변경요청서_근거상세.jpg|}}


  * 닫기 : 화면을 종료한다.
