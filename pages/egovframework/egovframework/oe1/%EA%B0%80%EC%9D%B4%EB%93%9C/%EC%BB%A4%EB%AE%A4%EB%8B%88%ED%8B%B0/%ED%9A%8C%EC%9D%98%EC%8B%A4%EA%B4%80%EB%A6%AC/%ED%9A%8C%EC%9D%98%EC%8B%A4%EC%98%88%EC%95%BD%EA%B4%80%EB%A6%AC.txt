====== 개요 ====== 
회의실별 예약현황을 볼 수 있도록 기능을 제공한다.

====== 기능 설명 ======

  - 회의실예약현황 달력  : 회의실예약현황을 달력으로 보여준다. 
  - 회의실예약 목록  : 회의실예약 목록을 월간 리스트으로 보여준다.
  - 회의실예약 상세 : 회의실예약 상세 내용을 보여준다. 
  - 회의실예약 등록 : 회의실예약 화면으로 이동한다. 
  - 회의실예약 수정 : 회의실예약 수정 화면으로 이동한다. 

====== 화면 설명 ======

===== 회의실예약현황 달력 =====
회의실예약현황을 월별 달력으로 보여준다.   

{{:egovframework:oe1:가이드:커뮤니티:회의실관리:회의실예약_달력.jpg|}}

  * 등록 : 달력의 날짜를 클릭하면 회의실예약 화면으로 이동한다. 
  * 상세 : 회의제목을 클릭하면 회의실예약 상세 화면으로 이동한다. 

===== 회의실예약현황 리스트 =====
회의실예약 목록을 리스트로 보여준다.   

{{:egovframework:oe1:가이드:커뮤니티:회의실관리:회의실예약_리스트.jpg|}}

  * 등록 : 달력의 날짜를 클릭하면 회의실예약 화면으로 이동한다. 
  * 상세 : 회의제목을 클릭하면 회의실예약 상세 화면으로 이동한다. 

===== 회의실예약 상세  =====
회의실예약 상세 내역을 볼 수 있다. 

{{:egovframework:oe1:가이드:커뮤니티:회의실관리:회의실예약_상세.jpg|}}

  * 수정 : 회의실예약 정보를 수정할 수 있는 편집 가능한 화면으로 이동한다. 단, 자신이 등록한 회의실예약만 수정 가능하다. 
  * 목록 : 회의실예약 목록 조회 화면으로 이동한다. 
  * 삭제 : 등록한 회의실예약을 삭제 한다. 단, 자신이 등록한 회의실예약만 삭제 가능하다. 

===== 회의실예약 등록 =====
회의실예약 내용을 등록할 수 있는 기능을 제공한다.

{{:egovframework:oe1:가이드:커뮤니티:회의실관리:회의실예약_등록.jpg|}}

  * 저장 : 입력한 회의실예약 내용을 등록한다.
  * 목록 : 회의실예약 목록 조회 화면으로 이동한다.
  * 초기화 : 입력한 회의실예약 내용을 공백으로 초기화 한다. 

{{:egovframework:oe1:가이드:커뮤니티:회의실관리:회의실예약_중복_체크_팝업.jpg|}}

  * 예약중복체크: 입력한 회의실예약 시간이 중복되었는지 확인하는 팝업이 뜬다.   확인 버튼을 눌러야만 중복 검사가 완료된다. 

===== 회의실예약 수정=====
회의실예약 내용을 수정할 수 있는 기능을 제공한다.

{{:egovframework:oe1:가이드:커뮤니티:회의실관리:회의실예약_수정.jpg|}}

  * 저장 : 입력한 회의실예약 내용을 수정한다.
  * 목록 : 회의실예약 목록 조회 화면으로 이동한다.
