

====== 개요 ====== 
UI아답터관리를 위한 UI아답터의 등록/수정/상세조회/목록검색 할 수 있는 기능을 제공한다.

====== 기능 설명 ======
  - UI아답터 목록 : UI아답터의 목록을 검색 조회한다.
  - UI아답터 상세 : UI아답터 정보를 상세 조회한다.
  - UI아답터 등록 : UI아답터 정보를 등록한다.
  - UI아답터 수정 : UI아답터 정보를 수정한다.
  

====== 화면 설명 ======

===== UI아답터 목록 =====
UI아답터 목록은 페이지당 10건씩 조회되며 검색조건은 **UI아답터ID**, **UI아답터명**이다. 

{{:egovframework:oe1:가이드:아키텍쳐정보관리:업무아키텍쳐관리:ui아답터목록.jpg|}}

  * 검색 : 선택 또는 입력한 검색조건을 기반으로 UI아답터 목록을 조회한다.
  * 등록 : UI아답터를 등록하기 위한 화면으로 이동한다.



===== UI아답터 상세  =====
UI아답터 목록에서 UI아답터명을 클릭하면 해당 UI아답터의 정보를 상세 조회한다.

{{:egovframework:oe1:가이드:아키텍쳐정보관리:업무아키텍쳐관리:ui아답터상세.jpg|}}

  * 수정 : UI아답터 정보를 수정할 수 있는 편집 가능한 화면으로 이동한다.
  * 삭제 : UI아답터 정보를 삭제한다.
  * 목록 : UI아답터 목록 조회 화면으로 이동한다. 



===== UI아답터 등록 =====
UI아답터를 등록하는 기능을 제공한다.

{{:egovframework:oe1:가이드:아키텍쳐정보관리:업무아키텍쳐관리:ui아답터등록.jpg|}}

  * 저장 : 입력한 UI아답터 정보를 등록한다. 
  * 목록 : UI아답터 목록 조회 화면으로 이동한다. 


===== UI아답터 수정 =====
UI아답터 상세 조회 화면에서 수정 버튼을 클릭하여 UI아답터 정보를 수정한다.

{{:egovframework:oe1:가이드:아키텍쳐정보관리:업무아키텍쳐관리:ui아답터수정.jpg|}}

  * 저장 : 입력한 UI아답터 정보를 수정한다. 
  * 목록 : UI아답터 목록 화면으로 이동한다.
