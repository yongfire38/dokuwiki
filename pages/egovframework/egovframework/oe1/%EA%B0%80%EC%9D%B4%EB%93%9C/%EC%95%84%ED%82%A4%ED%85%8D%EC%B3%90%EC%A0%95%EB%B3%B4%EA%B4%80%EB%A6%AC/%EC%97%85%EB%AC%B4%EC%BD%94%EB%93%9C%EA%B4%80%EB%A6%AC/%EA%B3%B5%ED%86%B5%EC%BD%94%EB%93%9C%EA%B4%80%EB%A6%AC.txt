

====== 개요 ====== 
업무시스템에서 사용하는 공통코드에 대한 정보를 등록/상세조회/목록검색/상세코드등록/사용 할 수 있는 기능을 제공한다.

====== 기능 설명 ======
  - 공통코드 목록 : 공통코드의 목록을 검색 조회한다.
  - 공통코드 상세 : 공통코드 정보를 상세 조회한다.
  - 공통코드 등록 : 공통코드 정보를 등록한다.
  - 공통코드 상세 등록 : 공통코드 상세정보를 등록한다.

====== 화면 설명 ======

===== 공통코드 목록 =====
공통코드 목록은 페이지당 10건씩 조회되며 검색조건은 **코드ID**, **코드ID명**이다. 

{{:egovframework:oe1:가이드:아키텍쳐정보관리:업무코드관리:공통코드목록.jpg|}}

  * 검색 : 선택 또는 입력한 검색조건을 기반으로 공통코드 목록을 조회한다.
  * 등록 : 공통코드를 등록하기 위한 화면으로 이동한다.
  * 사용/미사용 : 공통코드의 사용여부를 수정한다. 



===== 공통코드 상세  =====
공통코드 목록에서 코드ID를 클릭하면 해당 공통코드의 정보를 상세 조회한다.

{{:egovframework:oe1:가이드:아키텍쳐정보관리:업무코드관리:공통코드상세.jpg|}}

  * 등록 : 상세조회한 공통코드의 상세코드 정보를 등록하는 화면으로 이동한다.
  * 목록 : 공통코드 목록 조회 화면으로 이동한다. 
  * 사용/미사용 : 공통코드의 사용여부를 수정한다.



===== 공통코드 등록 =====
공통코드를 등록하는 기능을 제공한다.

{{:egovframework:oe1:가이드:아키텍쳐정보관리:업무코드관리:공통코드등록.jpg|}}

  * 저장 : 입력한 공통코드 정보를 등록한다. 
  * 목록 : 공통코드 목록 조회 화면으로 이동한다. 

===== 공통상세코드 등록 =====
공통상세코드를 등록하는 기능을 제공한다.

  * 저장 : 입력한 공통상세코드 정보를 등록한다. 
  * 목록 : 공통코드 목록 조회 화면으로 이동한다.
