
====== 개요 ====== 
프레임워크환경에 관한 설정정보를 등록하고 관리하는 기능을 제공한다.
관리자권한이 부여된 사용자에게만 프레임워크환경정보에대한 등록/수정/삭제 기능이 제공된다.

====== 기능 설명 ======
  - 프레임워크환경정보 목록 : 프레임워크환경정보 변경이력목록을 검색 조회한다.
  - 프레임워크환경정보 상세 : 프레임워크환경정보를 상세 조회한다.
  - 프레임워크환경정보 등록 : 프레임워크환경정보를 등록한다.
  - 프레임워크환경정보 수정 : 프레임워크환경정보를 수정한다.
  

====== 화면 설명 ======

===== 프레임워크환경정보 목록 =====
프레임워크환경정보 목록은 페이지당 10건씩 조회되며 검색조건은 **처리일** 이다. 
관리자권한이 있는 사용자가 로그온한 경우 입력된 정보가 한 건도 없을 경우에만 등록버튼이 보여진다. 

{{:egovframework:oe1:가이드:운영정보관리:환경정보관리:프레임워크환경정보목록_등록버튼출력.jpg|}}


{{:egovframework:oe1:가이드:운영정보관리:환경정보관리:프레임워크환경정보목록.jpg|}}


  * 검색 : 선택 또는 입력한 검색조건을 기반으로 프레임워크환경정보 변경이력목록을 조회한다.
  * 등록 : 프레임워크환경정보를 신규 등록하기 위한 화면으로 이동한다.



===== 프레임워크환경정보 상세  =====
프레임워크환경정보 목록에서 Presentation Layer를 클릭하면 해당 이력건에 대한 프레임워크환경정보를 상세 조회한다.


{{:egovframework:oe1:가이드:운영정보관리:환경정보관리:프레임워크환경정보상세.jpg|}}


  * 수정 : 프레임워크환경정보를 수정할 수 있는 편집 가능한 화면으로 이동한다.
  * 삭제 : 프레임워크환경정보를 삭제한다.
  * 목록 : 프레임워크환경정보 목록 조회 화면으로 이동한다. 



===== 프레임워크환경정보 등록 =====
프레임워크환경정보를 신규 등록하는 기능을 제공한다.


{{:egovframework:oe1:가이드:운영정보관리:환경정보관리:프레임워크환경정보등록.jpg|}}


  * 저장 : 입력한 프레임워크환경정보를 등록한다. 
  * 목록 : 프레임워크환경정보 목록 조회 화면으로 이동한다. 


===== 프레임워크환경정보 수정 =====
프레임워크환경정보 상세 조회 화면에서 수정 버튼을 클릭한 경우 프레임워크환경정보를 변경 및 정정할 수 있는 기능을 제공한다.
프레임워크환경정보가 변경된 경우 변경버튼을 이용해 최신 프레임워크환경정보를 등록하고 이전 데이터는 이력으로 남긴다.


{{:egovframework:oe1:가이드:운영정보관리:환경정보관리:프레임워크환경정보수정.jpg|}}


  * 변경 : 프레임워크환경정보가 변경된 경우, 입력한 변경정보를 최신 프레임워크환경정보로 등록한다. 
  * 정정 : 등록한 정보에대한 입력오류가 있는 경우, 정정해서 입력한 프레임워크환경정보를 수정한다. 
  * 목록 : 프레임워크환경정보 목록 화면으로 이동한다.
