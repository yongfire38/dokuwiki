====== 개요 ====== 
운영정보관리자가 표준용어를 등록/수정/상세조회/목록검색 할 수 있는 기능을 제공한다.

====== 기능 설명 ======

  - 용어사전 목록 : 용어사전 목록을 검색 조회한다.
  - 용어사전 상세 : 등록된 용어사전 내용에 대한 정보를 상세 조회한다.
  - 용어사전 등록 : 용어사전 상세정보를 등록한다.
  - 용어사전 수정 : 용어사전 상세정보를 수정한다.
  - 용어사전 엑셀 등록 : 운영정보담당자가 표준용어사전 엑셀파일을 일괄 등록 한다. 이때 표준용어사전 엑셀 파일은 제공된 엑셀파일만 사용 가능하다. 
  - 용어사전 엑셀 다운로드 : 등록된 용어사전 목록을 엑셀파일로 다운로드 받는다. 

====== 화면 설명 ======

===== 용어사전 목록 =====
용어사전 목록은 페이지당 10건씩 조회되며 검색조건은 **용어+동의어**, **영문명**, **용어설명** 이다. 

{{:egovframework:oe1:가이드:운영정보관리:용어관리:용어사전_목록.jpg|}}

  * 검색 : 선택 또는 입력한 검색조건을 기반으로 등록한 용어사전내용에 대한 정보를 목록으로 조회한다.
  * 등록 : 용어사전 내용을 등록하기 위한 화면으로 이동한다.
  * 엑셀등록 : 엑셀파일로 저장되에 있는 용어사전 내용을 일괄등록 하는 페이지로 이동한다.
  * 엑셀다운로드 : 용어사전 목록을 엑셀파일로 다운로드 받는다. 

===== 용어사전 상세  =====
용어사전 목록에서 용어명을 클릭하면 해당 용어에 대한 정보를 상세 조회한다.

{{:egovframework:oe1:가이드:운영정보관리:용어관리:용어사전_상세.jpg|}}

  * 수정 : 용어사전 정보를 수정할 수 있는 편집 가능한 화면으로 이동한다. 단, 자신이 등록한 용어만 수정 가능하다. 
  * 목록 : 용어사전 목록 조회 화면으로 이동한다. 
  * 삭제 : 등록한 용어를 삭제하는 화면으로 이동한다. 단, 자신이 등록한 용어만 삭제 가능하다. 

===== 용어사전 등록 =====
용어사전 내용을 등록할 수 있는 기능을 제공한다.
용어명, 영문명, 영문약어는 각각 중복체크하여 관리한다.
용어설명을 입력하지 않으면 용어명으로 입력한다.  

{{:egovframework:oe1:가이드:운영정보관리:용어관리:용어사전_등록.jpg|}}

  * 저장 : 입력한 용어사전 내용을 등록한다.
  * 목록 : 용어사전 목록 조회 화면으로 이동한다.
  * 초기화 : 입력한 용어사전 내용을 공백으로 초기화 한다. 


===== 용어사전 수정 =====
용어사전 상세 조회 화면에서 수정 버튼을 클릭한 경우  해당 용어사전에 대한 정보를 수정할 수 있는 기능을 제공한다.
수정시에는 용어명, 영문명, 영문 약어는 수정할 수 없다. 
수정을 원한다면 삭제 후 재등록 해야 한다. 

{{:egovframework:oe1:가이드:운영정보관리:용어관리:용어사전_수정.jpg|}}

  * 저장 : 입력한 용어사전 내용을 수정한다.
  * 목록 : 용어사전 목록 조회 화면으로 이동한다.

===== 엑셀등록화면  =====
엑셀파일로 만들어진 용어사전 목록을 일괄등록한다. 

{{:egovframework:oe1:가이드:운영정보관리:용어관리:용어사전_엑셀등록.jpg|}}


  * 저장 : 엑셀파일로 만들어진 용어사전 목록을 일괄 등록한다.  단, 일괄등록시 기존에 등록된 용어사전 내용을 모두 삭제 후 등록한다.  
  * 목록 : 용어사전 목록 조회 화면으로 이동한다.
  * 엑셀Templet다운로드 : 용어사전 일괄등록을 하려면 표준 양식이 있어야 한다.  엑셀Templet 표준을 다운 받을 수 있는 기능을 제공한다.
