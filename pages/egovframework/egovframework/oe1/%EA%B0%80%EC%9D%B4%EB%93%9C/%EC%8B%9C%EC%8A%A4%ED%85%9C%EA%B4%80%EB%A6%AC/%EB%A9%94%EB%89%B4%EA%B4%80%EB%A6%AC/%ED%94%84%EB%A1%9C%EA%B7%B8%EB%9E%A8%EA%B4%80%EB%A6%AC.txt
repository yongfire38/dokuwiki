====== 개요 ====== 
커뮤니케이션 관리의 Top 및 Left메뉴에서 이용할 수 있는 프로그램 정보를 생성 및 관리한다.
**  참고 : 메뉴관리를 위해서는 '프로그램관리' - '메뉴관리' - '메뉴생성관리'가 순차적으로 등록완료되어야한다.**

\\
====== 기능 설명 ======
  - 프로그램을 등록할 수 있다.
  - 프로그램을 수정할 수 있다.
  - 프로그램을 조회할 수 있다.
  - 프로그램을 삭제할 수 있다.

\\
====== 화면 설명 ======
===== 목록 화면 =====
{{:egovframework:oe1:가이드:시스템관리:메뉴관리:progrmlist.png|}}
  * 등록된 프로그램정보가 페이징처리되어 목록조회된다.
  * 프로그램ID나 프로그램명으로 조건검색을 할 수 있다.
  * 프로그램ID를 클릭하면 프로그램정보가 상세조회된다.
  * 등록 : 새로운 프로그램정보를 등록하기 위한 화면으로 이동한다.

===== 상세 화면 =====
{{:egovframework:oe1:가이드:시스템관리:메뉴관리:progrmdetail.png|}}
  * 해당 프로그램의 상세정보를 확인할 수 있다.
  * 수정 : 해당 프로그램 정보를 수정하는 화면으로 이동한다.
  * 삭제 : 해당 프로그램 정보를 삭제한다.
  * 목록 : 목록으로 돌아간다.

===== 등록/수정 화면 =====
  * 프로그램 등록은 1레벨, 2레벨, 3레벨이 모두 입력되어야 한다.
  * 연관된 1레벨, 2레벨, 3레벨을 순차적으로 입력될 필요는 없으나 가시성을 위해서는 순차적으로 입력하는 것이 좋다.

{{:egovframework:oe1:가이드:시스템관리:메뉴관리:progrmregist1.png|}}
  * 1레벨의 표준 등록 형식은 없으나 그림과 같은 형태로 입력하여 사용하는 것을 권장한다.(프로그램명의 ==는 메뉴관리에서 레벨구분의 가시성을 위하여 임의로 추가하였음) 
  * 1레벨은 URL이 없으므로 필수입력값이라도 임의로 입력하면 된다. 
  * 1레벨의 프로그램정보는 생성할 메뉴의 수에 관계없이 단 한번만 유일하게 입력되면 된다.
  * 수정 : 해당 프로그램 정보를 수정하는 화면으로 이동한다.
  * 삭제 : 해당 프로그램 정보를 삭제한다.
  * 목록 : 목록으로 돌아간다.

{{:egovframework:oe1:가이드:시스템관리:메뉴관리:progrmregist2.png|}}
  * 2레벨의 표준 등록 형식은 없으나 그림과 같은 형태로 입력하여 사용하는 것을 권장한다.(프로그램명의 ==는 메뉴관리에서 레벨구분의 가시성을 위하여 임의로 추가하였음) 
  * 2레벨은 URL이 없으므로 필수입력값이라도 임의로 입력하면 된다.
  * 수정 : 해당 프로그램 정보를 수정하는 화면으로 이동한다.
  * 삭제 : 해당 프로그램 정보를 삭제한다.
  * 목록 : 목록으로 돌아간다.

{{:egovframework:oe1:가이드:시스템관리:메뉴관리:progrmregist3.png|}}
  * 3레벨의 표준 등록 형식은 없으나 그림과 같은 형태로 입력하여 사용하는 것을 권장한다.
  * 3레벨은 URL과 직접 매핑되므로 실제 호출되어야 하는 서비스의 .do명을 올바르게 쓴다.
  * 저장: 해당 프로그램 정보 수정을 완료한다.  
  * 목록 : 목록으로 돌아간다.
