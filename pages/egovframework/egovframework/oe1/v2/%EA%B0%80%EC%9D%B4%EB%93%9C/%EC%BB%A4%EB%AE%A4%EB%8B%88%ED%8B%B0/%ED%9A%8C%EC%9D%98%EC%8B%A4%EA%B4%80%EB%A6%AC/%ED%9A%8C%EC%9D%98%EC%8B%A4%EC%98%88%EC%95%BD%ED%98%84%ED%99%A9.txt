====== 개요 ====== 
회의실별 예약현황을 볼 수 있도록 기능을 제공한다.

====== 기능 설명 ======

  - 회의실예약현황  : 회의실별 예약현황을 표로 보여준다. 
  - 회의실관리 상세 : 핑크색으로 처리된 칸을 클릭하면 회의실병 예약 현황을 보여준다. 
  - 회의실관리 등록 : 회의실예약 화면으로 이동한다. 


====== 화면 설명 ======

===== 회의실예약현황 목록 =====
회의실별 예약현황을 시간별로 보여준다. 
예약된 회의실 정보는 핑크색으로  30분 단위로  보여준다. 

{{:egovframework:oe1:가이드:커뮤니티:회의실관리:회의실예약현황.jpg|}}

  * 등록 : 회의실예약 내용을 등록하기 위한 화면으로 이동한다.

===== 회의실예약 상세  =====
회의실예약현황 목록에서 핑크색으로된 칸을 클릭하면 회의실예약 상세 내역을 볼 수 있다. 

{{:egovframework:oe1:가이드:커뮤니티:회의실관리:회의실예약_상세.jpg|}}

  * 수정 : 회의실예약 정보를 수정할 수 있는 편집 가능한 화면으로 이동한다. 단, 자신이 등록한 회의실예약만 수정 가능하다. 
  * 목록 : 회의실예약 목록 조회 화면으로 이동한다. 
  * 삭제 : 등록한 회의실예약을 삭제하는 화면으로 이동한다. 단, 자신이 등록한 회의실예약만 삭제 가능하다. 

===== 회의실예약 등록 =====
회의실예약 내용을 등록할 수 있는 기능을 제공한다.

{{:egovframework:oe1:가이드:커뮤니티:회의실관리:회의실예약_등록.jpg|}}

  * 저장 : 입력한 회의실예약 내용을 등록한다.
  * 목록 : 회의실예약 목록 조회 화면으로 이동한다.
  * 초기화 : 입력한 회의실예약 내용을 공백으로 초기화 한다.
