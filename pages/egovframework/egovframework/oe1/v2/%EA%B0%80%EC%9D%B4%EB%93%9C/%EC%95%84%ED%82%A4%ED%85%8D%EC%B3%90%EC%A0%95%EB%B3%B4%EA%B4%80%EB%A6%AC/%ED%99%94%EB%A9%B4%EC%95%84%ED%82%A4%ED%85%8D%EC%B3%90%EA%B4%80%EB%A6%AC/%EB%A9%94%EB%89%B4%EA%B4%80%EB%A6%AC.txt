

====== 개요 ====== 
업무시스템의 메뉴정보를 관리하기 위한 아키텍쳐메뉴의 등록/수정/상세조회/목록검색 할 수 있는 기능을 제공한다.

====== 기능 설명 ======
  - 아키텍쳐메뉴 목록 : 아키텍쳐메뉴의 목록을 검색 조회한다.
  - 아키텍쳐메뉴 상세 : 아키텍쳐메뉴 정보를 상세 조회한다.
  - 아키텍쳐메뉴 등록 : 아키텍쳐메뉴 정보를 등록한다.
  - 아키텍쳐메뉴 수정 : 아키텍쳐메뉴 정보를 수정한다.
  

====== 화면 설명 ======

===== 변경요청서 목록 =====
아키텍쳐메뉴 목록은 페이지당 10건씩 조회되며 검색조건은 **메뉴명**이다. 

{{:egovframework:oe1:가이드:아키텍쳐정보관리:화면아키텍쳐관리:아키텍쳐메뉴목록.jpg|}}

  * 검색 : 선택 또는 입력한 검색조건을 기반으로 아키텍쳐메뉴 목록을 조회한다.
  * 등록 : 아키텍쳐메뉴를 등록하기 위한 화면으로 이동한다.



===== 아키텍쳐메뉴 상세  =====
아키텍쳐메뉴 목록에서 메뉴명을 클릭하면 해당 아키텍쳐메뉴 정보를 상세 조회한다.

{{:egovframework:oe1:가이드:아키텍쳐정보관리:화면아키텍쳐관리:아키텍쳐메뉴상세.jpg|}}

  * 수정 : 아키텍쳐메뉴 정보를 수정할 수 있는 편집 가능한 화면으로 이동한다.
  * 삭제 : 아키텍쳐메뉴 정보를 삭제한다.
  * 목록 : 아키텍쳐메뉴 목록 조회 화면으로 이동한다. 



===== 아키텍쳐메뉴 등록 =====
아키텍쳐메뉴를 등록하는 기능을 제공한다.

{{:egovframework:oe1:가이드:아키텍쳐정보관리:화면아키텍쳐관리:아키텍쳐메뉴등록.jpg|}}

  * 저장 : 입력한 아키텍쳐메뉴 정보를 등록한다. 
  * 목록 : 아키텍쳐메뉴 목록 조회 화면으로 이동한다. 


===== 아키텍쳐메뉴 수정 =====
아키텍쳐메뉴 상세 조회 화면에서 수정 버튼을 클릭하여 아키텍쳐메뉴 정보를 수정한다.

{{:egovframework:oe1:가이드:아키텍쳐정보관리:화면아키텍쳐관리:아키텍쳐메뉴수정.jpg|}}

  * 저장 : 입력한 아키텍쳐메뉴 정보를 수정한다. 
  * 목록 : 아키텍쳐메뉴 목록 화면으로 이동한다.
