

====== 개요 ====== 
아키텍쳐구조를 컴포넌트별로 관리하기 위하여 컴포넌트의 등록/수정/상세조회/목록검색 할 수 있는 기능을 제공한다.

아키텍쳐 정보수집을 통해 저장된 아키텍쳐 정보(클래스/메서드 등)를 컴포넌트에 매핑하여 컴포넌트-클래스-메서드-쿼리/파라미터의 형식으로 아키텍쳐 정보를 관리할 수 있다.

*컴포넌트관리, 아키텍쳐정보조회, 화면관리 메뉴는 이클립스 Plug-in의 아키텍쳐 정보수집을 실행 후 사용하실 수 있습니다.

====== 기능 설명 ======
  - 컴포넌트 목록 : 컴포넌트의 목록을 검색 조회한다.
  - 컴포넌트 상세/수정 : 컴포넌트 정보를 상세조회/수정한다.
  - 컴포넌트 등록 : 컴포넌트 정보를 등록한다.
  - 클래스 조회 팝업 : 등록된 클래스 목록을 조회한다.
  

====== 화면 설명 ======

===== 컴포넌트 목록 =====
컴포넌트 목록은 페이지당 10건씩 조회되며 검색조건은 **업무명**, **컴포넌트명**이다. 
*참고 : 컴포넌트 관리의 업무명은 공통코드 OE1030에 업무명을 코드로 정의한 후 사용한다.

{{:egovframework:oe1:가이드:아키텍쳐정보관리:업무아키텍쳐관리:컴포넌트_목록.jpg|}}

  * 검색 : 선택 또는 입력한 검색조건을 기반으로 컴포넌트 목록을 조회한다.
  * 등록 : 컴포넌트 등록하기 위한 화면으로 이동한다.



===== 컴포넌트 상세/수정  =====
컴포넌트 목록에서 컴포넌트명을 클릭하면 해당 컴포넌트의 정보를 상세 조회한다.

{{:egovframework:oe1:가이드:아키텍쳐정보관리:업무아키텍쳐관리:컴포넌트_상세.jpg|}}

  * 저장 : 컴포넌트 정보를 수정한다.
  * 삭제 : 컴포넌트 정보를 삭제한다.
  * 목록 : 컴포넌트 목록 조회 화면으로 이동한다. 
  * 추가 : 클래스를 입력하기 위한 ROW를 증가시킨다.
  * 삭제 : 클래스를 입력하는 ROW를 감소시킨다.
  * 검색 : 클래스 목록을 조회하는 팝업창을 호출한다.



===== 컴포넌트 등록 =====
컴포넌트를 등록하는 기능을 제공한다.

{{:egovframework:oe1:가이드:아키텍쳐정보관리:업무아키텍쳐관리:컴포넌트_등록.jpg|}}

  * 저장 : 입력한 컴포넌트 정보를 등록한다. 
  * 목록 : 컴포넌트 목록 조회 화면으로 이동한다. 
  * 추가 : 클래스를 입력하기 위한 ROW를 증가시킨다.
  * 삭제 : 클래스를 입력하는 ROW를 감소시킨다.
  * 검색 : 클래스 목록을 조회하는 팝업창을 호출한다.


===== 클래스 조회팝업 =====
검색 버튼을 클릭하면 클래스 목록 정보를 조회한다.



  * 검색 : 선택 또는 입력한 검색조건을 기반으로 클래스 목록을 조회한다.
