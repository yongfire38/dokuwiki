====== 개요 ====== 
커뮤니케이션도구를 사용하기 위해서는 일정 권한 이상으로 로그인되어야 한다.

\\
====== 기능 설명 ======
  - 로그인**(메인화면에서도 로그인이 가능하나 로그인에 필요한 정보가 부족한 경우나 권한이 부족한 경우에 본 로그인 전용 화면으로 자동 링크되어 인증을 유도한다.)**
  - 아이디 저장
  - 아이디/비밀번호 찾기
  - 회원가입   

\\
====== 화면 설명 ======
===== 로그인 화면 =====
{{:egovframework:oe1:가이드:loginusr.png|}}
  * 아이디 : 등록된 아이디를 입력한다.
  * 비밀번호 : 등록된 비밀번호를 입력한다.
  * ID저장 : 아이디를 쿠키에 저장한다.
  * 아이디/비밀번호 찾기 : 등록된 아이디 및 비밀번호를 찾거나 변경한다.
  * 회원가입 : 회원으로 등록한다.
===== 아이디/비밀번호찾기 화면 =====
{{:egovframework:oe1:가이드:idpwsearch.png|}}
  * 아이디찾기 : 정보가 모두 일치하는 경우에는 아이디를 알려준다.
  * 비밀번호찾기 : 정보가 모두 일치하는 경우에는 임시비밀번호를 메일로 발송한다.
===== 회원가입 화면 =====
{{:egovframework:oe1:가이드:userinsert.png|}}
  * 가입신청 : 필요한 정보를 입력하면 가입신청이 완료된다. 
  * 취소 : 가입신청을 취소한다.
**단,회원가입 후 곧바로 로그인하여 서비스를 이용할 수는 없으며 관리자가 해당 아이디에 대한 권한을 설정한 후에 정상적인 로그인과 이용이 비로소 가능하다.**\\
