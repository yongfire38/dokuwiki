
====== 개요 ====== 
운영개선요청을 위한 등록/수정/상세조회/만족도저장/목록검색 할 수 있는 기능을 제공한다.

====== 기능 설명 ======
  - 운영개선요청 목록 : 운영개선요청정보 목록을 검색 조회한다.
  - 운영개선요청 상세 : 운영개선요청정보를 상세 조회한다.
  - 운영개선요청 등록 : 운영개선요청정보를 등록한다.
  - 운영개선요청 수정 : 운영개선요청정보를 수정한다.
  - 운영개선요청 만족도 저장 : 개선처리가 완료된 운영개선요청 처리내용에 대한 만족도를 입력한다.
  

====== 화면 설명 ======

===== 운영개선요청 목록 =====
운영개선요청정보 목록은 페이지당 10건씩 조회되며 검색조건은 **처리상태**, **업무구분**, **개선요청명**, **내용**, **요청자**이다. 

{{:egovframework:oe1:가이드:운영개선관리:운영개선요청관리:oper_01_01.jpg|}}

  * 검색 : 선택 또는 입력한 검색조건을 기반으로 운영개선요청정보 목록을 조회한다.
  * 개선요청명 : 운영개선요청 상세 화면으로 이동한다.
  * 등록 : 운영개선요청을 등록하기 위한 화면으로 이동한다.



===== 운영개선요청 상세  =====
운영개선요청 목록에서 개선요청명을 클릭하면 해당 운영개선요청의 정보를 상세 조회한다.

{{:egovframework:oe1:가이드:운영개선관리:운영개선요청관리:oper_01_02.jpg|}}

  * 수정 : 운영개선요청 정보를 수정할 수 있는 편집 가능한 화면으로 이동한다.
  * 삭제 : 운영개선요청 정보를 삭제한다.
  * 목록 : 운영개선요청 목록 조회 화면으로 이동한다. 



===== 운영개선요청 등록 =====
운영개선요청을 등록하는 기능을 제공한다.

{{:egovframework:oe1:가이드:운영개선관리:운영개선요청관리:oper_01_03.jpg|}}

  * 찾아보기 : 첨부파일을 선택하기 위한 팝업을 띄운다.
  * 저장 : 입력한 운영개선요청 정보를 등록한다. 
  * 목록 : 운영개선요청 목록 조회 화면으로 이동한다. 


===== 운영개선요청 수정 =====
처리상태가 [요청]인 운영개선요청 상세 조회 화면에서 수정 버튼을 클릭하여 운영개선요청 정보를 수정한다.

{{:egovframework:oe1:가이드:운영개선관리:운영개선요청관리:oper_01_04.jpg|}}

  * 찾아보기 : 첨부파일을 선택하기 위한 팝업을 띄운다.
  * 첨부파일 목록에서 파일명 옆의 X 이미지를 클릭하면 해당 첨부파일을 삭제한다.
  * 저장 : 입력한 운영개선요청 정보를 수정한다. 
  * 목록 : 운영개선요청 목록 화면으로 이동한다. 


===== 운영개선요청 만족도 저장 =====
운영개선요청 목록에서 처리상태가 [완료]이고 만족도 입력이 [N]인 개선요청명을 클릭하여 처리 만족도를 저장한다. 

{{:egovframework:oe1:가이드:운영개선관리:운영개선요청관리:oper_01_05.jpg|}}

  * 만족도저장 : 입력한 처리만족도 정보를 저장한다.
  * 목록 : 운영개선요청 목록 화면으로 이동한다.
