

====== 개요 ====== 

변경요청접수처리건에 대하여 운영개선담당자가 변경작업계획 및 변경작업결과를 등록/수정/상세조회/목록검색 할 수 있는 기능을 제공한다.
담당자 권한위임이 되지 않은 변경처리건인 경우, 변경작업계획 및 변경작업결과에 대해 운영개선관리자에게 검토요청을 할 수 있는 기능을 제공한다.


====== 기능 설명 ======

  - 변경작업 목록 : 운영개선담당자에게 할당된 변경작업목록을 검색 조회한다.
  - 변경작업계획 상세 : 변경작업계획 정보를 상세 조회한다.
  - 요청자 contact 정보 팝업 : 변경요청자의 contact 정보를 조회한다.
  - 변경작업계획 등록 : 변경작업계획 정보를 등록/수정한다.
  - 변경작업 상세 : 변경작업 정보를 상세 조회한다.
  - 변경작업 등록 : 변경작업 정보를 등록/수정한다.
  - 변경요청서 상세 팝업 : 변경요청 접수처리된 변경요청서 정보를 상세 조회한다.
  

====== 화면 설명 ======

===== 변경작업 목록 =====
변경작업 목록은 페이지당 10건씩 조회되며 검색조건은 **처리상태**, **긴급처리**, **접수일**이다. 


{{:egovframework:oe1:가이드:운영개선관리:변경작업관리:변경작업목록.jpg|}}


  * 검색 : 선택 또는 입력한 검색조건을 기반으로 변경작업 목록을 조회한다.


===== 변경작업계획 상세  =====
변경작업 목록에서 처리상태가 [계획검토]인 건을 클릭하면 해당 변경작업계획의 정보를 상세 조회한다.


{{:egovframework:oe1:가이드:운영개선관리:변경작업관리:변경작업계획상세.jpg|}}


  * 변경요청서상세 : 변경요청 접수처리된 변경요청서 정보를 팝업조회한다.
  * 목록 : 변경작업 목록 조회 화면으로 이동한다. 


===== 요청자 contact 정보 팝업 =====
변경요청자의 **전화번호**, **핸드폰번호**, **이메일주소** 정보를 조회한다.

{{:egovframework:oe1:가이드:운영개선관리:변경작업관리:요청자연결정보.jpg|}}
  * 닫기 : 화면을 종료한다.



===== 변경작업계획 등록 =====
변경작업 목록 화면에서 처리상태가 [접수]/[계획]인 건을 클릭한 경우 변경작업계획 정보를 등록/수정할 수 있는 기능을 제공한다.


{{:egovframework:oe1:가이드:운영개선관리:변경작업관리:변경작업계획등록.jpg|}}

  * 운영개선담당자에게 분배된 변경요청건에 대해 최초로 작업계획을 등록하는 화면이다.
  * 변경요청서상세 : 변경요청 접수처리된 변경요청서 정보를 팝업조회한다.
  * 저장 : 입력한 변경작업계획 정보를 등록/수정한다.
  * 목록 : 변경작업 목록 조회 화면으로 이동한다.

{{:egovframework:oe1:가이드:운영개선관리:변경작업관리:변경작업계획등록_검토버튼01.jpg|}}

  * 운영개선담당자가 작업계획을 등록 한 후, 변경요청건이 담당자 계획검토 권한위임이 아닌 경우 처리 화면이다.
  * 계획검토요청 : 운영개선관리자에게 변경작업계획 검토를 요청한다.
  * 저장 : 입력한 변경작업계획 정보를 등록/수정한다.
  * 목록 : 변경작업 목록 조회 화면으로 이동한다.

{{:egovframework:oe1:가이드:운영개선관리:변경작업관리:변경작업계획등록_담당자검토버튼01.jpg|}}

  * 변경요청서상세 : 변경요청 접수처리된 변경요청서 정보를 팝업조회한다.
  * 운영개선담당자가 작업계획을 등록 한 후, 변경요청건이 담당자 계획검토 권한위임인 경우 처리 화면이다.
  * 담당자검토 :  변경담당자에게 계획검토에대한 권한위임이 된 경우 변경담당자가 변경작업계획을 검토완료한다.
  * 저장 : 입력한 변경작업계획 정보를 등록/수정한다.
  * 목록 : 변경작업 목록 조회 화면으로 이동한다.

===== 변경작업 상세  =====
변경작업 목록에서 처리상태가 [완료검토]/[완료]인 건을 클릭한 경우 해당 변경작업결과정보를 상세 조회한다.


{{:egovframework:oe1:가이드:운영개선관리:변경작업관리:변경작업상세.jpg|}}


  * 목록 : 변경작업 목록 조회 화면으로 이동한다. 



===== 변경작업 등록 =====
변경작업 목록 화면에서 처리상태가 [작업]인 건을 클릭한 경우 변경작업결과정보를 등록/수정할 수 있는 기능을 제공한다.


{{:egovframework:oe1:가이드:운영개선관리:변경작업관리:변경작업등록.jpg|}}

  * 운영개선담당자가 분배받은 변경요청건에 대한 변경작업완료 후, 결과를 최초로 등록하는 화면이다.
  * 저장 : 입력한 변경작업결과 정보를 등록/수정한다.
  * 목록 : 변경작업 목록 조회 화면으로 이동한다.

{{:egovframework:oe1:가이드:운영개선관리:변경작업관리:변경작업등록_검토버튼.jpg|}}

  * 변경요청건이 담당자 완료검토 권한위임이 아닌 경우 처리화면이다.
  * 완료검토요청 : 운영개선관리자에게 변경작업결과 검토를 요청한다.
  * 저장 : 입력한 변경작업결과 정보를 등록/수정한다.
  * 목록 : 변경작업 목록 조회 화면으로 이동한다.

{{:egovframework:oe1:가이드:운영개선관리:변경작업관리:변경작업등록_변경완료버튼.jpg|}}

  * 변경요청건이 담당자 완료검토 권한위임인 경우 처리화면이다.
  * 변경완료 : 변경담당자에게 변경완료검토에대한 권한위임이 된 경우 변경담당자가 변경작업결과에대해 검토완료한다.
  * 저장 : 입력한 변경작업결과 정보를 등록/수정한다.
  * 목록 : 변경작업 목록 조회 화면으로 이동한다.


===== 변경요청서 상세팝업 =====
변경요청 접수처리된 변경요청서 정보를 상세 조회한다. 


{{:egovframework:oe1:가이드:운영개선관리:변경작업관리:변경요청서상세팝업.jpg|}}


  * 요청근거 : 변경요청서를 등록한 요청시스템의 출처 정보를 조회한다.
  * 닫기 : 화면을 종료한다.
