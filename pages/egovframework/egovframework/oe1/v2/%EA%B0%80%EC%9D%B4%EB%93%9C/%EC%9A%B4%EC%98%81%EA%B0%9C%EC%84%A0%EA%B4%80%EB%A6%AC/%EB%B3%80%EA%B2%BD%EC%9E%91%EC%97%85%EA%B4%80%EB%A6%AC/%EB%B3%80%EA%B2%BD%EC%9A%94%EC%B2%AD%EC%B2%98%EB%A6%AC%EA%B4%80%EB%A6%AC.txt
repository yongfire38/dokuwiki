

====== 개요 ====== 
등록된 변경요청건에 대한 변경작업진행을 위하여 운영개선관리자가 변경요청건에 대한 영향도분석 및 변경담당자 정보를 등록/수정/상세조회/목록검색 할 수 있는 기능을 제공한다.

====== 기능 설명 ======

  - 변경요청처리 목록 : 변경요청처리 목록을 검색 조회한다.
  - 변경요청처리 상세 : 변경요청처리 정보를 상세 조회한다.
  - 요청자 contact 정보 팝업 : 변경요청자의 contact 정보를 조회한다.
  - 변경요청처리 등록 : 변경요청처리 정보를 등록한다.
  - 변경요청처리 수정 : 변경요청처리 정보를 수정한다.
  - 변경요청서 상세 팝업 : 변경요청 접수처리된 변경요청서 정보를 상세 조회한다.  

====== 화면 설명 ======

===== 변경요청처리 목록 =====
변경요청처리 목록은 페이지당 10건씩 조회되며 검색조건은 **처리상태**, **긴급여부**, **업무구분**, **변경요청명**, **변경담당자명**이다. 


{{:egovframework:oe1:가이드:운영개선관리:변경작업관리:변경요청처리목록.jpg|}}


  * 검색 : 선택 또는 입력한 검색조건을 기반으로 변경요청처리 목록을 조회한다.



===== 변경요청처리 상세  =====
변경요청처리 목록에서 처리상태가 [접수]인 변경요청명을 클릭하면 해당 변경요청건에 대한 접수처리 정보를 상세 조회한다.


{{:egovframework:oe1:가이드:운영개선관리:변경작업관리:변경요청처리_상세.jpg|}}


  * 변경요청서상세 : 변경요청 접수처리된 변경요청서 정보를 팝업조회한다.
  * 수정 : 변경요청건에대한 접수처리 정보를 수정할 수 있는 편집 가능한 화면으로 이동한다.
  * 목록 : 변경요청처리 목록 조회 화면으로 이동한다. 


===== 요청자 contact 정보 팝업 =====
변경요청자의 **전화번호**, **핸드폰번호**, **이메일주소** 정보를 조회한다.

{{:egovframework:oe1:가이드:운영개선관리:변경작업관리:요청자연결정보.jpg|}}
  * 닫기 : 화면을 종료한다.



===== 변경요청처리 등록 =====
변경요청처리 목록에서 처리상태가 [요청]인 변경요청명을 클릭한 경우 해당 변경요청건에대한  접수처리정보를 등록할 수 있는 기능을 제공한다.


{{:egovframework:oe1:가이드:운영개선관리:변경작업관리:변경요청처리_등록.jpg|}}

  * 변경요청서상세 : 변경요청 접수처리된 변경요청서 정보를 팝업조회한다.
  * 담당자찾기 : 운영개선담당자 정보를 팝업조회한다.
  * 저장 : 입력한 변경요청접수처리 정보를 등록한다.
  * 목록 : 변경요청처리 목록 조회 화면으로 이동한다.
  * 담당자권한위임의 [계획검토]에 체크를 하면, 변경작업계획에 대해 운영개선관리자에 의한 계획검토가 PASS된다. 
  * 담당자권한위임의 [변경완료검토]에 체크를 하면, 변경작업결과에 대해 운영개선관리자에 의한 완료검토가 PASS된다.

===== 변경요청처리 수정 =====
변경요청처리 상세조회 화면에서 수정 버튼을 클릭한 경우  변경요청건에대한 접수처리정보를 수정할 수 있는 기능을 제공한다.


{{:egovframework:oe1:가이드:운영개선관리:변경작업관리:변경요청처리_수정.jpg|}}


  * 담당자찾기 : 운영개선담당자 정보를 팝업조회한다.
  * 저장 : 입력한 변경요청처리 정보를 수정한다.
  * 목록 : 변경요청처리 목록 조회 화면으로 이동한다.



===== 변경요청서 상세팝업 =====
변경요청 접수처리된 변경요청서 정보를 상세 조회한다.


{{:egovframework:oe1:가이드:운영개선관리:변경작업관리:변경요청서상세팝업.jpg|}}


  * 요청근거 : 운영개선요청시스템으로부터 변경이관 처리된 경우, 운영개선요청서정보를 조회한다.
  * 닫기 : 화면을 종료한다.
