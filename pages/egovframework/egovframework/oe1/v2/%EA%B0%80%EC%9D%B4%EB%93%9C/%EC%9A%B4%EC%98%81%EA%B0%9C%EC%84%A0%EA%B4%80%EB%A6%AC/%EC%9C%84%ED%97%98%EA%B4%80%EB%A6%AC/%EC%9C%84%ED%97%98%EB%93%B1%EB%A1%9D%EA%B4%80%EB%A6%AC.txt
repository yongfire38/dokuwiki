

====== 개요 ====== 
식별된 위험을 관리하기 위하여 운영개선관리자가 위험을 등록/수정/상세조회/목록검색 할 수 있는 기능을 제공한다.

====== 기능 설명 ======

  - 위험관리 목록 : 위험관리정보 목록을 검색 조회한다.
  - 위험 상세 : 등록된 위험에대한 정보를 상세 조회한다.
  - 위험 등록 : 식별된 위험에대한 정보를 등록한다.
  - 위험 수정 : 등록한 위험에대한 정보를 수정한다.
  - 담당자 찾기 팝업 : 위험에 대해 예방활동을 할 담당자를 지정하기 위해 운영개선담당자 정보를 조회한다.
  - 운영개선요청 목록 팝업 : 등록한 위험대한 근거정보를 등록하기 위해 운영개선요청정보를 목록으로 조회한다.
  

====== 화면 설명 ======

===== 위험관리 목록 =====
위험관리 목록은 페이지당 10건씩 조회되며 검색조건은 **처리상태**, **위험구분**, **위험도**, **위험명**, **위험내용**이다. 

{{:egovframework:oe1:가이드:운영개선관리:위험관리:risk_01_01.jpg|}}

  * 검색 : 선택 또는 입력한 검색조건을 기반으로 등록한 위험에대한 정보를 목록으로 조회한다.
  * 등록 : 식별한 위험을 등록하기 위한 화면으로 이동한다.


===== 위험 상세  =====
위험관리 목록에서 위험명을 클릭하면 해당 위험에 대한 정보를 상세 조회한다.

{{:egovframework:oe1:가이드:운영개선관리:위험관리:risk_01_02.jpg|}}

  * 수정 : 위험관리 정보를 수정할 수 있는 편집 가능한 화면으로 이동한다.
  * 목록 : 위험관리 목록 조회 화면으로 이동한다. 


===== 위험 등록 =====
식별한 위험을 등록할 수 있는 기능을 제공한다.

{{:egovframework:oe1:가이드:운영개선관리:위험관리:risk_01_03.jpg|}}

  * 담당자찾기 : 운영개선담당자 정보를 팝업조회한다.
  * 찾아보기 : 첨부파일을 선택하기 위한 팝업을 띄운다.
  * 저장 : 입력한 위험관리 정보를 등록한다.
  * 목록 : 위험관리 목록 조회 화면으로 이동한다.

===== 위험 수정 =====
위험상세조회 화면에서 수정 버튼을 클릭한 경우  해당 위험에대한 관리정보를 수정할 수 있는 기능을 제공한다.

{{:egovframework:oe1:가이드:운영개선관리:위험관리:risk_01_04.jpg|}}

  * 담당자찾기 : 운영개선담당자 정보를 팝업조회한다.
  * 찾아보기 : 첨부파일을 선택하기 위한 팝업을 띄운다.
  * 첨부파일 목록에서 파일명 옆의 X 이미지를 클릭하면 첨부파일을 삭제한다.  
  * 삭제 : 관련개선요청목록 개선요청건의 삭제 버튼을 클릭하면 수정된 위험내용을 저장하고 관련개선요청을 삭제한다.  
  * 관련개선요청추가 : 운영개선요청정보를 목록으로 팝업조회한다.
  * 저장 : 입력한 위험관리 정보를 수정한다.
  * 목록 : 위험관리 목록 조회 화면으로 이동한다.



===== 담당자 지정 팝업  =====
운영개선담당자정보를 목록으로 조회한다.

{{:egovframework:oe1:가이드:운영개선관리:위험관리:risk_01_05.jpg|}}

  * 선택 : 해당 운영개선담당자를 선택한다.


===== 운영개선요청 목록 팝업 =====
운영개선요청 목록은 페이지당 10건씩 조회되며 검색조건은 **처리상태**, **업무구분**, **세부구분**, **개선요청명**, **내용**, **요청자**이다. 

{{:egovframework:oe1:가이드:운영개선관리:위험관리:risk_01_06.jpg|}}

  * 검색 : 선택 또는 입력한 검색조건을 기반으로 운영개선요청정보를 목록으로 조회한다.
  * 선택 : 체크된 운영개선요청건들을 등록된 위험에 대한 근거정보로 처리한다.
  * 닫기 : 화면을 종료한다.
