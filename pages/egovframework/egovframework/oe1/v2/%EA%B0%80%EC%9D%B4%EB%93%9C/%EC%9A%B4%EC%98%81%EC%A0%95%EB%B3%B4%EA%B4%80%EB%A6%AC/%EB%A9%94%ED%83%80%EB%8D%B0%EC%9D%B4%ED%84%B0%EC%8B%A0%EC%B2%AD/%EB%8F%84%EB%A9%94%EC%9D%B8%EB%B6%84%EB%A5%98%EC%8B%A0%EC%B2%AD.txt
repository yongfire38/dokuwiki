====== 개요 ====== 
사용자가 도메인분류정보의 신청/수정/신청취소/신청목록조회/신청상세조회를 할 수 있는 기능을 제공한다.

====== 기능 설명 ======

  - 도메인분류신청등록 : 도메인분류신청 정보를 등록한다.
  - 도메인분류신청수정 : 등록된 도메인분류신청 내용에 대한 정보를 수정한다.
  - 도메인분류신청목록조회 : 도메인분류신청 목록을 검색 조회한다.
  - 도메인분류신청상세조회 : 등록된 도메인분류신청 내용에 대한 정보를 상세 조회한다.

====== 화면 설명 ======

===== 도메인분류신청등록 =====
도메인분류 신청 정보를 등록할 수 있는 기능을 제공한다.

{{:egovframework:oe1:가이드:운영정보관리:메타데이터:도메인분류신청등록.jpg|}}

  * 임시저장 : 입력한 도메인분류 신청 정보를 임시로 저장한다.
  * 신청처리 : 입력한 도메인분류 정보를 신청 처리한다.
  * 목록 : [도메인분류신청목록조회] 화면으로 이동한다.


===== 도메인분류신청수정 =====
[도메인분류신청상세조회] 화면에서 수정 버튼을 클릭한 경우 해당 도메인분류신청에 대한 정보를 수정할 수 있는 기능을 제공한다.

{{:egovframework:oe1:가이드:운영정보관리:메타데이터:도메인분류신청수정.jpg|}}

  * 임시저장 : 입력한 도메인분류신청정보를 임시로 저장한다.
  * 신청처리 : 입력한 도메인분류신청정보를 신청 한다.
  * 목록 : [도메인분류신청목록조회] 화면으로 이동한다.

===== 도메인분류신청목록조회 =====
도메인분류신청 목록은 페이지당 10건씩 조회되며 검색조건은 **처리상태**, **도메인분류명**, **신청인** 이다. 

{{:egovframework:oe1:가이드:운영정보관리:메타데이터:도메인분류신청목록조회.jpg|}}

  * 검색 : 선택 또는 입력한 검색조건을 기반으로 등록한 도메인분류신청에 대한 정보를 목록으로 조회한다.
  * 신청 : [도메인분류신청등록] 화면으로 이동한다.

===== 도메인분류신청상세조회  =====
[도메인분류신청목록조회] 화면에서 [신청처리ID]을 클릭하면 해당 도메인분류신청에 대한 상세정보를 조회한다.

{{:egovframework:oe1:가이드:운영정보관리:메타데이터:도메인분류신청상세조회.jpg|}}

  * 신청취소 : 조회된 도메인분류신청에 대한 정보를 삭제 처리한다.
  * 수정 : 조회된 도메인분류신청을 수정할 수 있는 [도메인분류신청수정] 화면으로 이동한다.
  * 목록 : [도메인분류신청목록조회] 화면으로 이동한다.
