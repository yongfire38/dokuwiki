====== 기능목록 관리 가이드 ======

===== 개요 =====
기능목록 관리는 적용점검 대상 프로젝트의 컴포넌트 목록을 등록하여 FP 적용률 분석 및 산출내역서 관리에 활용하기 위한 기능들의 목록을 작성하고 수정하는 기능이다. 기능목록 관리는 컴포넌트들의 등록, 수정, 삭제, 조회 기능을 가지고 있으며, 엑셀파일 형태로 컴포넌트 목록을 사전 등록하는 것을 지원한다. 화면상에 표시된 각 버튼들을 클릭하여 해당 기능을 수행할 수 있다.

{{:egovframework:oe1:적용점검:가이드:기능목록_관리:기능목록관리2.png|}}

  - 검색창 : 기능목록을 컴포넌트명, 중분류명, 소분류명으로 검색할 수 있다.
  - 전체/일부 선택 체크박스 : 최상단의 체크박스를 선택할 경우 모든 컴포넌트들이 선택 혹은 선택 해제된다. 각 컴포넌트들의 체크박스를 선택할 경우 해당 컴포넌트가 선택 또는 선택해제된다. 선택된 컴포넌트들을 일괄적으로 삭제하거나 재사용 컴포넌트로 변경할 수 있다.
  - 컴포넌트 수정 : 컴포넌트의 이름을 클릭하면 해당 컴포넌트의 중분류명, 소분류명, 컴포넌트명, UFP, 재사용 여부를 수정할 수 있다.
  - 기능버튼
     - 재사용만 표시 : 재사용된 컴포넌트들만 표시하고, 재사용이 아닌 컴포넌트는 일괄적으로 합계를 내어 보여준다.
    - Excel 등록 : 기능목록을 Excel로 일괄적으로 등록할 수 있는 화면으로 이동한다.
    - 항목추가 : 개별 컴포넌트를 새롭게 추가한다.
    - 선택삭제 : 체크박스로 선택된 컴포넌트들을 삭제한다.
    - 재사용 변경 : 선택된 컴포넌트는 재사용 컴포넌트로, 선택되지 않은 컴포넌트는 일반 컴포넌트로 변경한다.

===== 기능목록 등록 ======

새로운 컴포넌트를 기능목록에 추가한다.

{{:egovframework:oe1:적용점검:가이드:기능목록_관리:적용저검_기능목록관리_항목추가2.png|}}

    - 중분류 : 컴포넌트의 중분류명을 입력한다.
    - 소분류 : 컴포넌트의 소분류명을 입력한다.
    - 컴포넌트 : 해당 컴포넌트의 이름을 입력한다.
    - UFP : 해당 컴포넌트의 UFP(비조정 기능점수)값을 입력한다.
    - 재사용 여부 : 해당 컴포넌트가 특정 컴포넌트를 재사용하여 만들었을 경우 재사용을, 아닐 경우 재사용 안함을 선택한다.
    - 기능버튼
      - 초기화 : 입력한 내용을 모두 초기화 시킨다.
      - 저장 : 입력한 내용을 저장하여 기능목록에 추가한다.
      - 닫기 : 입력한 내용을 저장하지 않고 창을 닫는다.

===== 기능목록 수정 =====

기존에 등록된 컴포넌트의 세부 내용을 수정한다.

{{:egovframework:oe1:적용점검:가이드:기능목록_관리:적용저검_기능목록관리_항목수정.png|}}

     - 중분류 : 컴포넌트의 중분류명을 수정한다.
    - 소분류 : 컴포넌트의 소분류명을 수정한다.
    - 컴포넌트 : 해당 컴포넌트의 이름을 수정한다.
    - UFP : 해당 컴포넌트의 UFP(비조정 기능점수)값을 수정한다.
    - 재사용 여부 : 해당 컴포넌트가 특정 컴포넌트를 재사용하여 만들었을 경우 재사용을, 아닐 경우 재사용 안함을 선택한다.
    - 기능버튼
      - 초기화 : 수정한 내용을 모두 원래 상태로 복구 시킨다.
      - 저장 : 수정한 내용을 저장하여 기능목록에 추가한다.
      - 닫기 : 수정한 내용을 저장하지 않고 창을 닫는다.


===== 기능목록 삭제 및 재사용 변경 =====

목록에 있는 컴포넌트들 중 일부를 삭제하거나 재사용 여부를 변경한다. 대상이 되는 컴포넌트는 해당 줄을 클릭하거나 체크박스를 체크하여 선택할 수 있으며, 표상단의 체크박스를 클릭하여 일괄적으로 선택하거나 선택해제할 수 있다.

{{:egovframework:oe1:적용점검:가이드:기능목록_관리:적용점검_기능목록관리_삭제변경.png|}}

     - 선택삭제 : 선택된 컴포넌트를 삭제한다.
    - 재사용 변경 : 선택된 컴포넌트를 재사용 컴포넌트로, 선택되지 않은 컴포넌트를 일반 컴포넌트로 다시 지정한다.
    

===== 기능목록 엑셀등록 =====

사전에 작성된 엑셀파일로 컴포넌트 목록을 일괄 등록한다. 사전에 등록되었던 컴포넌트들은 모두 삭제된다.

입력할 Excel 양식은 다음 파일을 사용한다. 엑셀파일의 등록이 수행될 때, 양식과 맞지 않는 행은 모두 무시된다. 각 분류명 또는 컴포넌트명이 길이제한을 초과한 경우, UFP값이 숫자가 아니거나, 범위에서 벗어난 경우, 재사용 여부가 Y/N 또는 O/X로 기입되지 않은 행은 모두 무시되며, 정상적으로 입력된 행만 목록에 추가된다.

{{:egovframework:oe1:적용점검:가이드:기능목록_관리:fptemplate.xls|}}

{{:egovframework:oe1:적용점검:가이드:기능목록_관리:적용점검_기능목록관리_엑셀등록.png|}}

     - 찾아보기 : 업로드할 엑셀파일을 검색한다.
    - 등록 : 검색된 엑셀파일을 등록한다. xls파일만 등록할 수 있다.
