====== 모바일 디바이스API New Template Project Wizard =====

===== 개요 =====
   * 표준프레임워크 기반의 모바일 디바이스 API 실행환경을 이용하여 앱 어플리케이션 개발시 개발자 편의성을 위하여 모바일 디바이스 API 실행환경 코드를 포함하고 있는 모바일 디바이스API 실행환경 프로젝트 자동 생성 마법사를 제공한다.

===== 설명 =====

  * 표준프레임워크 기반의 모바일 디바이스 API 실행환경을 통해 앱 어플리케이션 및 서버 어플리케이션 표준 코드와 가이드 프로그램 프로젝트를 생성한다.
  * 본 Wizard는 ** 모바일 디바이스 API 실행환경 표준코드, 표준 샘플 템플릿, 가이드 프로그램 프로젝트 생성 ** 기능을 제공한다.
  {{:egovframework:hyb:hdev:imp:ios:기능흐름도_ios.png?700}} \\

  - ** 템플릿을 이용하여 프로젝트를 생성한다. ** 
  - ** 관련 리소스 www 폴더를 프로젝트 폴더 내에 복사한다. ** 
  - ** 복사한 www 폴더를 프로젝트에 추가한다. **

  * 모바일 디바이스API 실행환경 템플릿이 Xcode에 추가되어 있어야한다.

===== 템플릿 추가 =====

  * Xcode 환경에서 모바일 디바이스API 실행환경을 이용할 수 있도록 템플릿을 추가하는 방법이다.
  * [[egovframework:hyb:hdev:ios:add_ios_template|Xcode에 템플릿 추가하기]]

===== 사용법 =====

  * 템플릿 추가를 통해 만들어진 모바일 디바이스API 실행환경 템플릿을 이용하여 프로젝트를 생성한다.

  - ** 생성한 프로젝트 > 우 클릭 > Show in Finder ** \\ {{:egovframework:hyb:hdev:imp:ios:ios_template_1.png}}
  - ** 해당 폴더에 관련 리소스 파일 [www.zip]을 복사한 후, 더블 클릭 > 압축 해제 ** \\ {{:egovframework:hyb:hdev:imp:ios:ios_template_2.png}}
  - ** 압축해제한 www 폴더를 Xcode 프로젝트 파일에 Drag&Drop 을 한다. ** 
  - ** 아래와 같이 체크 및 옵션을 선택 > Finish ** \\ {{:egovframework:hyb:hdev:imp:ios:ios_template_3.png}}
  - ** 프로젝트 생성/추가 완료 및 실행 ** \\ {{:egovframework:hyb:hdev:imp:ios:ios_template_4.png}}
