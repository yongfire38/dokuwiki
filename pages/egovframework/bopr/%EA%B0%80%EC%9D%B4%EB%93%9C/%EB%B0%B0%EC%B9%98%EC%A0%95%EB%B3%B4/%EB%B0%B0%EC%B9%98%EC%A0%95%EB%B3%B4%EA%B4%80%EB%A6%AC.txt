====== 개요 ======
사용 중인 배치의 정보를 관리할 수 있도록 한다.

====== 기능 설명 ======
  - 사용 중인 배치정보를 검색할 수 있다.
  - 새로운 배치정보를 등록할 수 있다.
  - 사용 중인 배치정보를 수정할 수 있다.
  - 사용 중인 배치정보를 삭제할 수 있다.
\\



====== 화면 설명 ======


===== 목록 화면 =====
{{:egovframework:bopr:가이드:배치정보:EgovBatchInfoList.png|}}

  * 목록 검색
    - 업무구분(코드ID:BO001)①을 선택한다.
    - 배치ID, Job이름(배치명) 검색조건②을 선택한다.
    - 검색 조건에 대해 검색할 검색어를 입력③한다. (검색조건②을 선택하면 입력창이 활성화 된다.)
    - 검색 버튼④을 클릭하거나 검색어 입력란③에서 키보드의 'Enter' 버튼을 클릭하여 검색 기능을 실행한다.

  * 등록 화면 호출
    - '등록' 버튼⑤을 클릭한다.

  * 배치 정보 삭제
    - 삭제할 데이터의 체크박스⑦를 선택한다. (목록 타이틀의 체크박스를 선택하면 전체체크/해제 기능이 실행된다.)
    - '삭제' 버튼⑥을 클릭한다.
    - FTP 비밀번호 입력 팝업에서 비밀번호 입력하고 '등록' 버튼을 클릭하거나 키보드의 'Enter' 버튼을 클릭한다.

  * 상세 정보 검색
    - 목록 화면에서 상세 정보를 검색할 데이터의 Job이름(배치명)⑧을 클릭한다.

  * 페이지 이동 ⑨
    - 숫자 클릭 : 해당 숫자의 페이지로 이동한다.
    - {{:egovframework:bopr:가이드:배치정보:icon_prev.png|}} : 이전 페이지목록(10Page 단위)로 이동한다.
    - {{:egovframework:bopr:가이드:배치정보:icon_next.png|}} : 다음 페이지목록(10Page 단위)로 이동한다.
    - {{:egovframework:bopr:가이드:배치정보:icon_prevend.png|}} : 맨 처음 페이지로 이동한다.
    - {{:egovframework:bopr:가이드:배치정보:icon_nextend.png|}} : 마지막 페이지로 이동한다.
    - 조회 건수가 100개 이하여서 페이지가 열개 이하인 경우 경우 화살표는 표시되지 않는다.

{{:egovframework:bopr:가이드:배치정보:egovftppwdpopup.png|}}

  * FTP 비밀번호 입력창에 비밀번호를 입력하고 '등록' 버튼을 클릭하여 비밀번호를 전송한다.

===== 등록 화면 =====
{{:egovframework:bopr:가이드:배치정보:EgovBatchInfoRegist.png|}}

  * 목록 이동
    - '목록' 버튼①을 클릭한다.

  * 배치정보 신규 등록
    - 배치설명③을 입력한다.
    - 업무구분④을 선택한다.(코드ID: BO014)
    - 온라인실행여부⑤를 선택한다.
    - 설정파일 등록 : '파일선택'(혹은 찾아보기) 버튼⑥을 클릭하여 등록할 설정파일을 선택한다. (반드시 형식에 맞는 xml 파일이어야 합니다.)
    - 설정파일 배포경로 설정 : 배포될 경로를 입력⑦한다.
    - 첨부파일 등록 : '파일선택'(혹은 찾아보기) 버튼⑧을 클릭하여 추가로 등록할 파일을 선택한다.
    - 첨부파일 배포경로 설정 : 배포될 경로를 입력⑨한다.
    - 첨부파일 추가 취소 : 'delete' 버튼⑩을 클릭하여 첨부된 파일을 삭제한다.
    - '등록' 버튼②을 클릭한다.
    - FTP 비밀번호 입력 팝업에서 비밀번호 입력하고 '등록' 버튼을 클릭하거나 키보드의 'Enter' 버튼을 클릭한다.

===== 상세 화면 =====
{{:egovframework:bopr:가이드:배치정보:EgovBatchInfoDetail.png|}}
  * 목록 이동
    - '목록' 버튼①을 클릭한다.

  * 배치정보 수정
    - 배치설명④을 입력한다.
    - 업무구분⑤을 선택한다.(코드ID: BO014)
    - 온라인실행여부⑥를 선택한다.
    - 설정파일 변경 : 설정파일의 '파일선택'(혹은 찾아보기) 버튼⑦을 클릭하여 등록할 설정파일을 선택한다. (반드시 형식에 맞는 xml 파일이어야 합니다.)
    - 설정파일 배포경로 변경 : 배포될 경로를 입력⑧한다.
    - 첨부파일 추가 : 첨부파일의 '파일선택'(혹은 찾아보기) 버튼⑨을 클릭하여 추가로 등록할 파일을 선택한다.
    - 첨부파일 배포경로 변경 : 배포될 경로를 입력⑩한다.
    - 첨부파일 삭제 : 'delete' 버튼⑪을 클릭하여 첨부된 파일을 삭제한다.
    - '수정' 버튼②을 클릭한다.
    - FTP 비밀번호 입력 팝업에서 비밀번호 입력하고 '등록' 버튼을 클릭하거나 키보드의 'Enter' 버튼을 클릭한다.

  * 배치정보 삭제
    - '삭제' 버튼③을 클릭한다.
    - FTP 비밀번호 입력 팝업에서 비밀번호 입력하고 '등록' 버튼을 클릭하거나 키보드의 'Enter' 버튼을 클릭한다.

=== 등록, 수정, 삭제 시 FTP 정보 ===

{{:egovframework:bopr:step11.settingglobalproperties.png|}}

  FTP의 정보는 '공통(사용자관리)>연동서비스관리>FTP연동서비스관리' 메뉴에서 변경할 수 있습니다.

  FTP의 정보의 'FTP연동번호'가 변경되었을 경우에는 globals.properties 파일의
  FTP.CFG.id, FTP.BATCH.id 속성을 변경하여 등록할 수 있습니다.

  FTP의 정보의 'FTP경로'는 FTP서버의 FTP홈을 기본으로 합니다.
  ex)FTP서버의 홈이 '/root/usr/' 인경우 FTP경로가 '/WEB-INF/resources/' 라고 한다면
  배치 파일은 'root/usr/WEB-INF/resources/' 하위에 저장됩니다..
