

====== 개요 ====== 
파일시스템 정보를 관리하기 위하여 모니터링도구관리자가 파일시스템를 등록/수정/상세조회/목록검색 할 수 있는 기능을 제공한다.

====== 기능 설명 ======

  - 파일시스템 정보 목록 : 파일시스템 정보 목록을 검색 조회한다.
  - 파일시스템 정보 상세 : 등록된 파일시스템에대한 정보를 상세 조회한다.
  - 파일시스템 정보 등록 : 식별된 파일시스템에대한 정보를 등록한다.
  - 파일시스템 정보 수정 : 등록한 파일시스템에대한 정보를 수정한다.
  

====== 화면 설명 ======

===== 파일시스템 정보 목록 =====
파일시스템 정보  목록은 페이지당 10건씩 조회되며 검색조건은 시스템명,운영체제유형,파일시스템명이다. 

{{:egovframework:oe2:가이드:어드민:filesystemlist.gif|}}

  * 검색 : 선택 또는 입력한 검색조건을 기반으로 등록한 파일시스템대한 정보를 목록으로 조회한다.
  * 점검 : 등록된 파일시스템에 대하여 사용률이 임계치를 초과여부에 대한 점검한다.
  * 등록 : 파일시스템 정보를 등록하기 위한 화면으로 이동한다.

===== DB서비스 정보 상세  =====
파일시스템 정보 목록에서 데이터소스명을 클릭하면 해당 DB서비스에 대한 정보를 상세 조회한다.

{{:egovframework:oe2:가이드:어드민:filesystemdetail.gif|}}

  * 목록 : 파일시스템 정보 목록 조회 화면으로 이동한다. 
  * 삭제 : 해당 파일시스템 정보를 삭제한다.
  * 수정 : 파일시스템 정보를 수정할 수 있는 편집 가능한 화면으로 이동한다.


===== 파일시스템 정보 등록 =====
파일시스템 정보를 등록할 수 있는 기능을 제공한다.

{{:egovframework:oe2:가이드:어드민:filesystemregist.gif|}}

  * 저장 : 입력한 파일시스템 정보를 등록한다.
  * 목록 : 파일시스템 정보 목록 조회 화면으로 이동한다.
  * 초기화 : 입력한 등록정보를 초기화한다.
  * **에이전트 URL이 입력되지 않을 경우 정상적인 동작이 되지 않을 수 있다.**

===== 파일시스템 정보  수정 =====
파일시스템 정보 상세조회 화면에서 수정 버튼을 클릭한 경우  해당 파일시스템에대한 관리정보를 수정할 수 있는 기능을 제공한다.

{{:egovframework:oe2:가이드:어드민:filesystemupdate.gif|}}

  * 저장 : 입력한 파일시스템 정보를 수정한다.
  * 목록 : 파일시스템 정보 목록 조회 화면으로 이동한다.
  * 초기화 : 입력한 수정정보를 초기화한다.
