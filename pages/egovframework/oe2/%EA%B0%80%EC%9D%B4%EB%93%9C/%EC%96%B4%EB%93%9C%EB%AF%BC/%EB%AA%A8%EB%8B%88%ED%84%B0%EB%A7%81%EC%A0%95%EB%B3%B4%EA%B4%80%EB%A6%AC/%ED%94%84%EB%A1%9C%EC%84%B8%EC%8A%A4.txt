

====== 개요 ====== 
프로세스 정보를 관리하기 위하여 모니터링도구관리자가 프로세스를 등록/수정/상세조회/목록검색 할 수 있는 기능을 제공한다.

====== 기능 설명 ======

  - 프로세스 정보 목록 : 프로세스 정보 목록을 검색 조회한다.
  - 프로세스 정보 상세 : 등록된 프로세스에대한 정보를 상세 조회한다.
  - 프로세스 정보 등록 : 식별된 프로세스에대한 정보를 등록한다.
  - 프로세스 정보 수정 : 등록한 프로세스에대한 정보를 수정한다.
  

====== 화면 설명 ======

===== 프로세스 정보 목록 =====
프로세스 정보  목록은 페이지당 10건씩 조회되며 검색조건은 프로세스명, 시스템명이다. 

{{:egovframework:oe2:가이드:어드민:processinfolist.gif|}}

  * 검색 : 선택 또는 입력한 검색조건을 기반으로 등록한 프로세스에대한 정보를 목록으로 조회한다.
  * 등록 : 프로세스 정보를 등록하기 위한 화면으로 이동한다.


===== 프로세스 정보 상세  =====
프로세스 정보 목록에서 프로세스명을 클릭하면 해당 프로세스에 대한 정보를 상세 조회한다.

{{:egovframework:oe2:가이드:어드민:processinfodetail.gif|}}

  * 수정 : 프로세스 정보를 수정할 수 있는 편집 가능한 화면으로 이동한다.
  * 목록 : 프로세스 정보 목록 조회 화면으로 이동한다. 


  *** 모니터링 도구(에이전트) 설치 위치가 Windows일 경우 정상적으로 지원되지 않는다. **


===== 프로세스 정보 등록 =====
모니터링 대상 프로세스을 등록할 수 있는 기능을 제공한다.

{{:egovframework:oe2:가이드:어드민:processinforegist.gif|}}

  * 저장 : 입력한 프로세스 정보를 등록한다.
  * 목록 : 프로세스 정보 목록 조회 화면으로 이동한다.

===== 프로세스 정보  수정 =====
프로세스 정보 상세조회 화면에서 수정 버튼을 클릭한 경우 해당 프로세스에대한 관리정보를 수정할 수 있는 기능을 제공한다.

{{:egovframework:oe2:가이드:어드민:processinfoupdate.gif|}}

  * 저장 : 입력한 프로세스 정보를 수정한다.
  * 목록 : 프로세스 정보 목록 조회 화면으로 이동한다.
