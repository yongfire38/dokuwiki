

====== 개요 ====== 
컨테이너 정보를 관리하기 위하여 모니터링 관리자가 컨테이너 정보를 등록/수정/상세조회/목록검색 할 수 있는 기능을 제공한다.

====== 기능 설명 ======

  - 컨테이너 정보 목록 : 컨테이저 정보 목록을 검색 조회한다.
  - 컨테이너 정보 상세 : 등록된 컨테이너에 대한 정보를 상세 조회한다.
  - 컨테이너 정보 등록 : 모니터링 대상 컨테이너에 대한 정보를 등록한다.
  - 컨테이너 정보 수정 : 등록한 컨테이너에 대한 정보를 수정한다. 

====== 화면 설명 ======

===== 컨테이너 정보 목록 =====
컨테이너 정보 목록은 페이지당 10건씩 조회되며 검색조건은 **컨테이너명**, **서버명**, **WAS종류**이다. 

{{:egovframework:oe2:가이드:어드민:containerlist.gif|}}

  * 검색 : 선택 또는 입력한 검색조건을 기반으로 등록한 컨테이너 정보를 목록으로 조회한다.
  * 등록 : 신규 컨테이너 정보를 등록하기 위한 화면으로 이동한다.


===== 컨테이너 정보 상세  =====
컨테이너 정보 목록에서 컨테이너명을 클릭하면 해당 컨테이너에 대한 정보를 상세 조회한다.

{{:egovframework:oe2:가이드:어드민:containerdetail.gif|}}

  * 목록 : 컨테이너 정보 목록 조회 화면으로 이동한다. 
  * 삭제 : 해당 컨테이너 정보를 삭제한다. 
  * 수정 : 컨테이너 정보를 수정할 수 있는 편집 가능한 화면으로 이동한다.


===== 컨테이너 정보 등록 =====
컨테이너 정보를 등록할 수 있는 기능을 제공한다.

{{:egovframework:oe2:가이드:어드민:containerregist.gif|}}

  * 저장 : 컨테이너 정보를 등록한다.
  * 목록 : 컨테이너 정보 목록 조회 화면으로 이동한다.
  * 초기화 : 입력한 컨테이너 정보를 초기화한다.

==== 컨테이너 정보 등록시 주의사항 ====
=== 1.컨테이너명 ===
WAS의 종류가 JEUS일 경우 WAS에 등록되어 있는 컨테이너명을 확인하여 입력해야 한다.

Ex) JEUSMain.xml의 node명이 **was3**이고 engine-container명이 **oe2ex1**일 경우 컨테이너명은 **was3_oe2ex1**로 등록해야 한다.
<code:XML>
<?xml version="1.0" encoding="UTF-8"?>
<jeus-system version="6.0" xmlns="http://www.tmaxsoft.com/xml/ns/jeus">
<node>
   <name>was3</name>
       <engine-container>
           <name>oe2ex1</name>
                  ...
       </engine-container>
         ...
</node>
</code>

WAS의 종류가 WebLogic일 경우 WAS에 생성한 **서버명**을 **컨테이너명**으로 등록하여야 한다.(WebLogic은 서버단위가 컨테이너 단위임)  

=== 2.RMI포트 ===
RMI포트는 에이전트 설치에서 설정한 포트번호를 입력한다. [[egovframework:oe2:설치:에이전트_설치|에이전트 설치]]참조

=== 3.JNDI포트 ===
JNDI포트는 모니터링 대상서버가 JBoss, Jeus, WebLogic일 경우에만 설정하도록 구성되어 있다.

== * WAS별 JNDI 포트 확인법 ==
|Jboss | conf/jboss-service.xml에서 JNDI포트 확인.|
|Jeus | vhost.properties 파일에서 JNDI포트 확인.|
|WebLogic | 서버실행 Shell에 등록되어있는 t3 포트나 admin포트.|

===== 컨테이너 정보 수정 =====
컨테이너 정보 상세화면에서 수정 버튼을 클릭한 경우  해당 컨테이너 정보를 수정할 수 있는 기능을 제공한다.

{{:egovframework:oe2:가이드:어드민:containerupdate.gif|}}
 
  * 저장 : 수정한 컨테이너 정보를 저장한다.
  * 목록 : 컨테이너 정보 목록 조회 화면으로 이동한다.
  * 초기화 : 입력한 컨테이너 정보를 초기화한다.
