

====== 개요 ====== 
모니터링 대상 서버를 관리하기 위하여 모니터링 관리자가 서버정보를 등록/수정/상세조회/목록검색 할 수 있는 기능을 제공한다.

====== 기능 설명 ======

  - 서버 정보 목록 : 서버정보 목록을 검색 조회한다.
  - 서버 정보 상세 : 등록된 서버에 대한 정보를 상세 조회한다.
  - 서버 정보 등록 : 모니티렁 대상 서버에대한 정보를 등록한다.
  - 서버 정보 수정 : 등록한 서버에대한 정보를 수정한다.  

====== 화면 설명 ======

===== 서버 정보 목록 =====
서버 정보 목록은 페이지당 10건씩 조회되며 검색조건은 **서버명**, **서버IP**, **WAS종류**이다. 

{{:egovframework:oe2:가이드:어드민:serverlist.gif|}}

  * 검색 : 선택 또는 입력한 검색조건을 기반으로 등록한 서버에대한 정보를 목록으로 조회한다.
  * 등록 : 신규 서버 정보를 등록하기 위한 화면으로 이동한다.


===== 서버 정보 상세  =====
서버 정보 목록에서 서버명을 클릭하면 해당 서버에 대한 정보를 상세 조회한다.

{{:egovframework:oe2:가이드:어드민:serverdetail.gif|}}

  * 목록 : 서버 목록 조회 화면으로 이동한다. 
  * 삭제 : 해당 서버 정보를 삭제한다.
  * 수정 : 서버 정보를 수정할 수 있는 편집 가능한 화면으로 이동한다.
  * 서버관리자아이디: 등록한 어플리케이션 서버의 관리자 아이디를 등록한다.
  * 서버관리자패스워드: 어플리케이션 서버의 관리자 패스워드를 등록한다.

===== 서버 정보 등록 =====
신규 서버 정보를 등록할 수 있는 기능을 제공한다.

{{:egovframework:oe2:가이드:어드민:serverregist.gif|}}

  * WAS종류 : Jboss, Jeus, Tomcat, WebLogic을 지원한다.
  * 저장 : 입력한 서버 정보를 등록한다.
  * 목록 : 서버 정보 목록 조회 화면으로 이동한다.
  * 초기화: 입력한 정보를 초기화한다.

===== 서버 정보 수정 =====
서버 정보 상세조회 화면에서 수정 버튼을 클릭한 경우 해당 서버에 대한 정보를 수정할 수 있는 기능을 제공한다.

{{:egovframework:oe2:가이드:어드민:serverupdate.gif|}}

  * WAS종류 : Jboss, Jeus, Tomcat, WebLogic을 지원한다.
  * 저장 : 입력한 서버 정보를 등록한다.
  * 목록 : 서버 정보 목록 조회 화면으로 이동한다.
  * 초기화: 입력한 정보를 초기화한다.
