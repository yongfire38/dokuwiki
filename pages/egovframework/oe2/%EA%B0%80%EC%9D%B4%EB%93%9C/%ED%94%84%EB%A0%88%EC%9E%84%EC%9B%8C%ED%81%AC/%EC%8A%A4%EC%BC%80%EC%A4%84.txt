====== 개요 ====== 
프레임워크 스케줄 페이지는 전자정부 프레임워크 실행환경의 Scheduling서비스에 대한 모니터링 정보를 제공한다.

실행환경 Scheduling 서비스는 오픈소스 소프트웨어로 Quartz 스케쥴러를 사용하며, Scheduling 서비스 어플리케이션 서버 내에서 주기적으로 발생하거나 반복적으로 발생하는 작업을 지원하는 기능을 한다. 
\\
  * 실행환경 [[eGovframework:RTE:FDL:Scheduling]] 서비스




\\

====== 기능 설명 ======
  - 스케줄 정보 목록 : 프레임워크의 스케줄 정보 목록을 보여줍니다.

\\
====== 화면 설명 ======

===== 스케줄 정보 목록 =====
프레임워크 스케줄 정보 목록을 제공한다. 검색조건은 **서버명**, **컨테이너명**이다.

{{:egovframework:oe2:가이드:프레임워크:scheduleinfo.gif|}}

  * 검색 : 서버명과 컨테이너명을 조건으로 스케줄 정보를 등록한다.
  * 작업명 : 스케줄 작업명을 관리하는 팝업 화면을 보여준다.

  * 스케줄명 : 등록되 스케줄명을 보여준다.
  * 작업명 : DB에 등록된 스케줄에 대한 스케줄 작업명을 보여준다.
  * 작업클래스 : 스케줄잡이 실행되는 작업 클래스를 보여준다.
  * 트리거유형 : 스케줄의 티리거 유형을 보여준다.
  * 실행유형 : 스케줄잡의 싱행주기를 보여준다.

===== 스케줄 작업명 목록 팝업 =====
스케줄 작업명 목록은 페이지당 10건씩 조회되며 검색조건은 **작업영문명**, **작업한글명**이다. 
\\
{{:egovframework:oe2:가이드:프레임워크:jobNmList.jpg|}}

  * 등록 : 스케줄 작업명을 등록하기 위한 화면으로 이동한다.
  * 닫기 : 스케줄 작업명 목록 팝업을 종료한다.

===== 스케줄 작업명 등록 팝업 =====
스케줄 작업명을 등록할 수 있는 기능을 제공한다.

{{:egovframework:oe2:가이드:프레임워크:jobNmRegist.jpg|}}

  * 저장 : 입력한 스케줄 작업명을 등록한다.
  * 목록 : 스케줄 작업명 목록 팝업 화면으로 이동한다.
  * 초기화 : 입력한 스케줄 작업명을 초기화한다.

===== 스케줄 작업명 수정 팝업 =====
스케줄 작업명 목록 팝업 화면에서 작업명을 클릭한 경우  해당 사용자에 대한 관리정보를 수정할 수 있는 기능을 제공한다.

{{:egovframework:oe2:가이드:프레임워크:jobNmUpdate.jpg|}}

  * 저장 : 입력한 스케줄작업명을 수정한다.
  * 목록 : 스케줄 작업명 목록 팝업 화면으로 이동한다.
  * 초기화 : 입력한 스케줄 작업명을 초기화한다.
  * 삭제 : 해당 스케줄 작업명을 삭제한다.
