====== 개요 ====== 
OS 런타임 모니터링 정보 화면은 OS정보(OS 벤더,프로세스 수),JVM정보를 제공한다.

OS, JVM, 컴파일러 정보를 실시간으로 확인 가능하며, 상세 수치를 구체적으로 확인할 수 있다.\\
OS, JVM, 컴파일러 정보는 어드민에서 설정하는 RMI 포트를 통해 정보를 가져오며, JConsole 상에서도 동일한 OS 및 Runtime 정보를 얻을 수 있다.\\


====== 기능 설명 ======
  - OS 런타임 모니터링 목록 : OS 런타임 모니터링 수집 정보의 목록을 보여준다.

\\

====== 화면 설명 ======

===== HTTP 서비스 목록 =====
OS와 런타임 정보를 나타내는 화면이다. 검색 조건은 서버명과 컨테이너명이다.

{{:egovframework:oe2:가이드:프레임워크:osruntime.jpg|}}


  * 검색 : 선택 또는 입력한 검색조건을 기반으로 서비스 목록을 조회한다.
  * 리프레쉬 : 최신 모니터링 정보를 재조회한다.

==== 화면 검색 용어 설명 ====
  * Operating System : 운영환경의 종류 및 버젼
  * Architecture : 아키텍쳐
  * 프로세스의 수 : 마지막으로 CPU 데이터가 수집되었던 때의 시간
  * Java Virtual Machine : 자바 가상 머신(JVM)의 종류
  * 벤더 : 사용하고 있는 JVM의 제작 회사
  * JIT 컴파일러 : 자바언어로 만들어진 프로그램을 해석해서 실행시켜주는 소프트웨어

==== 화면 상세 정보 ====

  * OS 정보, JVM 정보, 컴파일러 정보에 대하여 정보를 제공한다.
  * 서버 별, 컨테이너 별 OS 런타임 모니터링 정보를 사용자에게 제공한다.
