====== 개요 ====== 
프레임워크 적용 클래스 페이지는 프레임워크 적용 클래스 모니터링 정보를 제공합니다.

\\
====== 기능 설명 ======
  - 프레임워크 적용 클래스 모니터링 정보 목록 : 프레임워크 적용 클래스 모니터링 정보 목록을 보여줍니다.

\\
====== 화면 설명 ======

===== 프레임워크 적용 클래스 모니터링 정보 목록 =====
전자정부 프레임워크 적용 클래스를 모니터링 하는 화면이다. 검색조건은 서버명, 컨테이너명이다.

{{:egovframework:oe2:가이드:프레임워크:모델.jpg|}}


  * 검색 : 선택 또는 입력한 검색조건을 기반으로 서비스 목록을 조회한다.

==== 화면 상세 정보 ====
  * 모니터링 대상인 어플리케이션의 환경에서 IBM JDK를 사용한 경우에는 클래스의 정보를 모니터링 할 수 없다. 이 경우, SUN JDK를 사용하기를 귄장한다.
  * 컨테이너 별 전자정부 프레임워크가 적용된 클래스의 모니터링 정보를 사용자에게 제공한다.
