====== 개요 ====== 
로딩 클래스 모니터링 페이지는 WAS상에 로딩된 클래스 정보를 제공한다.

로딩 클래스 모니터링 정보 목록 화면을 통하여 실시간으로 반영되는 로딩 클래스의 최신정보를 확인할 수 있다. 로딩된 클래스의 추이를 실시간으로 확인 가능하다.\\
로딩 클래스 모니터링 정보는 어드민 설정 중 RMI 포트를 통해 데이터를 가져오며, JConsole 에서도 동일한 정보를 확인 할 수 있다.\\


====== 기능 설명 ======
  - 로딩 클래스 모니터링 정보 목록 : 로딩 클래스의 모니터링 정보 목록을 보여준다.

\\

====== 화면 설명 ======

===== 로딩 클래스 모니터링 정보 목록 =====
로딩클래스의 정보를 보여주는 화면이다. 검색조건은 서버명, 컨테이너명이다.

{{:egovframework:oe2:가이드:프레임워크:로딩클래스.jpg|}}


  * 검색 : 선택 또는 입력한 검색조건을 기반으로 모니터링 목록을 조회한다.
  * 리프레쉬 : 최신 모니터링 정보를 재전송한다.


==== 화면 검색 용어 설명 ====
  * Total Loaded 클래스 수 : 전체 로딩된 클래스의 수. Total Class란 그래프에서 쓰이는 용어
  * Loaded 클래스 수 : 로드된 클래스의 수로. 전체 클래스의 수에 로드되지 않은 클래스의 수를 빼면 로드된 클래스의 수를 구할 수 있음. Loaded Class란 그래프에서 쓰이는 용어
  * UnLoaded 클래스 수 : 로드되지 않은 클래스 수. Unloaed Class란 그래프에서 쓰이는 용어

==== 화면 상세 정보 ====
  * JFreeChar를 이용하여 현재 로딩된 클래스 수 대한 시간 별 퍼센트 그래프가 그려졌다.
  * 컨테이너 별 로딩된 클래스 모니터링 정보를 사용자에게 제공한다.
  * 당일 로딩 클래스 모니터링 정보가 있는 컨테이너만 화면에 조회된다.
