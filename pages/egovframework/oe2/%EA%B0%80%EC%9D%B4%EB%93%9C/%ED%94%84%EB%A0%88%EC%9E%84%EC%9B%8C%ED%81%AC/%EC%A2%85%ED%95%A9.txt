====== 개요 ====== 
WAS 종합 화면은 프레임워크를 사용하는 WAS의 cpu,memory, thread, session 등의 성능정보와 jdbc pool 정보 등 Framework 화면에서 제공하는 항목들의 요약정보를 보여준다.

본 WAS 종합 화면에서는 프레임워크(CPU, 메모리, 쓰레드) 및 JDBC POOL 의 4가지 데이터를 서버와 컨테이너 별로 보여준다.\\
오픈소스 JFreeChart를 사용하여 실시간 모니터링 정보를 화면에 보여주며, 이를 통해 사용자가 설정한 임계치의 초과 여부를 쉽게 파악할 수 있다.


\\

====== 기능 설명 ======
  - WAS 종합 화면 : 프레임워크를 사용하면서 모니터링이 필요한 항목들에 대한 요약정보를 실시간으로 보여주는 화면이다.

\\
====== 화면 설명 ======

===== WAS 종합 화면 =====
{{:egovframework:oe2:가이드:프레임워크:was메인.jpg|}}

  * 리프레쉬 시간 : 리프레쉬 시간을 설정하여 주기적으로 메인화면을 재조회한다.
  * 리프레쉬 : 사용자가 수동적으로 화면을 재조회한다.
  * 검색 : WAS 메인 화면은 서버와 컨테이너 단위로 데이터가 조회된다.
  * uptime : 컨테이너의 기동시간. 컨테이너 별로 새로 기동한 시간을 나타낸다.
==== 화면 상세 정보 ====

  * WAS 종합 화면은 실시간 화면으로, 조회되는 시간은 WAS 종합 화면 타이틀 옆에 표시된다. 
  * 서버 별, 컨테이너 별 모니터링 상태를 상태 표시를 하여 사용자에게 제공한다.
  * 어드민 설정에서 서버와 컨테이너를 설정하였으나, 컨테이너가 기동되지 않은 경우는 그래프가 표시되지 않는다.
  * 설정한 컨테이너가 기동되어 있으나, 모니터링 설정 여부가 off로 된 경우는 그래프 형태만 나오고 임계치 및 데이터는 표시되지 않는다. 
  * 임계치는 현재 데이터가 있는 경우에만 표시된다.
