====== 개요 ====== 
최근 서비스 현황은 모니터링된 것 중 가장 최근의 5개( 설정에서 변경가능 )를 보여주는 기능입니다. HTTP 서비스를 기준으로 보여주며 상세세이지에서 HTTP 요청에대한 트리정보, HTTP 서비스 요약, 어플리케이션 서비스 요약, JDBC 서비스 요약, SQL 서비스 요약 정보를 확인할 수 있다.

\\
====== 기능 설명 ======
  - 최근 서비스 현황 목록 : 최근 모니터링 된 목록을 보여준다.
  - 최근 서비스 현황 상세정보 : 모니터링의 상세정보를 보여준다.

\\
====== 화면 설명 ======

===== 최근 서비스 현황 목록 =====
{{:egovframework:oe2:가이드:어플리케이션:EgovSequenceList.png}}

  * 전체(inclusive) mode : 해당 레이어를 수행하는데 걸린 시간( 다른 레이어의 수행시간은 포함 )
  * 단계(exclusive) mode : 해당 레이어만 수행하는데 걸린 시간( 다른 레이어의 수행시간은 제외 )

  * URL      : 실행된 HTTP 요청의 URL을 보여준다.
  * 총시간   : HTTP 요청의 총 실행시간을 보여준다.
  * 호출갯수 : HTTP 요청이 몇 번이아 호출 되었는지를 보여준다.
  * Avg      : 평균실행시간을 보여줍니다.총실행시간 / 호출갯수
  * Adj.Avg  : 처음 실행시간을 제외한 평균을 보여준다. ( 총실행시간 - 최초실행시간 ) / ( 호출갯수 - 1 )
  * Min      : HTTP요청의 실행시간이 가장 짧았을 때의 실행시간을 보여준다.
  * Max      : HTTP요청의 실행시간이 가장 기었을 때의 실행시간을 보여준다.
  * First    : HTTP요청을 처음 실행했을 때의 값을 보여준다.
  * Last     : HTTP요청을 마지막 실행했을 때의 값을 보여준다.

\\

===== 최근 서비스 현황 상세정보 =====
최근 서비스 현황 목록에서 대상 요청 URL 부분을 클릭하면 최근 서비스 현황 상세정보 화면에서 선택된 대상의 상세정보를 확인할 수 있다.

{{:egovframework:oe2:가이드:어플리케이션:EgovSequenceDetail.png}}

  * 트리구조         : 최근 서비스 현황의 실행 정보를 트리 구조로 보여준다.

  * HTTP 서비스 요약 : 위의 트리구조 중 HTTP 서비스의 요약정보를 보여준다.
  * 어플리케이션 서비스 요약 : 위의 트리구조 중 어플리케이션 서비스의 요약정보를 보여준다.
  * JDBC 서비스 요약 : 위의 트리구조 중 JDBC 서비스의 요약정보를 보여준다.
  * SQL 서비스 요약  : 위의 트리구조 중 SQL 서비스의 요약정보를 보여준다.
