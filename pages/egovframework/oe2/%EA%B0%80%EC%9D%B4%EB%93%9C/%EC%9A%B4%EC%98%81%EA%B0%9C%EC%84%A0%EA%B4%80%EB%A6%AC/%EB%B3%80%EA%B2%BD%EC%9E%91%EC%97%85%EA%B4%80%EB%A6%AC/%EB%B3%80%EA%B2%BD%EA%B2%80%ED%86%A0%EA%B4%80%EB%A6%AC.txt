


====== 개요 ====== 
관리자가 각 변경담당자들이 승인을 요청한 검토요청건에 대해 검토/상세조회/목록검색 할 수 있는 기능을 제공한다.

====== 기능 설명 ======
  - 변경검토 요청 건 목록 : 변경검토 요청 건 목록을 검색조회한다.
  - 변경작업계획검토 : 관리자가 각 변경담당자들이 승인요청한 변경작업계획을 검토승인한다.
  - 변경작업완료검토 : 관리자가 각 변경담당자들이 승인요청한 변경작업을 검토승인한다.
  - 변경요청서 상세 팝업 : 변경요청처리에 묶인 변경요청서의 상세 정보를 조회한다.  

====== 화면 설명 ======

===== 변경검토 요청 건 목록 =====
변경검토 요청 건 목록은 페이지당 10건씩 조회되며 검색조건은 **검토요청구분**, **긴급처리**, **접수일**이다. 



  * 검색 : 선택 또는 입력한 검색조건을 기반으로 변경검토 요청 건 목록 목록을 조회한다.



===== 변경작업계획검토  =====
변경검토 요청 건 목록에서 처리상태가 계획검토인 것을 클릭하면 관리자가 각 변경담당자들이 승인요청한 변경작업계획을 검토승인할 수 있는 기능을 제공한다.


  * 변경요청서조회 : 변경요청처리에 묶인 변경요청서 정보를 조회한다.
  * 검토이력조회 : 해당 요청건데 대해 반려된 이력을 조회한다.
  * 계획검토 : 해당 검토건을 계획검토결과에 따라 반려/승인 한다.
  * 목록 : 변경검토 요청 건 목록 조회 화면으로 이동한다. 


===== 변경작업완료검토  =====
변경검토 요청 건 목록에서 처리상태가 완료검토인 것을 클릭하면 관리자가 각 변경담당자들이 승인요청한 변경작업을 검토승인 할 수 있는 기능을 제공한다.


  * 변경요청서조회 : 변경요청처리에 묶인 변경요청서 정보를 조회한다.
  * 검토이력조회 : 해당 요청건데 대해 반려된 이력을 조회한다.
  * 완료검토 : 해당 검토건을 완료검토결과에 따라 반려/승인 한다.
  * 목록 : 변경검토 요청 건 목록 조회 화면으로 이동한다. 



===== 변경요청서 상세팝업 =====
변경요청처리에 묶인 변경요청서 정보를 상세 조회한다.


  * 요청근거 : 변경요청서를 등록한 요청시스템의 출처 정보를 조회한다.
  * 닫기 : 화면을 종료한다.
