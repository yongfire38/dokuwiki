====== 공통컴포넌트 적용율 데이터 관리 ======

공통컴포넌트 적용율 데이터 관리는 FP적용율 분석과 산출내역서 작성 시 필요한 초기 데이터들을 관리하는 기능이다. 여기서 입력한 값들은 FP적용율을 분석하거나 산출내역서를 작성할 때 초기값 또는 계산시 쓰이는 상수값으로 입력되게 된다. 기초 데이터들이 입력되지 않은 겨우 해당 항목을 작성할 때 값들은 0으로 초기화되어 입력되게 된다. 그러나 FP적용율을 분석할 때 혹은 산출내역서를 작성할 때 반복작업을 줄여주고, 정확한 내역을 생성하기 위해 필요하기 때문에 해당 항목들을 수행하기 전에 선행작업으로 값을 입력해주는 것이 좋다.



===== 기능점수 데이터 관리 ======

기능점수 데이터 관리는 기능점수 목록을 출력하기 위하여 필요한 기초데이터들을 관리하는 기능이다. 여기서 입력한 데이터들은 FP 적용률 분석을 실시할 때, 임의의 값이 입력되지 않은 항목들의 초기값으로 입력되게 된다. 만약 기능점수 데이터가 저장되지 않은 경우 FP 적용률을 분석할 때는 모든 값이 0으로 초기화되어 입력되게 된다. 

{{:egovframework:oe1:적용점검:가이드:공통컴포넌트_적용율_데이터_관리:적용점검_공통컴포넌트_적용율데이터관리_기능점수데이터관리1.png|}}

기능점수 데이터가 입력되지 않은 상태 혹은 입력된 기능점수가 삭제된 상태가 되면 기능점수 데이터를 새로 입력하는 등록화면이 나오게 된다. 저장버튼을 누르면 입력한 기능점수 데이터가 저장되어 초기데이터로 활용할 수 있게 된다.

{{:egovframework:oe1:적용점검:가이드:공통컴포넌트_적용율_데이터_관리:적용점검_공통컴포넌트_적용율데이터관리_기능점수데이터관리2.png|}}

이미 기능점수가 데이터가 입력된 상태라면 해당 데이터를 조회할 수 있는 조회화면으로 이동한다. 이 상태에서 기능점수를 삭제하거나 수정할 수 있다.

{{:egovframework:oe1:적용점검:가이드:공통컴포넌트_적용율_데이터_관리:적용점검_공통컴포넌트_적용율데이터관리_기능점수데이터관리3.png|}}

이미 입력된 기능점수 데이터는 수정할 수 있다. 수정 후 저장버튼을 누르면 수정된 데이터가 저장된다.


===== 산출내역서 데이터 관리======

산출내역서 기초데이터는 산출내역서를 작성할 때 필요한 기초데이터와 단가 등 상수값을 입력하는 기능이다.

{{:egovframework:oe1:적용점검:가이드:공통컴포넌트_적용율_데이터_관리:적용점검_공통컴포넌트_적용율데이터관.png|}}

산출내역서 기초데이터 관리에서는 조회, 수정, 등록 화면을 한 화면에서 처리하게 된다. 산출내역서 산정에 필요한 기초값들을 입력하고 저장버튼을 누르면 입력한 값들이 저장되어서 산출내역서 작성시 활용할 수 있게 되며, 이미 입력된 데이터의 수정도 같은 방식으로 할 수 있다.
