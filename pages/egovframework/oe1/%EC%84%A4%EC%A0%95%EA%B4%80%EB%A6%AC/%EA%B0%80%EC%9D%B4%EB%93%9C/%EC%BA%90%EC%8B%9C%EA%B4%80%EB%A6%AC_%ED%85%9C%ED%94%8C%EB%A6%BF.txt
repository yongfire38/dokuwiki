====== 캐시 관리(템플릿) ======
===== 1. 개요 ======
캐시관리는 전자정부표준프레임워크 실행환경에서 제공하는 캐시 서비스를 실제 소스 레벨에서 사용하기 위한 가이드를 소스 템플릿으로 제공하여 캐시 서비스를
쉽게 이용할 수 있게 도와준다.

캐시 서비스에 대한 자세한 내용은 [[https://www.egovframe.go.kr/wiki/doku.php?id=egovframework:rte:fdl:cache]] 참고한다.

==== 캐시관리(템플릿) ====
캐시에 대한 등록,수정,삭제,리로딩에 대한 기능을 구현할 수 있는 소스 템플릿을 미리보기로 제공한다.

=== 캐시 등록 ===
설정관리 도구 메인화면 >> 좌측 메뉴 >> 캐시관리(템플릿) >> 캐시등록

    * 설정관리 도구 표현
아래와 같이 캐시 등록을 위한 값을 입력 받는다.

{{:egovframework:oe1:설정관리:가이드:캐시등록.jpg|}}
    * 버튼 설명
    - Element 추가: 캐시안에 Key, Value 쌍으로 들어가있는 Element 를 추가할 경우 사용
    - Element 삭제: 캐시안에 Key, Value 쌍으로 들어가있는 Element 를 삭제할 경우 사용
    - 코드템플릿 생성: 입력 받은 데이터를 이용하여 캐시 등록 코드 템플릿 생성
