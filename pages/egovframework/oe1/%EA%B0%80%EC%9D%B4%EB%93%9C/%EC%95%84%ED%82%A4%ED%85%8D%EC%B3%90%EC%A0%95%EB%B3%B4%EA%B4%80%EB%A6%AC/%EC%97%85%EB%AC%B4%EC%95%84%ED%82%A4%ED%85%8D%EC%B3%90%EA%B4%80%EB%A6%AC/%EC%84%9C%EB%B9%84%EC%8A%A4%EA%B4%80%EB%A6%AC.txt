

====== 개요 ====== 
현재 사용하고 있거나 사용예정인 서비스에 관한 정보를 등록/수정/삭제/관리하는 기능을 제공한다.

====== 기능 설명 ======
  - 서비스정보 목록 : 서비스정보 목록을 검색 조회한다.
  - 서비스정보 상세 : 서비스정보를 상세 조회한다.
  - 서비스정보 등록 : 서비스정보를 등록한다.
  - 서비스정보 수정 : 서비스정보를 수정한다.
  

====== 화면 설명 ======

===== 서비스 목록 =====
등록된 서비스의 목록을 검색조회한다. 서비스 목록은 페이지당 10건씩 조회되며 검색조건은 **서비스명**, **제공자** 이다. 


{{:egovframework:oe1:가이드:아키텍쳐정보관리:업무아키텍쳐관리:서비스정보목록.jpg|}}


  * 검색 : 선택 또는 입력한 검색조건을 기반으로 서비스 목록을 조회한다.
  * 등록 : 새로운 서비스정보를 등록하기 위한 화면으로 이동한다.



===== 서비스 상세  =====
서비스정보를 상세조회한다. 서비스 목록화면에서 서비스명을 클릭하면 해당 서비스 정보를 상세 조회한다.


{{:egovframework:oe1:가이드:아키텍쳐정보관리:업무아키텍쳐관리:서비스정보상세.jpg|}}


  * 수정 : 서비스정보를 수정할 수 있는 편집 가능한 화면으로 이동한다.
  * 삭제 : 서비스를 삭제한다.
  * 목록 : 서비스 목록 조회 화면으로 이동한다. 



===== 서비스 등록 =====
사용하고자하는 서비스정보를 등록하는 기능을 제공한다.


{{:egovframework:oe1:가이드:아키텍쳐정보관리:업무아키텍쳐관리:서비스정보등록.jpg|}}


  * 저장 : 입력한 서비스정보를 등록한다. 
  * 초기화 : 모든 입력항목을 초기화한다.
  * 목록 : 서비스 목록 조회 화면으로 이동한다. 



===== 서비스 수정 =====
서비스정보 상세 조회 화면에서 수정 버튼을 클릭하여 서비스 정보를 수정한다.


{{:egovframework:oe1:가이드:아키텍쳐정보관리:업무아키텍쳐관리:서비스정보수정.jpg|}}


  * 저장 : 입력한 서비스정보를 수정한다. 
  * 목록 : 서비스 목록 화면으로 이동한다.
