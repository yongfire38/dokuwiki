

====== 개요 ====== 
화면 아키텍쳐구조를 화면별로 관리하기 위하여 화면의 등록/수정/상세조회/목록검색 할 수 있는 기능을 제공한다.

아키텍쳐 정보수집을 통해 저장된 어노테이션 정보를 해당 화면에 매핑하여 화면에서 사용하는 기능의 클래스/메서드/쿼리/파라미터 정보를 상세 조회할 수 있다.

*컴포넌트관리, 아키텍쳐정보조회, 화면관리 메뉴는 이클립스 Plug-in의 아키텍쳐 정보수집을 실행 후 사용하실 수 있습니다.

====== 기능 설명 ======
  - 화면정보 목록 : 화면의 목록을 검색 조회한다.
  - 화면정보 상세/수정 : 화면 정보를 상세조회/수정한다.
  - 화면정보 등록 : 화면 정보를 등록한다.
  - 메서드 조회 팝업 : 등록된 메서드 목록을 조회한다.
  

====== 화면 설명 ======

===== 화면정보관리 목록 =====
화면 목록은 페이지당 10건씩 조회되며 검색조건은 **시스템명**, **업무명**, **화면명**, **화면영문명**이다. 
*참고 : 화면관리의 시스템명은 OE1031 업무명은 OE1030에 정보를 코드로 관리하여 사용한다.

{{:egovframework:oe1:가이드:아키텍쳐정보관리:화면아키텍쳐관리:화면정보목록.jpg|}}

  * 검색 : 선택 또는 입력한 검색조건을 기반으로 화면 목록을 조회한다.
  * 등록 : 화면을 등록하기 위한 화면으로 이동한다.



===== 화면정보관리 상세/수정  =====
화면정보 목록에서 화면정보를 클릭하면 해당 화면정보를 상세 조회한다.

{{:egovframework:oe1:가이드:아키텍쳐정보관리:화면아키텍쳐관리:화면정보상세.jpg|}}

  * 저장 : 화면정보를 수정한다.
  * 삭제 : 화면정보를 삭제한다.
  * 목록 : 화면정보 목록 조회 화면으로 이동한다. 
  * 추가 : 메서드를 입력하기 위한 ROW를 증가시킨다.
  * 삭제 : 메서드를 입력하는 ROW를 감소시킨다.
  * 검색 : 메서드 목록을 조회하는 팝업창을 호출한다.



===== 화면정보 등록 =====
화면정보를 등록하는 기능을 제공한다.

{{:egovframework:oe1:가이드:아키텍쳐정보관리:화면아키텍쳐관리:화면정보등록.jpg|}}

  * 저장 : 입력한 화면정보등록한다. 
  * 목록 : 화면정보 목록 조회 화면으로 이동한다. 
  * 추가 : 메서드를 입력하기 위한 ROW를 증가시킨다.
  * 삭제 : 메서드를 입력하는 ROW를 감소시킨다.
  * 검색 : 메서드 목록을 조회하는 팝업창을 호출한다.


===== 메서드 조회팝업 =====
검색 버튼을 클릭하면 메서드 목록 정보를 조회한다.



  * 검색 : 선택 또는 입력한 검색조건을 기반으로 메서드 목록을 조회한다.
