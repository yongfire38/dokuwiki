====== 개요 ====== 
커뮤니케이션 도구 시스템에서 사용하는 공통코드를 관리한다.
  
\\

====== 기능 설명 ======
  - 공통코드를 등록할 수 있다.
  - 공통코드를 수정할 수 있다.
  - 공통코드를 조회할 수 있다.
  - 공통코드를 삭제할 수 있다.


\\
====== 화면 설명 ======
===== 목록 화면 =====
{{:egovframework:oe1:가이드:시스템관리:운영환경코드관리:commlist.png|}}
  * 검색조건을 선택하고 검색어를 입력한 후 조회버튼을 클릭하여 등록된 운영환경공통코드명을 조회한다.
  * 코드ID를 클릭하여 운영환경공통코드 상세조회 화면으로 이동한다.
  * 등록 : 운영환경공통코드 등록 화면으로 이동한다.

===== 상세 화면 =====
{{:egovframework:oe1:가이드:시스템관리:운영환경코드관리:commdetail.png|}}
  * 수정 : 운영환경 공통코드 수정 화면으로 이동한다.
  * 삭제 : 현재 운영환경 공통코드를 미사용으로 처리한다.
  * 목록 : 운영환경 공통코드 목록 화면으로 이동한다.

===== 등록 화면 =====
{{:egovframework:oe1:가이드:시스템관리:운영환경코드관리:commregist.png|}}
  * 전 항목이 모두 필수 입력 항목이다.
  * 저장 : 입력사항을 완성하고 저장버튼을 클릭하여 공통코드를 등록 처리한다.
  * 목록 :운영환경공통코드 목록화면으로 이동한다.
**  * 운영환경분류코드가 사전에 등록되어야 분류코드를 선택할 수 있다.**

===== 수정 화면 =====
{{:egovframework:oe1:가이드:시스템관리:운영환경코드관리:commupdt.png|}}
  * 분류코드와 코드ID는 수정이 불가하다.
  * 저장 : 수정사항을 입력하고 저장버튼을 클릭하여 운영환경 공통코드를 수정 처리한다.
  * 목록 : 운영환경 공통코드 목록 화면으로 이동한다.
