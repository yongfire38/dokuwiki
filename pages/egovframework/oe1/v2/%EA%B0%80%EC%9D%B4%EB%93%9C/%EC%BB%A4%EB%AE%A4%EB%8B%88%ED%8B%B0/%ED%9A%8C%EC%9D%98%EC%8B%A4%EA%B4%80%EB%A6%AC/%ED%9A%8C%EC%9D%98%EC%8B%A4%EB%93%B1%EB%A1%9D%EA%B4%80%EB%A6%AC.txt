====== 개요 ====== 
커뮤니티관리자가 회의실정보를 등록/수정/상세조회/목록검색 할 수 있는 기능을 제공한다.

====== 기능 설명 ======

  - 회의실관리 목록 : 회의실관리 목록을 검색 조회한다.
  - 회의실관리 상세 : 등록된 회의실관리 내용에 대한 정보를 상세 조회한다.
  - 회의실관리 등록 : 회의실관리 상세정보를 등록한다.
  - 회의실관리 수정 : 회의실관리 상세정보를 수정한다.

====== 화면 설명 ======

===== 회의실관리 목록 =====
회의실관리 목록은 페이지당 10건씩 조회되며 검색조건은 **회의실명**, **회의실설명** 이다. 

{{:egovframework:oe1:가이드:커뮤니티:회의실관리:회의실관리_목록.jpg|}}

  * 검색 : 선택 또는 입력한 검색조건을 기반으로 등록한 회의실관리 내용에 대한 정보를 목록으로 조회한다.
  * 등록 : 회의실관리 내용을 등록하기 위한 화면으로 이동한다.

===== 회의실관리 상세  =====
회의실관리 목록에서 문서명을 클릭하면 해당 용어에 대한 정보를 상세 조회한다.

{{:egovframework:oe1:가이드:커뮤니티:회의실관리:회의실관리_상세.jpg|}}

  * 수정 : 회의실관리 정보를 수정할 수 있는 편집 가능한 화면으로 이동한다. 단, 자신이 등록한 회의실만 수정 가능하다. 
  * 목록 : 회의실관리 목록 조회 화면으로 이동한다. 
  * 삭제 : 등록한 회의실을 삭제하는 화면으로 이동한다. 단, 자신이 등록한 회의실만 삭제 가능하다. 

===== 회의실관리 등록 =====
회의실관리 내용을 등록할 수 있는 기능을 제공한다.

{{:egovframework:oe1:가이드:커뮤니티:회의실관리:회의실관리_등록.jpg|}}

  * 회의실 관리자 선택 목록
{{:egovframework:oe1:가이드:커뮤니티:회의실관리:회의실관리_관리자_선택.jpg|}}

  * 저장 : 입력한 회의실관리 내용을 등록한다.
  * 목록 : 회의실관리 목록 조회 화면으로 이동한다.
  * 초기화 : 입력한 회의실관리 내용을 공백으로 초기화 한다. 


===== 회의실관리 수정 =====
회의실관리 상세 조회 화면에서 수정 버튼을 클릭한 경우  해당 회의실관리에 대한 정보를 수정할 수 있는 기능을 제공한다.

{{:egovframework:oe1:가이드:커뮤니티:회의실관리:회의실관리_수정.jpg|}}

  * 저장 : 입력한 회의실관리 내용을 수정한다.
  * 목록 : 회의실관리 목록 조회 화면으로 이동한다.
