====== 개요 ====== 
운영정보관리자가 문서이력를 등록/수정/상세조회/목록검색 할 수 있는 기능을 제공한다.

====== 기능 설명 ======

  - 문서이력관리 목록 : 문서이력관리 목록을 검색 조회한다.
  - 문서이력관리 상세 : 등록된 문서이력관리 내용에 대한 정보를 상세 조회한다.
  - 문서이력관리 등록 : 문서이력관리 상세정보를 등록한다.
  - 문서이력관리 수정 : 문서이력관리 상세정보를 수정한다.

====== 화면 설명 ======

===== 문서이력관리 목록 =====
문서이력관리 목록은 페이지당 10건씩 조회되며 검색조건은 **문서명**, **문서설명**,  **산출단계**,  **문서종류** 이다. 

{{:egovframework:oe1:가이드:운영정보관리:문서이력관리_목록.jpg|}}

  * 검색 : 선택 또는 입력한 검색조건을 기반으로 등록한 문서이력관리 내용에 대한 정보를 목록으로 조회한다. 검색조건 중  산출단계를 변경하면 하위 문서종류가 바뀐다. 
  * 등록 : 문서이력관리 내용을 등록하기 위한 화면으로 이동한다.

===== 문서이력관리 상세  =====
문서이력관리 목록에서 문서명을 클릭하면 해당 용어에 대한 정보를 상세 조회한다.

{{:egovframework:oe1:가이드:운영정보관리:문서일력관리_상세.jpg|}}

  * 수정 : 문서이력관리 정보를 수정할 수 있는 편집 가능한 화면으로 이동한다. 단, 자신이 등록한 문서만 수정 가능하다. 
  * 목록 : 문서이력관리 목록 조회 화면으로 이동한다. 
  * 삭제 : 등록한 문서를 삭제하는 화면으로 이동한다. 단, 자신이 등록한 문서만 삭제 가능하다. 

===== 문서이력관리 등록 =====
문서이력관리 내용을 등록할 수 있는 기능을 제공한다.

{{:egovframework:oe1:가이드:운영정보관리:문서이력관리_등록.jpg|}}

  * 저장 : 입력한 문서이력관리 내용을 등록한다.
  * 목록 : 문서이력관리 목록 조회 화면으로 이동한다.
  * 초기화 : 입력한 문서이력관리 내용을 공백으로 초기화 한다. 


===== 문서이력관리 수정 =====
문서이력관리 상세 조회 화면에서 수정 버튼을 클릭한 경우  해당 문서이력관리에 대한 정보를 수정할 수 있는 기능을 제공한다.

{{:egovframework:oe1:가이드:운영정보관리:문서이력관리_수정.jpg|}}

  * 저장 : 입력한 문서이력관리 내용을 수정한다.
  * 목록 : 문서이력관리 목록 조회 화면으로 이동한다.
