====== 개요 ====== 
관리자가 자료사전정보의 목록검색/등록/수정/상세조회/이력정보조회를 할 수 있는 기능을 제공한다.

====== 기능 설명 ======

  - 자료사전목록조회 : 자료사전 목록을 검색 조회한다.
  - 자료사전등록 : 자료사전 정보를 등록한다.
  - 자료사전수정 : 등록된 자료사전 내용에 대한 정보를 수정한다.
  - 자료사전상세조회 : 등록된 자료사전 내용에 대한 정보를 상세 조회한다.
  - 자료사전이력조회팝업 : 자료사전 내용에 대한 이력정보를 조회한다.

====== 화면 설명 ======

===== 자료사전목록조회 =====
자료사전 목록은 페이지당 10건씩 조회되며 검색조건은 **사용여부**, **자료사전명**, **처리자** 이다. 

{{:egovframework:oe1:가이드:운영정보관리:메타데이터:자료사전목록조회.jpg|}}

  * 검색 : 선택 또는 입력한 검색조건을 기반으로 등록한 자료사전에 대한 정보를 목록으로 조회한다.
  * 등록 : [자료사전등록] 화면으로 이동한다.

===== 자료사전등록 =====
자료사전 내용을 등록할 수 있는 기능을 제공한다.

{{:egovframework:oe1:가이드:운영정보관리:메타데이터:자료사전등록.jpg|}}

  * 저장 : 입력한 자료사전정보를 등록한다.
  * 목록 : [자료사전목록조회] 화면으로 이동한다.


===== 자료사전수정 =====
[자료사전상세조회] 화면에서 수정 버튼을 클릭한 경우  해당 자료사전에 대한 정보를 수정할 수 있는 기능을 제공한다.

{{:egovframework:oe1:가이드:운영정보관리:메타데이터:자료사전수정.jpg|}}

  * 저장 : 입력한 자료사전정보를 수정한다.
  * 목록 : [자료사전목록조회] 화면으로 이동한다.

===== 자료사전상세조회  =====
[자료사전목록조회] 화면에서 [자료사전ID]를 클릭하면 해당 자료사전에 대한 상세정보를 조회한다.

{{:egovframework:oe1:가이드:운영정보관리:메타데이터:자료사전상세조회.jpg|}}

  * 삭제/재사용 : 조회된 자료사전에 대한 사용여부를 미사용/사용으로 변경 처리한다.
  * 수정 : 조회된 자료사전을 수정할 수 있는 [자료사전수정] 화면으로 이동한다.
  * 목록 : [자료사전목록조회] 화면으로 이동한다.

===== 자료사전이력조회팝업  =====
[자료사전상세조회] 화면에서 [처리일자]를 클릭한 경우 해당 자료사전 처리에 대한 이력정보를 상세 조회한다.

{{:egovframework:oe1:가이드:운영정보관리:메타데이터:자료사전이력상세조회.jpg|}}

  * 닫기 : 팝업 화면을 닫는다.
