


====== 개요 ====== 
변경담당자들이 변경작업계획 및 변경작업결과에 대해 검토요청을 의뢰한 건에 대해 운영개선관리자가 계획검토/완료검토/상세조회/목록검색 할 수 있는 기능을 제공한다.

====== 기능 설명 ======
  - 변경검토 요청 건 목록 : 변경작업계획 및 변경작업결과에 대한 검토요청 건 목록을 검색조회한다.
  - 변경작업계획검토 : 운영개선관리자가 각 변경담당자들이 검토요청한 변경작업계획을 승인/반려한다.
  - 변경작업완료검토 : 운영개선관리자가 각 변경담당자들이 검토요청한 변경작업을 승인/반려한다.
  - 변경요청서상세 팝업 : 변경요청서 상세정보를 조회한다.  
  - 변경작업계획검토이력 팝업 : 변경작업계획 검토요청건에 대한 반려이력정보를 조회한다.  
	- 변경작업결과검토이력 팝업 : 변경작업결과 검토요청건에 대한 반려이력정보를 조회한다.  

====== 화면 설명 ======

===== 변경검토 요청 건 목록 =====
변경검토 요청 건 목록은 페이지당 10건씩 조회되며 검색조건은 **검토요청구분**, **긴급처리**, **접수일**이다. 


{{:egovframework:oe1:가이드:운영개선관리:변경작업관리:변경검토승인건목록.jpg|}}


  * 검색 : 선택 또는 입력한 검색조건을 기반으로 변경검토 요청 건 목록을 조회한다.



===== 변경작업계획검토  =====
변경검토요청 건 목록에서 처리상태가 [계획검토]인 건을 클릭하면 운영개선관리자가 각 변경담당자들이 검토요청한 변경작업계획을 승인/반려할 수 있는 기능을 제공한다.


{{:egovframework:oe1:가이드:운영개선관리:변경작업관리:변경계획검토.jpg|}}


  * 변경요청서상세 : 변경요청서 상세정보를 팝업조회한다.
  * 검토이력조회 : 해당 변경작업계획 검토요청건에 대한 반려이력정보를 팝업조회한다.
  * 계획검토 : 해당 변경작업계획 검토요청건을 계획검토결과에 따라 승인/반려 처리 한다.
  * 목록 : 변경작업 검토 요청 건 목록 조회 화면으로 이동한다. 


===== 변경작업완료검토  =====
변경검토요청 건 목록에서 처리상태가 [완료검토]인 것을 클릭하면 운영개선관리자가 각 변경담당자들이 검토요청한 변경작업결과를 승인/반려 할 수 있는 기능을 제공한다.


{{:egovframework:oe1:가이드:운영개선관리:변경작업관리:변경완료검토.jpg|}}


  * 변경요청서상세 : 변경요청서 상세정보를 팝업조회한다.
  * 검토이력조회 : 해당 변경작업결과 검토요청건에 대한 반려이력정보를 팝업조회한다.
  * 완료검토 : 해당 변경작업결과 검토요청건을 완료검토결과에 따라 승인/반려 처리 한다.
  * 목록 : 변경작업검토 요청 건 목록 조회 화면으로 이동한다. 



===== 변경요청서 상세팝업 =====
변경요청서 정보를 상세 조회한다.

  * 요청근거 : 변경요청서를 등록한 요청시스템의 출처 정보를 조회한다.
  * 닫기 : 화면을 종료한다.
