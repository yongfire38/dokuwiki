

====== 개요 ====== 

운영개선담당자가 분배된 위험에 대한 예방활동결과를 저장/완료처리/상세조회/목록검색을 할 수 있는 기능을 제공한다.
운영개선담당자가 이외 사용자에게는 위험예방활동정보를 상세조회/목록검색을 할 수 있는 기능을 제공한다.


====== 기능 설명 ======

  - 위험예방활동관리 목록 : 등록된 위험에 대한 담당자정보 및 예방활동결과정보를 목록으로 검색 조회한다.
  - 위험예방활동정보 상세 : 운영개선담당자에 의해 진행된 예방활동정보를 상세 조회한다.
  - 위험예방활동정보 처리 : 운영개선담당자가 분배된 위험에 대한 예방활동결과정보를 저장/완료처리 한다.
  

====== 화면 설명 ======

===== 위험예방활동관리 목록 =====
위험예방활동관리 목록은 페이지당 10건씩 조회되며 검색조건은 **처리상태**, **위험구분**, **위험도**, **위험명**, **위험내용**이다. 

{{:egovframework:oe1:가이드:운영개선관리:위험관리:risk_02_01.jpg|}}

  * 검색 : 선택 또는 입력한 검색조건을 기반으로 위험예방활동관리 목록을 조회한다.
  * 처리 : 로그인한 사용자가 담당자인 건에 처리 버튼이 보여지며, 위험예방활동정보 처리 화면으로 이동한다.


===== 위험예방활동정보 상세  =====
위험예방활동관리 목록에서 위험명을 클릭하면 해당 위험대한 예방활동정보를 상세 조회한다.

{{:egovframework:oe1:가이드:운영개선관리:위험관리:risk_02_02.jpg|}}

  * 목록 : 위험예방활동관리 목록 조회 화면으로 이동한다. 



===== 위험예방활동정보 처리 =====
위험예방활동관리 목록 화면에서 처리버튼을 클릭한 경우 위험예방활동정보를 저장하고 완료처리할 수 있는 기능을 제공한다.

{{:egovframework:oe1:가이드:운영개선관리:위험관리:risk_02_03.jpg|}}

  * 완료처리 : 운영개선담당자가 분배된 위험에 대해 예방활동결과를 입력하고 완료처리 한다.
  * 저장     : 운영개선담당자가 분배된 위험에 대해 예방활동을 한 정보를 저장한다.
  * 목록     : 위험예방활동관리 목록 조회 화면으로 이동한다.
