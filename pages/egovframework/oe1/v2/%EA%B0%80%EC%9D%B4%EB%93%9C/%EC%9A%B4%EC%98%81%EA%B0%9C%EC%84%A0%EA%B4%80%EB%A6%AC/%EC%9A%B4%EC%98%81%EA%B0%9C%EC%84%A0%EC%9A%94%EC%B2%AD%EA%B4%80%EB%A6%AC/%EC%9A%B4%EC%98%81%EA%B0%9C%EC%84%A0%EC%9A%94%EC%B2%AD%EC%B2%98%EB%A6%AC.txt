

====== 개요 ====== 
운영개선담당자에게 운영개선요청 처리를 위한 저장/목록검색 할 수 있는 기능을 제공한다.

====== 기능 설명 ======
  - 운영개선요청 처리 목록 : 운영개선요청 처리 목록을 검색 조회한다.
  - 요청자 contact 정보 팝업 : 요청자의 contact 정보를 조회한다.
  - 운영개선요청 처리 저장 : 운영개선요청 처리 정보를 등록한다.
  - 운영개선요청 처리 상세조회 : 운영개선요청 처리 정보를 상세조회한다.
  

====== 화면 설명 ======

===== 운영개선요청 처리 목록 =====
운영개선요청 처리 목록은 페이지당 10건씩 조회되며 검색조건은 **처리상태**, **업무구분**, **개선요청명**, **내용**, **요청자**이다. 

{{:egovframework:oe1:가이드:운영개선관리:운영개선요청관리:oper_03_01.jpg|}}

  * 검색 : 선택 또는 입력한 검색조건을 기반으로 운영개선요청 처리 목록을 조회한다.
  * 개선요청명 : 운영개선요청 처리 저장 화면으로 이동한다.
  * 요청자명 : 요청자 contact 정보 조회를 위한 팝업을 띄운다.
 
 
===== 요청자 contact 정보 팝업 =====
요청자의 **전화번호**, **핸드폰번호**, **이메일주소** 정보를 조회한다.

{{:egovframework:oe1:가이드:운영개선관리:운영개선요청관리:oper_02_06.jpg|}}

  * 닫기 : 화면을 종료한다.
 

===== 운영개선요청 처리 저장 =====
운영개선요청 처리 목록에서 처리상태가 [접수], [작업중]인 개선요청명을 클릭하면 운영개선요청 처리 정보를 저장하는 기능을 제공한다.

{{:egovframework:oe1:가이드:운영개선관리:운영개선요청관리:oper_03_02.jpg|}}

  * 찾아보기 : 첨부파일을 선택하기 위한 팝업을 띄운다.
  * 첨부파일 목록에서 파일명 옆의 X 이미지를 클릭하면 첨부파일을 삭제한다.
  * 완료처리 : 운영개선요청 처리상태를 완료로 수정한다. 
  * 저장 : 입력한 운영개선요청 처리 정보를 등록한다. 
  * 목록 : 운영개선요청 처리 목록 조회 화면으로 이동한다. 


===== 운영개선요청 처리 상세조회 =====
운영개선요청 처리 목록에서 처리상태가 [완료]인 개선요청명을 클릭하면 운영개선요청 처리 정보를 상세조회 한다.

{{:egovframework:oe1:가이드:운영개선관리:운영개선요청관리:oper_03_03.jpg|}}

  * 목록 : 운영개선요청 처리 목록 조회 화면으로 이동한다.
