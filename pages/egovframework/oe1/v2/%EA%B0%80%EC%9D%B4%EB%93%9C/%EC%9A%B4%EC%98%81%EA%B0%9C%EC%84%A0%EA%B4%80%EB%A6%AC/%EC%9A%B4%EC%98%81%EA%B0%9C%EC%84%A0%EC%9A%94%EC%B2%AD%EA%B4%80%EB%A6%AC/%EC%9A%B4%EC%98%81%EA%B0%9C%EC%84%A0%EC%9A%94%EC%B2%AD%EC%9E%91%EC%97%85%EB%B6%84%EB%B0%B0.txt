

====== 개요 ====== 
운영개선관리자에게 운영개선요청에 대한 작업분배/변경이관/요청종료/위험등록/목록검색 할 수 있는 기능을 제공한다.

====== 기능 설명 ======
  - 운영개선요청 작업분배 목록 : 운영개선요청 작업분배 목록을 검색 조회 한다.
  - 요청자 contact 정보 팝업 : 요청자의 contact 정보를 조회한다.
  - 운영개선요청 작업분배 : 운영개선요청 관리자가 운영개선요청에 대한 작업분배를 한다.
  - 운영개선요청 변경이관 팝업 : 운영개선요청 관리자가 운영개선요청을 변경이관 한다.
  - 운영개선요청 요청종료 팝업 : 운영개선요청 관리자가 운영개선요청을 요청종료 한다.
  - 운영개선요청 위험등록 : 운영개선요청 관리자가 운영개선요청을 근거로 하는 위험을 등록 한다.  

====== 화면 설명 ======

===== 운영개선요청 작업분배 목록 =====
운영개선요청 목록은 페이지당 10건씩 조회되며 검색조건은 **처리상태**, **업무구분**, **개선요청명**, **내용**, **요청자**이다. 

{{:egovframework:oe1:가이드:운영개선관리:운영개선요청관리:oper_02_01.jpg|}}

  * 검색 : 선택 또는 입력한 검색조건을 기반으로 운영개선요청 목록을 조회한다.
  * 개선요청명 : 운영개선요청 작업분배 화면으로 이동한다.
  * 요청자명 : 요청자 contact 정보 조회를 위한 팝업을 띄운다.
  * 위험등록 : 운영개선요청을 근거로하는 위험을 등록하기 위한 화면으로 이동한다.


===== 요청자 contact 정보 팝업 =====
요청자의 **전화번호**, **핸드폰번호**, **이메일주소** 정보를 조회한다.

{{:egovframework:oe1:가이드:운영개선관리:운영개선요청관리:oper_02_06.jpg|}}

  * 닫기 : 화면을 종료한다.


===== 운영개선요청 작업분배  =====
운영개선요청 작업분배 목록에서 처리상태가 [요청], [접수]인 개선요청명을 클릭하여 해당 운영개선요청에 대한 작업분배 정보를 저장한다.

{{:egovframework:oe1:가이드:운영개선관리:운영개선요청관리:oper_02_02.jpg|}}

  * 담당자찾기 : 운영개선담당자 정보를 팝업조회한다.
  * 변경이관 : 운영개선요청을 변경으로 이관 등록하기 위한 팝업을 띄운다.
  * 요청종료 : 운영개선요청을 종료하기 위한 팝업을 띄운다.
  * 저장 : 운영개선요청 작업분배 정보를 삭제한다.
  * 목록 : 운영개선요청 작업분배 목록 조회 화면으로 이동한다. 


===== 운영개선요청 변경이관 팝업 =====
운영개선요청을 변경으로 이관하여 등록하는 기능을 제공한다.

{{:egovframework:oe1:가이드:운영개선관리:운영개선요청관리:oper_02_03.jpg|}}

  * 저장 : 입력한 변경요청서 정보를 등록한다. 
  * 닫기 : 화면을 종료한다. 


===== 운영개선요청 요청종료 팝업 =====
운영개선요청을 종료 처리하는 기능을 제공한다.

{{:egovframework:oe1:가이드:운영개선관리:운영개선요청관리:oper_02_07.jpg|}}

  * 저장 : 입력한 요청종료사유 정보를 등록한다. 
  * 닫기 : 화면을 종료한다.


===== 운영개선요청 위험등록 =====
운영개선요청 상세 조회 화면에서 수정 버튼을 클릭하여 운영개선요청 정보를 수정한다.

{{:egovframework:oe1:가이드:운영개선관리:운영개선요청관리:oper_02_04.jpg|}}

  * 첨부파일 : 첨부파일을 선택하기 위한 팝업을 띄운다. 
  * 저장 : 입력한 위험 정보를 등록한다. 
  * 목록 : 위험관리 목록 화면으로 이동한다.
