====== 스페이스관리 ======
==== 개요 ====
스페이스를 등록, 수정, 삭제하고 스페이스별 사용자 및 사용자롤을 할당하는 기능이다.

===== 1. 스페이스 등록 =====

[화면캡쳐]\\
① 스페이스명, 스페이스키, 설명을 입력한다.\\
② '등록' 버튼을 클릭한다.\\
③ 스페이스가 등록된 후 스페이스목록 화면으로 이동한다.\\
④ '목록' 버튼을 클릭하여 스페이스목록 화면으로 이동한다.\\
⑤ '초기화' 버튼을 클릭하여 작성한 내용을 초기화한다.\\

===== 2. 스페이스 목록 =====

[화면캡쳐]\\
① 스페이스 목록을 조회한다.\\
② '사용자할당' 버튼을 클릭하여 스페이스 사용자등록 화면으로 이동한다.\\
③ '수정' 버튼을 클릭하여 스페이스수정 화면으로 이동한다.\\
④ '스페이스등록' 버튼을 클릭하여 스페이스등록 화면으로 이동한다.\\
⑤ 삭제할 스페이스를 선택한 후 '스페이스삭제' 버튼을 클릭하여 삭제한다.(멀티삭제 가능)\\

===== 3. 사용자 할당 =====

[화면캡쳐]\\
① 사용자 및 사용자롤을 선택한다.\\
② '사용자등록' 버튼을 클릭하여 사용자 및 사용자롤을 등록한다.\\
③ '목록' 버튼을 클릭하여 스페이스목록 화면으로 이동한다.
④ '삭제' 버튼을 클릭하여 등록된 사용자의 롤을 삭제한다.

===== 4. 스페이스 수정 =====

[화면캡쳐]\\
① 스페이스명, 스페이스키, 설명을 입력한다.\\
② '수정' 버튼을 클릭하여 스페이스를 수정한다.\\
③ '삭제' 버튼을 클릭하여 스페이스를 삭제한 후 스페이스목록 화면으로 이동한다.\\
④ '초기화' 버튼을 클릭하여 작성한 내용을 초기화한다.\\
⑤ '목록' 버튼을 클릭하여 스페이스목록 화면으로 이동한다.
